# 会话报告

CrabCode 会话报告插件:从本地会话记录(`~/.crabcode/projects`)生成可交互的 HTML 用量报告,覆盖 token 消耗、缓存命中、代理与技能使用、高开销提示词,便于分享与复盘。

## 安装

在 CrabCode 插件市场中添加 `session-report`,或通过 `/plugin` 安装。

## 技能(按需自动触发)

| 技能 | 说明 |
|---|---|
| `session-report` | 解析本地会话转写文件,生成带样式的 HTML 报告(用量、缓存、代理、技能、高开销提示) |

## 使用入口

直接说"给我生成本次会话的用量报告"或"分析一下最近的会话开销",CrabCode 会自动匹配本技能。
