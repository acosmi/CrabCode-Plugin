# 连接器

## 工具引用机制

插件文件用 `~~类别` 作为占位符,代表用户在该类别下连接的具体工具。例如 `~~accounting` 可指用友好会计、金蝶精斗云或任何提供 MCP 服务的财务系统。

插件是**工具无关**的——它们以类别(财务、收款、CRM、设计等)而非具体产品来描述工作流。`.mcp.json` 预配置了具体的 MCP 服务,但同类别下的任何 MCP 服务都可使用。

## 本插件的连接器

**已内置(官方 MCP 直连)** —— 凭证在 `/plugin → 管理插件 → 本插件 → 配置` 中填写,按需填、填哪个启用哪个。未填凭证的连接器不会启动,对应工作流自动走降级路径,不影响其它功能。

| 类别 | 内置服务 | 需要配置 | 启用能力 |
|---|---|---|---|
| 收款 | **支付宝**(官方 MCP) | 应用 APPID + 应用私钥 + 支付宝公钥(开放平台申请) | 创建收款链接、按单号查单、退款——每一步都先经你确认 |
| 团队 IM / 日程 | **钉钉**(官方 MCP) | Client ID + Client Secret(开放平台应用) | 消息推送、简报通知、日程 |
| 团队 IM / 日历 / 文档 | **飞书**(官方 MCP) | App ID + App Secret(自建应用) | 消息推送、日历、文档 |
| 在线文档 / 表格 | **腾讯文档**(官方 MCP) | API 访问令牌 | 模板、报表、导出物 |
| CRM / 营销 | **HubSpot** | OAuth 登录 | 线索、商机、营销活动、客服工单(国内 CRM 选型待定,暂保留) |

**待接入(国内替代方案落地前的过渡口径)** —— 对应工作流不依赖连接器也能运行,按技能内提示改用导出文件、粘贴数据或人工执行:

| 类别 | 目标服务 | 过渡做法 |
|---|---|---|
| 财务 / 记账 | 用友好会计 / 金蝶精斗云 | 从记账软件导出 CSV/Excel 或粘贴报表 |
| 收款(补充) | 微信支付 | 商户平台账单导出 |
| 电子签 | 众律宝(自营) | 产出终稿文档,由你经众律宝或纸面签署 |
| 设计 | 自营云端设计 | 产出完整设计说明与文案,由你在设计工具执行 |
| 邮件 | 腾讯企业邮 | 邮件粘贴进来,草稿由你复制发送 |
| 网盘 | 阿里云盘 | 本地文件 / 腾讯文档 |

> 收款数据说明:支付宝官方 MCP 只做「建收款链接 / 查单 / 退款」,**不提供交易流水批量导出**。现金流、对账类技能所需的交易数据,从支付宝商家平台 / 微信支付商户平台导出账单(CSV)后交给技能分析。

> 替换背景见仓库 `docs/audit/2026-06-23-crabwork插件海外SaaS连接器替换国内设施-现状审计与方案.md`;本插件是阶段 1 试点。
