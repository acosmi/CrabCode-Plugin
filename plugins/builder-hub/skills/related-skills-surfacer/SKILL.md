# /builder-hub:related-skills-surfacer

基于当前任务上下文,推荐关联的技能(skill)。

## Workflow

1. 解析当前任务:从用户描述或上下文中提取关键词、目标与场景。
2. 匹配候选技能:
   - 在已安装技能中查找描述(description)与触发词相关的项。
   - 在 CrabCode 插件市场 `marketplace.json` 中查找未安装但相关的项。
3. 计算相关度:按关键词命中、场景契合、常见组合关系排序。
4. 区分来源:标明每个推荐项是已安装还是需从市场安装。
5. 给出简短的使用理由,说明它如何衔接当前任务。

## Output

返回:

- 已安装的相关技能(可直接调用)。
- 市场中可安装的相关技能(附安装提示)。
- 每项的相关理由与衔接说明。

## Next Steps

- 直接调用推荐的已装技能继续当前任务。
- 用 `/builder-hub:skill-installer` 安装市场中的相关项。
- 用 `/builder-hub:registry-browser` 进一步检索同类技能。
