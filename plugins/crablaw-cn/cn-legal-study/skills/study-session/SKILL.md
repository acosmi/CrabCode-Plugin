# /cn-legal-study:study-session

【教育备考用途，非针对个案的法律意见】

围绕考生提出的具体知识点或疑难,开展讲解式答疑会话,直到该考点被理解掌握。

## Workflow

1. 定位疑问:确认考生卡点(概念不懂、要件混淆、法条不会用、案例看不透),界定所属学科与考试要求。
2. 讲解核心:用准确而通俗的方式拆解概念与规则,先给结论再给原理;核心规则标 `[已核验-来源]` 或 `[模型知识-待核]`。
3. 举例佐证:用贴近考点的小案例或对比例说明适用边界,辨析易混概念。
4. 随问随答:针对考生追问继续解释,允许多轮深挖,逐步逼近真正卡点。
5. 即时检测:用一两道小问当场检验是否掌握,纠正残留误解;法条口径不确定处标 `[模型知识-待核]`,提示以现行法条与官方教材为准。
6. 小结固化:总结要点、易错提示与一句话记忆口径。

## Output

- 知识点讲解(结论+原理)。
- 举例与易混辨析。
- 多轮追问应答记录。
- 当场检测题与反馈。
- 要点小结与记忆口径。
- 来源标签标注。
