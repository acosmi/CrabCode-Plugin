# CrabCode 示例技能集

CrabCode 官方示例技能合集:演示算法艺术、视觉设计、品牌样式、文档共创、内部通讯、MCP 服务器编写、技能编写、GIF 制作、主题样式、Web 产物构建与 Playwright 网页测试等能力,可直接使用,也可作为编写自有技能的范本。

## 安装

在 CrabCode 插件市场中添加 `crabcode-example-skills`,或通过 `/plugin` 安装。

## 技能(按需自动触发)

| 技能 | 说明 |
|---|---|
| `algorithmic-art` | p5.js 生成艺术:流场、粒子系统、种子随机与参数探索 |
| `brand-guidelines` | 将项目视觉规范(色板、字体、点缀形状)应用到演示、文档或页面 |
| `canvas-design` | 以设计哲学产出 .png/.pdf 海报与静态视觉作品 |
| `doc-coauthoring` | 结构化文档共创流程:提案、技术方案、决策文档、RFC、PRD |
| `frontend-design` | 高设计品质的生产级前端界面与组件 |
| `internal-comms` | 内部通讯写作:进展/计划/问题更新、公告、FAQ 等 |
| `mcp-builder` | 编写高质量 MCP 服务器,把外部服务接入代理工具 |
| `skill-creator` | 从零编写、改进与评测代理技能 |
| `slack-gif-creator` | 制作适配 Slack 的动画 GIF |
| `theme-factory` | 十套预置主题为幻灯片、文档、报告与落地页配色配字 |
| `web-artifacts-builder` | React + Tailwind + shadcn/ui 构建多组件 HTML 产物 |
| `webapp-testing` | 用 Playwright 与本地 Web 应用交互:验证功能、截图、查看日志 |

## 使用入口

直接描述任务(如"做一张生成艺术海报""帮我写个 MCP 服务器"),CrabCode 会自动匹配技能;也可在编写自有技能时参考本插件的 SKILL.md 结构。
