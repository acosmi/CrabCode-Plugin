# AI API 开发

面向 LLM 应用开发的 CrabCode 插件:以供应商中立的方式构建、调试与优化调用大模型 API 的应用,覆盖 API 形态选型、提示词缓存、工具调用、结构化输出、流式传输与跨 SDK 迁移。

## 安装

在 CrabCode 插件市场中添加 `ai-api-dev`,或通过 `/plugin` 安装。

## 技能(按需自动触发)

| 技能 | 说明 |
|---|---|
| `ai-api-dev` | 编写调用 LLM API 的代码(补全、工具调用、结构化输出、流式)、排查 API 报错、优化 token 与缓存成本、在不同供应商 SDK 之间迁移时触发 |

## 使用入口

直接描述你的 LLM 应用开发任务(如"帮我给这个服务加上工具调用"或"把这段代码迁到另一家 SDK"),CrabCode 会自动匹配本技能。
