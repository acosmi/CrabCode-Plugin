# fin-core 中国数据源适配器(中文优先)

> fin-core 底座沿用上游(Apache-2.0)的通用建模引擎,其建模机制(DCF/三表/可比/WACC 等)是普适的,**保留英文正文**;但**数据来源与若干参数默认改为中国口径**。本文件是各底座技能的中国数据源/参数对照,**中文优先**,使用建模技能时先读本文件再套用英文流程。
>
> 边界:本文件只给"数据从哪取""参数怎么选"的对照与惯例,**不充当法规口径来源**;凡涉及会计准则细节,以 CAS 原文及各 cn-* 板块经核对的 sources 为准。

---

## 一、数据来源对照(US → 中国)

| 用途 | 上游(美国)原默认 | 中国对应权威来源 |
|---|---|---|
| 上市公司定期报告/财务数据 | SEC EDGAR、10-K / 10-Q / 8-K | **巨潮资讯网(cninfo.com.cn,沪深北法定披露平台)**、上交所/深交所/北交所官网、港股用 HKEXnews(披露易) |
| 招股/募集文件 | S-1 | A股招股说明书(交易所/巨潮)、港股 IPO 招股章程(HKEXnews) |
| 行情/估值/一致预期 | Bloomberg / FactSet / Capital IQ | Wind、同花顺 iFinD、东方财富 Choice、聚源/朝阳永续(一致预期) |
| 行业分类 | GICS / SIC | **申万行业分类**(本套件 cn-equity-research/sources 内,注意以 2021 版代码为准)、证监会行业分类 |
| 无风险利率 | 美国 10Y Treasury | **中国 10 年期国债到期收益率**(中债登/中国债券信息网) |
| 汇率/计价货币 | USD | 人民币 CNY(境内);港股/中概可能为 HKD/USD,需注明计价货币与汇率来源(中国外汇交易中心 CFETS 中间价) |
| 公司财报报表科目 | US GAAP | **企业会计准则(CAS)**;科目→报表行项映射见 cn-equity-research/sources/mappings/CAS科目映射 |

## 二、参数惯例(中国市场,均为惯例而非法规)

- **无风险利率 Rf**:取建模时点中国 10 年期国债到期收益率。
- **股权风险溢价 ERP**:A股常用区间需引用可查来源(如 Wind/券商策略报告)并注明取数日;**不要凭记忆写死数值**。
- **Beta**:用 A股/港股可比公司相对沪深300/恒生指数回归,或行业 Beta;注明指数基准。
- **永续增长 g**:不超过长期名义 GDP 增速预期,需给出处。
- **税率**:企业所得税法定 25%,高新技术企业等优惠税率需按标的实际适用注明(以标的披露为准)。
- **可比公司**:同申万行业、同板块(主板/科创板/创业板/北交所/港股)优先;跨市场可比须做流动性与制度差异调整(参考 cn-equity-research/sources/benchmarks)。

## 三、披露与计价提示

- A股财报为合并报表口径、人民币元;注意"扣非归母净利润"与归母净利润的区分(中国研究惯例)。
- 港股/中概股注意计价货币、准则(部分用 IFRS/US GAAP)与 A/H 溢价。
- 任何模型输出须注明:数据来源、取数日期、计价货币、所用准则。
