# 流程函

为中国投行并购交易出具正式流程函，明确四大审批线时间表和各阶段责任方。

> **重组审批**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（交易所3月+证监会15工作日）
> **反垄断**：`sources/laws/经营者集中申报标准.md`（180自然日最长审查+抢跑处罚）
> **国资交易**：`sources/laws/国资交易_外资准入要点.md`（挂牌≥20工作日/评估1年）
> **披露阈值**：`sources/mappings/披露阈值表_v1.csv`（5%/20%/30%触发点）

## 操作步骤

### Step 1 — 确认交易参数

**输入**：用户提供的交易信息
**输出**：交易概述段

| 参数 | 必填 |
|------|------|
| 标的公司全称 | 是 |
| 交易类型（控股转让/参股/增资/合并） | 是 |
| 拟释放比例 | 是 |
| 交易规模范围（亿元） | 是 |
| 我方角色（独家FA/联席FA/买方顾问） | 是 |

**停止条件**：以上5项任一不可得 → 停止，向用户确认。

### Step 2 — 监管审批总览

**输入**：Step 1 参数 + 标的性质
**输出**：审批矩阵

| 审批/程序 | 是否触发 | 预估周期 |
|-----------|---------|---------|
| 重大资产重组审核 | 总资产/营收/净资产 ≥**50%**（净资产>5000万） | 受理后 **3个月** + **15个工作日** |
| 经营者集中申报 | 全球>**120亿**+各自>**8亿** / 中国>**40亿**+各自>**8亿** | 简易1-2月/普通3-5月/复杂6-18月 |
| 国资审批 | 标的涉及国有产权 | 4-8月（审计+评估+挂牌） |
| 外资安全审查 | 外资取得实际控制权+敏感领域 | **45个工作日** |
| ODI审批 | 中国买方跨境并购 | 1-3月 |

### Step 3 — 四阶段时间表

**输入**：Step 2 的审批触发情况
**输出**：

**第一阶段**（Week 1-4）：NDA→Teaser→买方长名单→CIM→VDR
**第二阶段**（Week 5-10）：Teaser分发→首轮IoI→短名单→尽调开放→管理层演示
**第三阶段**（Week 11-16）：补充尽调→二轮报价→SPA初稿→排他谈判→SPA签署
**第四阶段**（Week 17-30）：经营者集中→国资评估审批→外资安全审查→证监会/交易所→交割

**提醒**：建议在第一阶段即启动反垄断申报前商谈（1-4周）。

### Step 4 — 抢跑警示（Gun-jumping）

在反垄断批准/默许通过之前，以下行为可能构成"抢跑"：
- 交易交割/支付对价
- 实质性整合业务（合并IT系统、统一管理）
- 交换竞争性敏感信息（定价、客户名单）
- **后果**：处上一年度销售额 **10%以下** 罚款+责令停止/恢复原状

### Step 5 — 各方义务清单

| 责任方 | 核心义务 |
|--------|---------|
| 委托方 | 提供真实完整信息；配合尽调；内部决策程序 |
| 财务顾问 | 方案设计；估值建议；中介协调；监管沟通 |
| 法律顾问 | 法律尽调+SPA起草+法律意见书 |
| 审计师 | 财务审计（CAS） |
| 评估师 | 资产评估（国资：必须经核准/备案；有效期 **1年**） |
| 反垄断律师 | 申报材料编写与提交 |

### Step 6 — 信息披露时间表（上市公司）

| 阶段 | 时限 |
|------|------|
| 预案公告 | 董事会决议后 |
| 草案公告 | 交易所审核前 |
| 实施进展 | 每 **至少30日** |
| 持续督导 | **≥1个完整会计年度** |

### Step 7 — 输出流程函

按标准格式输出：抬头→交易概述→审批总览→时间表→抢跑警示→义务清单→信披时间表→签署栏+附件（法规依据索引）。

### Step 8 — 合规声明

```
⚠️ 合规声明：本流程函仅为交易执行计划之概述，不构成法律建议。
具体时限受监管机构审查力度、反垄断商谈情况、国资审批层级等因素影响，可能调整。
所有法定时限以法规原文为准。流程函最终版本须经法律顾问审核后方可正式发出。
```
