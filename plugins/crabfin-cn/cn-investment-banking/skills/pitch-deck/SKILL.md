# 提案幻灯片(Pitch Deck)

为中国投行并购业务制作客户提案幻灯片，12页标准结构，嵌入A股壳价值、业绩承诺、控制权溢价等中国特有评述。

> **格式参考**：`sources/templates/teaser_脱敏模板.md`（Teaser内容结构→pitch扩展）
> **重组规则**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（定价80%/锁定期/业绩承诺）
> **反垄断**：`sources/laws/经营者集中申报标准.md`（120亿/40亿/8亿）
> **国资外资**：`sources/laws/国资交易_外资准入要点.md`（负面清单/32号令）

## 操作步骤

### Step 1 — 封面 + 执行摘要

**输入**：客户/标的基本信息
**输出**：
- Page 1 封面：客户公司/标的名称 — 并购交易提案 | 交易类型 | 日期 | 财务顾问 | 机密标签
- Page 2 执行摘要：一句话交易描述 + 交易规模 + 3个核心投资主题

### Step 2 — 标的公司概览 + 行业市场

**输入**：公司资料 + 行业研究
**输出**：
- Page 3 标的概览（按 `sources/templates/teaser_脱敏模板.md` 公司简介格式）
- Page 4 行业与市场机遇（规模/CAGR/驱动因素）

### Step 3 — 竞争格局 + 投资亮点

**输入**：竞对数据 + 公司核心优势
**输出**：
- Page 5 竞争格局对标表（收入/毛利率/核心壁垒）
- Page 6 5大投资亮点（各含量化支撑）

### Step 4 — 财务表现 + 收入拆分

**输入**：经审计财报
**输出**：
- Page 7 财务摘要表（近2-3年营收/YoY/毛利率/EBITDA/净利率/经营现金流）+ 收入拆分（业务板块/产品线）

**数据不可得fallback**：未经审计数据标注"管理层报表，未经审计"。

### Step 5 — 估值框架

**输入**：多方法估值结果
**输出**：
- Page 8 估值框架表（P/E+EV/EBITDA+DCF → 推荐估值区间）

**特别标注**（来自 `sources/laws/重大资产重组管理办法_2023修订要点.md`）：
- 国资标的：估值须以经核准/备案评估结果为法定定价基础
- 发行股份：最低发行价 ≥ 基准价 **80%**

### Step 6 — 交易架构 + 流程时间表

**输入**：交易方案设计
**输出**：
- Page 9 交易架构表（标的/结构/支付/治理/留任/退出）
- Page 10 交易流程时间表（含Teaser→IoI→尽调→报价→SPA→经营者集中→国资审批→交割）

**合规标注**：
- 反垄断触发检查（全球合计>**120亿**+各自>**8亿** / 中国合计>**40亿**+各自>**8亿**）
- 外资准入检查（对照负面清单：禁止/限制/安全审查）

### Step 7 — 风险 + 团队

**输入**：风险评估 + 团队信息
**输出**：
- Page 11 关键风险与缓释（审批/整合/估值/融资）
- Page 12 团队介绍与资质（MD/ED/D/VP/Associate + tombstone）

### Step 8 — A股交易专项评述

在pitch deck中须包含：
- **控制权溢价**：A股控股权转让通常含10-30%溢价
- **壳价值**：重组上市等同IPO审核，控制权变更后**36个月**内购买资产指标≥**100%**
- **业绩承诺**：向控股股东/实控人购买+基于未来收益法估值→**强制**业绩承诺
- **锁定期**：一般投资者**12个月**/控股股东**36个月**/重组上市**24个月**

### Step 9 — 合规声明

```
⚠️ 合规声明：本提案仅供初步讨论，不构成出售要约或投资建议。
财务数据和估值参数来源于公开信息和合理假设，未经独立验证。
涉及国资标的的，估值不可替代法定资产评估。
交易最终结构和条件以正式协议和监管批准为准。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `merger-model` | 估值框架 | 对价区间/EPS影响 |
| `sector-overview` | 行业分析 | 市场规模/CAGR/竞争格局 |
| `buyer-list` | 确定交易对手方 | 潜在买方短名单 |
| `deal-tracker` | 流程时间表 | 审批时限 |
| `teaser` | pitch后分发teaser | Teaser单页 |

## 产出物路由

- 提案需要交付 .pptx 幻灯片成品时,调用 `crabcode-office-suite:crabcode-presentations` 生成;12 页结构先以 markdown 大纲与用户确认后再落文件。
- 若触发时报 Unknown skill,说明办公套件未安装:引导用户通过 `/plugin` 安装 `crabcode-office-suite` 后重试;安装完成前先以 markdown 逐页呈现内容供用户确认。
