# 数据包整理

为中国投行并购交易各阶段整理标准化数据包，覆盖财务、业务、披露、反垄断、国资、外资、ODI七大模块。

> **披露阈值**：`sources/mappings/披露阈值表_v1.csv`（5%/20%/30%触发点）
> **重组规则**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（≥50%/≥5000万标准）
> **反垄断**：`sources/laws/经营者集中申报标准.md`（120亿/40亿/8亿审查标准）
> **国资规则**：`sources/laws/国资交易_外资准入要点.md`（32号令/评估1年有效期/负面清单）

## 操作步骤

### Step 1 — 确定数据包场景

**输入**：交易阶段/用户需求
**输出**：启用的数据包模块清单

| 场景 | 启用模块 |
|------|---------|
| 卖方尽调准备 | 模块1(财务) + 模块2(业务) + 模块5(国资，如适用) |
| 买方尽调 | 模块1+2+3(披露)+4(反垄断)+5(国资)+6(外资) |
| 监管申报准备 | 模块3+4+5+6+7(ODI) |
| 估值建模 | 模块1+2（核心数据） |

### Step 2 — 模块1：财务数据包

**输入**：公司财报
**输出**：

| 数据项 | 状态 | 来源 | 关键性 |
|--------|------|------|--------|
| 近3年经审计财务报表（CAS编制） | | | 高 |
| 审计调整项/非经常性损益明细 | | | 高 |
| 管理层报表（如与审计差异>5%须专项说明） | | | 中 |
| 分部财务数据（业务板块/产品线/区域） | | | 高 |
| 关联交易明细（对手/金额/定价/必要性） | | | 高 |
| 税务数据（汇算清缴/税收优惠/转让定价文档） | | | 中 |

**停止条件**：审计报告不可得 → 标注"财务数据未经审计，不可用于交易定价"。

### Step 3 — 模块2：业务数据包

**输入**：业务运营数据
**输出**：

| 数据项 | 状态 | 来源 | 关键性 |
|--------|------|------|--------|
| 产能与产量（各基地设计产能/实际产量/利用率） | | | 高 |
| 销售数据（量价拆分：分产品/区域/渠道） | | | 高 |
| 前二十大客户（收入贡献/合作年限/合同期限/集中度趋势） | | | 高 |
| 前十大供应商及替代可得性 | | | 中 |
| 研发数据（费用率/在研管线/核心专利） | | | 中 |

**客户集中度检查**：前五大客户集中度若超过 **30%**（来自 `sources/mappings/披露阈值表_v1.csv`）须专项说明。

### Step 4 — 模块3：披露数据包

**输入**：标的公司股权结构 + 交易方案
**输出**：对照 `sources/mappings/披露阈值表_v1.csv` 的触发点检查：

| 触发点 | 阈值 | 需要的数据 | 状态 |
|--------|------|-----------|------|
| 持股首次达到 | **5%** | 标的股权结构、买方当前持股、交易后持股 | |
| 持股增减变动 | 每 **±5%** | 12个月内多次交易累计计算表 | |
| 持股20%但<30% | **20%** | 详式权益变动报告书所需：买方资金来源/后续计划 | |
| 持股30%并继续收购 | **30%** | 全体股东名单/持股比例/要约价格测算(不低于前6个月最高价) | |
| 成为第一大股东/实控人 | 任一触发 | 详式报告+财务顾问核查意见（即使持股<20%也触发） | |
| 协议收购 | ≥**5%** | 3日内报告+公告 | |

**重大资产重组认定**（来自 `sources/laws/重大资产重组管理办法_2023修订要点.md`）：

| 指标 | 阈值 | 当前值 | 是否触发 |
|------|------|--------|---------|
| 总资产 | ≥**50%** | | |
| 营业收入 | ≥**50%** | | |
| 净资产 | ≥**50%且>5000万元** | | |
| 重组上市(借壳) | 控制权变更后**36个月**内购买资产指标≥**100%** | | |

### Step 5 — 模块4：反垄断数据包

**输入**：交易各方营业额数据
**输出**（来自 `sources/laws/经营者集中申报标准.md`）：

| 数据项 | 用途 | 阈值 | 当前值 | 触发？ |
|--------|------|------|--------|--------|
| 全球合计营业额 | 方案一 | ≥ **120亿元人民币** | | |
| 至少两个经营者中国境内营业额 | 方案一/二 | 各自 ≥ **8亿元** | | |
| 中国境内合计营业额 | 方案二 | ≥ **40亿元** | | |
| 交易后市场份额 | 简易程序判断 | 通常<15% | | |
| 交易架构/控制权认定 | 经营者集中判断 | 取得控制权或决定性影响 | | |

**审查周期预估**（来自同文件）：
- 简易案件：**1-2个月**（公示期 **10个自然日**）
- 普通案件：**3-5个月**
- 复杂案件：**6-18个月**
- 法定最长：立案后 **30自然日**（第一阶段）+ **90自然日**（第二阶段）+ **60自然日**（延期）= **180自然日**

**数据不可得fallback**：若无法获取精确营业额，标注"待买方/标的提供营业额数据"，不可编造。

### Step 6 — 模块5：国资交易数据包

**输入**：标的性质（是否涉及国资）
**输出**（来自 `sources/laws/国资交易_外资准入要点.md`）：

| 数据项 | 标准 | 状态 |
|--------|------|------|
| 资产评估报告 | 须经核准/备案（国资委12号令要求必须评估），有效期 **1年** | |
| 审计报告 | 最近一期经审计财务报告 | |
| 产权交易材料 | 首次信息披露 **≥20个工作日** | |
| 价款支付方案 | 一次性付清 / 分期（首付 **≥30%**+担保） | |
| 内部决策文件 | 董事会/股东会决议、国资监管机构批复 | |

**停止条件**：评估报告超期 → 标注"评估报告已超1年有效期，须重新评估"。

### Step 7 — 模块6：外资准入数据包

**输入**：买方身份（是否外资）
**输出**（来自 `sources/laws/国资交易_外资准入要点.md`）：

- 负面清单排查表（2024年版：全国版29条/自贸区版27条）
- 安全审查自查（军工/重要农产品/能源/重大装备/基础设施/运输服务/文化/信息技术/金融服务/关键技术 + 取得实际控制权 → 审查时限 **45个工作日** 可延长）

### Step 8 — 输出数据清单

**格式**（每个模块独立表）：

| 数据项 | 状态 | 来源 | 关键性 | 法规依据 |
|--------|------|------|--------|---------|
| | 已就绪/待补充/不适用 | 审计/管理层/第三方/公开 | 高/中/低 | 对应法规 |

### Step 9 — 风险揭示与合规声明

```
⚠️ 合规声明：
- 财务数据必须基于经审计数据（CAS），不可用管理层报表替代法规要求的审计数据
- 资产评估报告有效期1年，超期须重新评估
- 经营者集中申报数据须经反垄断律师确认
- 外资准入排查须经法律顾问确认
- 本数据清单为整理辅助工具，不替代法律/财务/税务专业意见
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `cim-builder` | 数据包→CIM填充 | CIM各章节所需数据的结构化版本 |
| `buyer-list` | 买方审查数据 | 买方分类/出资能力 |
| `merger-model` | 财务数据→估值模型 | 三表历史数据 |
| `process-letter` | 审批时间预估 | 各审批环节耗时 |
| `deal-tracker` | 数据整理状态跟踪 | 各数据项就绪进度 |
