# 交易进度跟踪

为中国投行并购交易全程跟踪四大审批线进度，识别瓶颈风险，输出结构化里程碑清单。

> **重组审批**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（交易所3月+证监会15工作日+锁定期）
> **反垄断**：`sources/laws/经营者集中申报标准.md`（30+90+60=180自然日）
> **国资交易**：`sources/laws/国资交易_外资准入要点.md`（挂牌≥20工作日/评估1年）
> **披露阈值**：`sources/mappings/披露阈值表_v1.csv`（5%/20%/30%触发点）

## 操作步骤

### Step 1 — 确定交易类型和适用审批线

**输入**：交易基本信息
**输出**：审批线启用矩阵

| 审批线 | 适用条件 | 是否启用 |
|--------|---------|---------|
| 证券监管 | 涉及A股上市公司重大资产重组 | |
| 经营者集中 | 达到申报标准（120亿/40亿/8亿） | |
| 国资交易 | 标的或买方涉及国有产权 | |
| 外商投资安全审查 | 外资取得实际控制权且属敏感领域 | |

**停止条件**：无法确定是否触发某审批线 → 标注"待法律顾问确认"但继续推进其他线。

### Step 2 — 证券监管审批线

**输入**：交易是否涉及发行股份
**输出**：里程碑列表

**不涉及发行股份**：
| 步骤 | 时限 | 关键动作 |
|------|------|---------|
| 董事会审议 | — | 预案公告（交易概况/标的资产/交易方式/定价依据） |
| 股东大会审议 | 董事会后尽快 | 出席股东所持表决权 **2/3以上** 通过 |
| 实施阶段 | 每 **至少30日** | 公告实施进展 |
| 持续督导 | **≥1个完整会计年度** | 独立财务顾问持续督导 |

**发行股份购买资产**：
| 步骤 | 时限 |
|------|------|
| 董事会决议+预案 | — |
| 交易所问询（如触发） | **10个工作日** |
| 股东大会（**2/3**通过） | 董事会后尽快 |
| 交易所重组委审核 | 受理后 **3个月** 内 |
| 证监会注册 | 交易所审核意见后 **15个工作日** 内 |
| 注册批文有效期 | **12个月** |

**锁定期**：一般投资者 **12个月** / 控股股东/实控人 **36个月** / 重组上市 **24个月**（特定情况36个月）

### Step 3 — 反垄断审批线

**输入**：经营者营业额数据
**输出**：审查时间线（来自 `sources/laws/经营者集中申报标准.md`）

| 步骤 | 时限 |
|------|------|
| 申报前商谈（建议） | 1-4周 |
| 第一阶段审查 | 立案后 **30个自然日** |
| 第二阶段（如需） | + **90个自然日** |
| 延期（如需） | + **60个自然日** |
| 最长总计 | **180个自然日** |

**简易案件**：公示期 **10个自然日**，总周期约1-2个月。

### Step 4 — 国资交易审批线

**输入**：标的/买方是否涉及国有产权
**输出**：国资流程里程碑

| 步骤 | 时限 |
|------|------|
| 审计+资产评估 | 1-3个月（12号令要求必须评估） |
| 评估核准/备案 | 视国资层级 |
| 产权交易机构挂牌 | 首次信息披露 **≥20个工作日** |
| 产权交易合同公示 | **≥5个工作日** |
| 价款支付+变更 | 首付 **≥30%**（如分期） |

**外安全审查**（如触发）：**45个工作日**可延长。

### Step 5 — 披露里程碑跟踪

**输入**：买方持股变动
**输出**：触发点检查（来自 `sources/mappings/披露阈值表_v1.csv`）

| 触发点 | 时限 | 义务 |
|--------|------|------|
| 持股 **5%** | **3个交易日** | 简式权益变动报告书+公告 |
| 每 **±5%** | **3个交易日** | 公告 |
| 持股 **20%但<30%** | — | 详式报告+财务顾问核查 |
| 持股 **30%** 继续收购 | — | 强制要约（期限 **30-60日**） |

### Step 6 — 风险预警

当以下情况发生时标记"红灯"：
1. 任一审批线超时限 **15日**
2. 反垄断审查进入第二阶段
3. 国资挂牌未征集到合格受让方（降价低于评估 **90%** 须批准）
4. 外资安全审查受理
5. 评估报告即将超 **1年** 有效期
6. 注册批文即将到期（**12个月**）
7. "抢跑"风险 — 反垄断批准前交割/业务整合（处罚可达上一年度销售额 **10%以下**）

### Step 7 — 输出

```markdown
| # | 里程碑 | 审批线 | 计划 | 实际 | 状态 | 前置条件 | 风险 |
|---|--------|--------|------|------|------|---------|------|
```

### Step 8 — 合规声明

```
⚠️ 合规声明：本交易进度跟踪仅供内部项目管理参考，不构成法律意见。
各审批时限基于公开法规，实际审理周期可能因个案复杂度而延长。
"抢跑"行为可能导致严重法律后果。使用前请经法律顾问确认。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `process-letter` | 审批流程需要出具法律文件 | 流程信/审批文件草稿 |
| `merger-model` | 对价/估值影响审批 | 对价测算/业绩承诺 |
| `datapack-builder` | 审批需要申报数据 | 反垄断/国资数据包 |
