# 并购对价/财务测算

为中国投行并购交易构建对价模型和合并财务影响分析，嵌入A股独有的80%定价底线、12/36月锁定期、国资90%降价门槛、30%分期首付和业绩承诺规则。

> **重组规则**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（定价/锁定期/业绩承诺）
> **国资规则**：`sources/laws/国资交易_外资准入要点.md`（评估底线/降价门槛/支付规则）
> **反垄断**：`sources/laws/经营者集中申报标准.md`（营业额阈值）

## 操作步骤

### Step 1 — 标的估值（Standalone）

**输入**：标的近3年经审计财报
**输出**：多方法估值区间

| 方法 | 适用场景 | A股调整 |
|------|---------|---------|
| **P/E法** | 盈利稳定 | 使用**扣非归母净利润** |
| **EV/EBITDA** | 资本密集/高折旧 | 资本结构套利 |
| **P/B法** | 金融/重资产 | 国资标的须以评估净资产为基础 |
| **DCF** | 现金流可预测 | WACC须反映中国资本市场资金成本 |
| **可比交易** | — | 优先参考A股同行业控股权溢价率 |

**国资标的核心规则**（来自 `sources/laws/国资交易_外资准入要点.md`）：
- 首次挂牌价 ≥ 经核准/备案的资产评估结果
- 评估报告有效期 **1年**
- 降价挂牌 < 评估结果**90%** → 须批准单位书面同意
- 分期付款首付 ≥ **30%** + 担保

**停止条件**：审计数据不可得 → 不可用管理层报表替代法规要求的审计数据。

### Step 2 — 发行股份定价（如涉及）

**输入**：定价基准日选择 + 历史股价
**输出**（来自 `sources/laws/重大资产重组管理办法_2023修订要点.md`）：

| 参数 | 规则 |
|------|------|
| 定价基准日 | 董事会决议公告日/股东大会决议公告日/发行期首日（三选一） |
| 最低发行价 | 不低于基准日前20/60/120个交易日均价的 **80%** |
| 锁定期 | 一般投资者 **12个月** / 控股股东 **36个月** / 重组上市 **24-36个月** |

### Step 3 — 对价区间与约束检查

**输入**：Step 1+2 的估值结果
**输出**：

| 情景 | P/E(x) | 隐含100%股权价值 | 交易对价（按XX%股权） |
|------|--------|-----------------|---------------------|
| 保守 | | | |
| 基准 | | | |
| 乐观 | | | |

**法规红线约束检查**：
- [ ] 国资交易对价 ≥ 经备案评估结果？
- [ ] 发行股份价格 ≥ 基准价80%？
- [ ] 要约价格 ≥ 前6个月最高收购价？

### Step 4 — 合并模拟

**输入**：买方+标的合并财务数据
**输出**：

**EPS增厚/稀释**：
合并后EPS = (买方净利润 + 标的净利润 ± 调整项) / (买方总股本 + 新发行股数)
调整项：协同效应 + PPA折旧摊销 − 融资费用 − 非经常性损益

**商誉** = 交易对价 − 标的可辨认净资产公允价值 → 须标注减值风险（A股监管重点）。

### Step 5 — 反垄断营业额检查

**输入**：合并口径营业额
**输出**（来自 `sources/laws/经营者集中申报标准.md`）：

| 模型项 | 阈值 | 实际值 | 触发？ |
|--------|------|--------|--------|
| 全球合计营业额 | > **120亿** | | |
| 买方中国营业额 | > **8亿** | | |
| 标的中国营业额 | > **8亿** | | |
| 中国合计营业额 | > **40亿** | | |

满足方案一或方案二任一即须申报。

### Step 6 — 融资结构

| 融资方式 | 比例 | 约束 |
|---------|------|------|
| 自有资金 | | — |
| 并购贷款 | | ≤交易对价 **60%**，期限≤7年 |
| 换股 | | 适用80%定价规则 |

偿债指标：利息覆盖倍数（EBITDA/利息）、杠杆率（有息负债/EBITDA）。

### Step 7 — 业绩承诺与补偿

**输入**：购买方身份 + 估值方法
**输出**（来自 `sources/laws/重大资产重组管理办法_2023修订要点.md`）：

| 情形 | 强制？ |
|------|--------|
| 向控股股东/实控人购买 + 基于未来收益法估值 | **强制**（股份+现金补偿） |
| 向非关联方购买 | 市场自主协商 |
| 重组上市（借壳） | **强制** |

对赌期承诺净利润 vs 实际净利润 vs 补偿金额。

### Step 9 — 合规声明

```
⚠️ 合规声明：本模型仅供内部财务分析参考，不构成投资建议或估值意见。
国资标的的估值模型不能替代法定资产评估。评估报告有效期为1年，超期须重新评估。
发行股份价格低于基准价80%可能导致审核不通过。
所有假设和结论须标注资料来源和不确定性区间。
```
