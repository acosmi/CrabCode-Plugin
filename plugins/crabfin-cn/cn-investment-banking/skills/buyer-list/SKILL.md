# 买方长名单/短名单生成

为中国投行并购交易生成潜在买方名单，按战略/财务/国资/外资四类筛选并按中国披露阈值和监管规则标注触发行情。

> **买方分类规则**：`sources/laws/国资交易_外资准入要点.md`（国资32号令/负面清单/安全审查）
> **披露阈值**：`sources/mappings/披露阈值表_v1.csv`（5%/20%/30%触发点）
> **反垄断申报**：`sources/laws/经营者集中申报标准.md`（120亿/40亿/8亿标准）
> **重组认定**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（≥50%标准）

## 操作步骤

### Step 1 — 确认交易参数

**输入**：用户提供的标的信息
**输出**：以下参数必须逐项确认后才能进入下一步：

| 参数 | 必选项 | 未确认时的处理 |
|------|--------|-------------|
| 标的行业（申万分类） | 是 | 停止，请用户提供 |
| 交易类型（控股权转让/少数股权/增资/合并） | 是 | 停止 |
| 拟释放比例 | 是 | 停止 |
| 预计交易规模（亿元） | 是 | 停止 |
| 股东性质（民营/国资/外资/混合） | 是 | 停止 |
| 是否涉及外资准入限制行业 | 是 | 对照负面清单逐项排查 |

**停止条件**：以上6项任一不可得 → 向用户确认后再继续。

### Step 2 — 按买方类别生成名单

**输入**：Step 1 的全部参数
**输出**：四类买方的候选名单及合规标注

#### A. 战略投资者（产业资本）

- **筛选标准**：同行业或上下游产业链企业，具有业务协同效应
- **优势**：估值容忍度高、可提供渠道/技术/产能协同
- **关注点**：同业竞争审查、反垄断申报风险

**披露触发**（来自 `sources/mappings/披露阈值表_v1.csv`）：
| 持股比例 | 披露义务 |
|---------|---------|
| 达到 **5%** | 3个交易日内编制简式权益变动报告书，通知上市公司+公告，3个交易日内不得买卖 |
| 每 **±5%** | 3个交易日内公告 |
| 达到 **20%但<30%** | 须编制详式权益变动报告书 |
| 达到 **30%** 并继续收购 | 触发强制要约收购义务 |

#### B. 财务投资者（PE/VC/并购基金）

- **筛选标准**：有行业专长的并购基金、成长型PE、专项并购SPV
- **优势**：决策快、不寻求经营控制（部分情形）、退出路径灵活
- **关注点**：退出要求（通常3-7年）、LBO杠杆空间、管理层留任
- **披露触发**：同上

#### C. 国资买方（央企/地方国企/国有资本平台）

- **筛选标准**：国资委直管央企、地方国资委下属企业、国有资本投资/运营公司（如国新、诚通）、产业引导基金

**国资交易特殊规则**（来自 `sources/laws/国资交易_外资准入要点.md`）：
| 规则 | 数值 |
|------|------|
| 国有产权进场挂牌 | 首次信息披露 **≥20个工作日** |
| 定价底线 | 不低于经核准/备案的评估结果 |
| 降价挂牌 | 低于评估结果 **90%** 须批准单位书面同意 |
| 产权交易合同公示 | **≥5个工作日** |
| 价款支付 | 一次性付清；分期须首付 **≥30%** + 担保 |
| 国有单位间划转 | 适用专门规定，可不进场交易 |

#### D. 外资买方（境外产业/PE/主权基金）

- **筛选标准**：亚太区有布局的跨国企业、全球PE（KKR/黑石/凯雷等）、主权基金（淡马锡/GIC/ADIA等）

**准入红线**（来自 `sources/laws/国资交易_外资准入要点.md` 负面清单）：
- **禁止外商投资**：稀土/放射性矿产/钨矿勘查开采、邮政/国内信件快递、新闻机构、广播电视播出/传输、互联网新闻信息服务/网络出版/视听节目/文化经营、义务教育/宗教教育
- **限制外商投资**：农作物新品种选育/种子生产（中方持股 **≥34%**）、增值电信业务（外资股比 **≤50%**）
- **安全审查**：投资军工/重要农产品/能源资源/重大装备/基础设施/运输服务/文化/信息技术/金融服务/关键技术且取得实际控制权的，须主动申报（审查时限 **45个工作日**，可延长）

**反垄断申报**（来自 `sources/laws/经营者集中申报标准.md`）：
- 方案一：全球合计营业额 **>120亿元** 且至少两个经营者中国境内各自 **>8亿元**
- 方案二：中国境内合计营业额 **>40亿元** 且各自 **>8亿元**

**数据不可得fallback**：若无法获取买方营业额数据，标注"反垄断申报标准待买方提供营业额数据后确认"。

### Step 3 — 长名单打分

**输入**：Step 2 的候选买方列表
**输出**：五维打分卡

| 维度 | 权重 | 说明 |
|------|------|------|
| 战略契合度 | 25% | 业务协同、技术互补 |
| 资金实力 | 20% | 自有资金+融资能力 |
| 交易确定性 | 20% | 决策链条长短、审批风险 |
| 估值预期 | 15% | 历史交易倍数区间 |
| 交割速度 | 10% | 内部决策+监管审批时间 |
| 管理层兼容性 | 10% | 投后管理风格、留任安排 |

### Step 4 — 短名单筛选

**输入**：Step 3 的评分排序
**输出**：三级分组

| 分组 | 标准 | 数量 |
|------|------|------|
| **Tier 1**（优先接触） | 评分前5，战略契合度高且交易确定性强 | 5家 |
| **Tier 2**（次选接触） | 评分6-10，具备资金实力但需进一步磨合 | 5家 |
| **Tier 3**（备选） | 剩余高分买方，用于竞争性报价场景 | 视情况 |

### Step 5 — 输出买方名单

**输出格式**（每买方一行，末尾标注合规触发点）：

```
| 买方名称 | 类别 | 战略 | 资金 | 确定性 | 估值 | 速度 | 管理 | 综合 | 优先级 |
|----------|------|------|------|--------|------|------|------|------|-------|
| [名称]   | A/B/C/D | 4 | 5 | 3 | 4 | 3 | 4 | 3.85 | Tier1 |
```

每买方附注：
- **披露触发点**：持股5%/20%/30%时对应的义务
- **审批要求**：国资审批/外资安全审查/经营者集中申报
- **接触策略**：直接引荐/中介接触/管理层路演

### Step 6 — 风险揭示与合规声明

```
⚠️ 免责声明：本买方名单仅供内部交易策划参考，不构成要约邀请或出售要约。
买方信息基于公开资料整理，未经独立验证。买方筛选标准和评分含有主观判断。

合规警示：
- 涉及国资标的的，定价必须以经核准/备案的评估结果为基础
- 外资买方须逐项排查负面清单和安全审查
- 买方名单最终须经法律顾问确认
- 严禁在签署NDA前向潜在买方披露标的非公开信息
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `sector-overview` | 确定标的行业定位 | 行业分类/竞争格局 |
| `process-letter` | 买方接触后需要流程把控 | 交易流程时间表 |
| `deal-tracker` | 买方名单确定后跟踪进展 | 各买方接触状态 |
| `strip-profile` | 需要了解公司股权披露义务 | 股东信息披露要求 |
