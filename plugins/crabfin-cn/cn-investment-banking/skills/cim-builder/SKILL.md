# 投资备忘录(CIM)编制

为中国投行并购交易编制保密信息备忘录(CIM)，按12章标准结构生成，嵌入中国特有的国资评估、披露规则、重组问询关注点。

> **CIM模板**：`sources/templates/CIM_脱敏模板.md`（12章+6附录完整结构，严格遵循）
> **重组规则**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（认定标准/锁定期/发行定价）
> **国资规则**：`sources/laws/国资交易_外资准入要点.md`（32号令进场/评估1年有效期）
> **反垄断**：`sources/laws/经营者集中申报标准.md`（120亿/40亿/8亿审查标准）
> **披露阈值**：`sources/mappings/披露阈值表_v1.csv`

## 操作步骤

### Step 1 — 执行摘要编制

**输入**：公司基本信息 + 交易框架
**输出**：按 `sources/templates/CIM_脱敏模板.md` 第1章结构：

- 公司定位（行业地位/成立时间/总部）
- 交易机会描述（交易类型/拟释放比例/预计规模）
- 核心财务表现（营业收入/EBITDA/归母净利润+YoY）
- 3-5句话投资主题小结

**必须包含**（来自 `sources/laws/重大资产重组管理办法_2023修订要点.md`）：预案阶段须披露交易概况、标的资产基本情况、交易方式、定价依据及合理性。

**停止条件**：无法获取公司核心财务数据 → 标注"数据待管理层提供"。

### Step 2 — 公司概况与股权结构

**输入**：工商登记信息 + 股东名册
**输出**（按模板第3章）：

- 基本情况表：公司全称/统一社会信用代码/注册资本/实缴资本/法定代表人/成立日期/注册地址/经营范围
- 股权结构表：

| 股东 | 持股比例 | 股东性质 | 备注 |
|------|---------|---------|------|
| | | 自然人/法人/国资/外资 | |

**合规检查**：
- 涉及国有股东 → 标注是否属于"国有及国有控股企业、国有实际控制企业"（32号令适用范围）
- 涉及外资股东 → 标注是否符合外商投资负面清单

### Step 3 — 行业与竞争格局

**输入**：行业研究数据 → 调用 `sector-overview`
**输出**（按模板第4章）：行业定义/规模/CAGR/细分拆解/趋势/竞争对标/壁垒分析（技术/客户/牌照/规模，可持续性强/中/弱）。

### Step 4 — 财务回顾与分析

**输入**：近3年经审计财报（CAS编制）
**输出**（按模板第6章）：

- 损益表摘要（3年，含YoY/毛利率/营业利润率/净利率）
- EBITDA调整（剔除非经常性损益）
- 资产负债表核心指标（资产负债率/有息负债率）
- 营运指标（应收/存货/应付周转天数、CCC、ROE、ROA、经营现金流/净利润、CAPEX）
- 收入质量分析（非经常性损益/关联交易/收入确认政策）

**数据不可得fallback**：未经审计的数据须标注"管理层报表，未经审计"。

### Step 5 — 法律与合规（敏感度最高章节）

**输入**：尽调发现 + 法规对照
**输出**（按模板第10章）：

#### 5.1 国资标的额外要求

| 检查项 | 标准 | 状态 |
|--------|------|------|
| 审计+资产评估完成 | 国资委12号令要求必须评估，不得以估值替代 | |
| 评估报告有效期 | **1年**（自评估基准日起） | |
| 进场交易程序 | 首次挂牌 **≥20个工作日** | |
| 定价底线 | 不低于经核准/备案的评估结果 | |

#### 5.2 股份发行检查（如涉及）

| 检查项 | 标准 |
|--------|------|
| 定价基准日选定 | 董事会决议公告日/股东大会决议公告日/发行期首日 三选一 |
| 发行定价 | 不低于基准日前20/60/120个交易日均价的 **80%** |

#### 5.3 重大资产重组认定（如涉及）

| 指标 | 阈值 | 当前值 | 是否触发 |
|------|------|--------|---------|
| 总资产 | 标的资产总额/上市公司总资产 **≥50%** | | |
| 营业收入 | 标的营收/上市公司营收 **≥50%** | | |
| 净资产 | 标的净资产/上市公司净资产 **≥50%且>5000万元** | | |
| 重组上市(借壳) | 控制权变更后 **36个月** 内购买资产指标 **≥100%**，或主营业务根本变化 | | |

#### 5.4 其他法律事项

- 未决诉讼/仲裁（标的金额/案由/进度/影响评估）
- 行政处罚（金额/原因/已整改否）
- 关联担保/对外担保

### Step 6 — 风险因素

**输入**：尽调发现
**输出**（按模板第11章，每风险含严重程度+缓释措施）：

| 风险类别 | 严重程度 | 缓释措施 |
|----------|---------|---------|
| 市场风险 | 高/中/低 | |
| 客户集中度风险（前5大>30%须专项） | 高/中/低 | |
| 核心人员流失风险 | 高/中/低 | |
| 原材料/供应链风险 | 高/中/低 | |
| 政策与监管风险 | 高/中/低 | |
| 技术迭代风险 | 高/中/低 | |

**反垄断风险专项**（如触发申报标准）：标注经营者集中审查风险和预估审查周期 — 简易1-2个月/普通3-5个月/复杂6-18个月。

### Step 7 — 交易信息与流程

**输入**：交易框架设计
**输出**（按模板第12章）：

- 交易框架表（标的/类型/尽调状态/预计交割）
- 交割条件清单：反垄断审批/外资安全审查（**45个工作日**可延长）/股东会批准/第三方同意/MAE
- 交易流程时间表：首轮IoI → 尽调开放 → 二轮报价 → 排他谈判 → 交割

**管理层锁定期**（来自重组办法）：控股股东/实控人认购股份锁定期 **36个月**，一般投资者 **12个月**。

### Step 8 — 风险揭示与合规声明

```
⚠️ 合规声明：
- CIM中的财务数据须基于经审计数据（中国企业会计准则CAS编制）
- 涉及国有资产的，定价必须以经核准/备案的评估结果为法定基础，不得以估值替代
- 法律章节标注"供参考——须人工复核"水印，最终版本须经法律顾问审核
- CIM首页须包含完整保密声明（参照模板声明格式）
- 重大资产重组涉及的重组报告书、独立财务顾问报告、法律意见书、审计/评估报告
  须满足重组办法对草案阶段的披露要求

⚠️ 免责声明：本CIM仅供获得事先书面同意的潜在投资方评估交易之目的而审阅。
信息部分来自管理层及公开来源，未经独立验证。不构成出售要约或招揽购买要约。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `sector-overview` | 第4章行业分析 | 行业规模/CAGR/竞争格局 |
| `buyer-list` | 交易框架设计时 | 潜在买方名单 |
| `datapack-builder` | 附录数据的系统化整理 | 财务/业务/披露数据包 |
| `process-letter` | 交易流程时间表 | 监管审批时间预估 |
| `deal-tracker` | CIM分发后的进展跟踪 | 买方反馈/尽调状态 |
