# 标的简况

为中国投行并购交易内部筛选或初步买方接触制作标的公司简况（1-2页），聚焦股权结构披露阈值、32号令国资属性和负面清单准入结论。

> **披露阈值**：`sources/mappings/披露阈值表_v1.csv`（5%/20%/30%触发点）
> **国资规则**：`sources/laws/国资交易_外资准入要点.md`（32号令/负面清单/安全审查）
> **重组规则**：`sources/laws/重大资产重组管理办法_2023修订要点.md`（≥50%认定标准）

## 操作步骤

### Step 1 — 公司基本信息卡

**输入**：工商信息 + 年报
**输出**：

| 项目 | 内容 |
|------|------|
| 公司全称 | |
| 统一社会信用代码 | |
| 注册资本/实缴资本（万元） | |
| 成立日期/注册地址 | |
| 行业分类（申万） | |
| 上市情况 | |
| 员工人数 | |

**停止条件**：无法获取工商信息 → 标注"数据来源于公开信息"，不编造。

### Step 2 — 股权结构分析 ⚠️ 核心

**输入**：股东名册
**输出**：

| 股东 | 持股% | 性质 | 实控人 | 国资(32号令) | 外资 |
|------|-------|------|--------|-------------|------|

**持股触发点分析**（来自 `sources/mappings/披露阈值表_v1.csv`）：

| 触发阈值 | 是否触发 | 说明 |
|----------|---------|------|
| 持股 ≥ **5%** | | 简式权益变动报告书义务 |
| 持股 ≥ **20%但<30%** | | 详式报告+财务顾问核查 |
| 持股 ≥ **30%** | | 继续收购触发强制要约 |
| 实控人/第一大股东 | | 即使<20%也触发详式报告 |

**国资属性判断**（来自 `sources/laws/国资交易_外资准入要点.md`）：
- 属于32号令适用范围？→ 须进场交易（首次挂牌 **≥20个工作日**）或适用非公开协议转让
- 须资产评估（12号令强制，有效期 **1年**）

**外资属性判断**：
- 行业在负面清单？→ "禁止"或"限制"（中方≥34%/外资≤50%等）
- 须安全审查？→ **45个工作日**可延长

### Step 3 — 财务摘要（近2年）

**输入**：经审计财报（CAS编制）
**输出**：营收/毛利/EBITDA/净利/总资产/净资产/资产负债率/经营现金流（含YoY）

**数据不可得fallback**：标注"管理层报表，未经审计"。

### Step 4 — 收入拆分 + 行业竞争

**输入**：分部数据 + 竞对信息
**输出**：业务板块收入占比/毛利占比/YoY + 竞争对标表

**客户集中度**：前五大>**30%** 须专项标注。

### Step 5 — 交易合规性初步筛查

| 检查项 | 阈值 | 状态 |
|--------|------|------|
| 重大资产重组 | 总资产≥**50%** / 营收≥**50%** / 净资产≥**50%且>5000万** | |
| 重组上市 | 控制权变更后**36个月**+指标≥**100%** | |
| 经营者集中 | 全球>**120亿**+各自>**8亿** / 中国>**40亿**+各自>**8亿** | |
| 国资交易 | 32号令适用范围 | |
| 外资准入 | 负面清单限制/禁止 | |
| 外资安全审查 | 敏感领域+实际控制权 | |
| 强制要约 | 持股≥**30%**继续收购 | |

### Step 6 — 初步交易适用性评估

| 买方类型 | 可行性 | 关注事项 |
|---------|--------|---------|
| 战略投资者 | 高/中/低 | 同业竞争/反垄断 |
| 财务投资者(PE) | 高/中/低 | 退出路径/杠杆空间 |
| 国资买方 | 高/中/低 | 进场交易/评估备案 |
| 外资买方 | 高/中/低 | 负面清单/安全审查 |

### Step 7 — 合规声明

```
⚠️ 合规声明：本Strip Profile仅供内部筛选和初步评估使用，对外分发前须脱敏处理。
涉及国资标的的，股权结构和控制路径须经法律核实。
涉及上市公司的，须遵守内幕信息管理规定。
在签署NDA前，仅可提供脱敏Teaser级别信息。持股触及5%即触发公开披露义务。
```
