## 一、企业国有资产交易(32号令核心要点)

### 1.1 适用范围
"国有及国有控股企业、国有实际控制企业"以下行为均受 32 号令管辖：
1. **产权转让**——履行出资人职责的机构、国有及国有控股企业、国有实际控制企业转让其对企业各种形式出资所形成权益的行为
2. **增资**——国有及国有控股企业、国有实际控制企业增加资本的行为(政府以增加资本金方式对国家出资企业的投入除外)
3. **资产转让**——国有及国有控股企业、国有实际控制企业的重大资产转让

### 1.2 国有产权转让核心流程

```
内部决策(董事会/股东会)
  → 可行性研究与方案论证
  → 审计 + 资产评估(必须!以经核准/备案的评估结果为定价基础)
  → 产权交易机构挂牌(首次正式信息披露 ≥ 20 个工作日)
  → 征集意向受让方
  → 确定受让方(竞价/协议转让须经批准)
  → 签订产权交易合同(≥ 5 个工作日公示)
  → 价款支付(原则上一次性付清;分期须首付 ≥ 30% + 提供担保)
  → 出具交易凭证 / 办理变更登记
```

### 1.3 定价核心规则

| 事项 | 规则 |
|---|---|
| **首次挂牌价** | 不得低于经核准/备案的评估结果 |
| **降价挂牌** | 未征集到意向受让方,可在此前挂牌价基础上降低,新挂牌价如**低于评估结果 90%** 的,须经批准单位书面同意 |
| **信息披露** | 首次信息公告期 **≥ 20 个工作日**(可在产权交易机构网站公告) |

### 1.4 可非公开协议转让的情形(不挂牌)

仅限以下特殊情况：
- 涉及主业处于关系国家安全、国民经济命脉的**重要行业和关键领域**
- 同一国有控股企业或国有实际控制企业**内部重组整合**
- 经国资监管机构批准的其他情形

## 二、国有企业资产评估(国资委 12 号令要点)

### 2.1 必须评估的情形
国有企业在发生下列事项时,**必须进行资产评估**(不得以估值替代)：
- 整体或部分改建为有限责任公司或股份有限公司
- 以非货币资产对外投资
- 合并/分立/清算
- 非上市公司国有股东股权比例变动
- 产权转让/资产转让/资产置换
- 整体资产或部分资产租赁/承包给非国有单位

### 2.2 评估结果核准/备案
- 经**核准或备案**的资产评估结果是确定交易价格的**法定基础**
- 评估报告有效期：自评估基准日起 **1 年**

## 三、外资准入：外商投资负面清单(2024年版)

### 3.1 负面清单概览
2024 版负面清单分为：

| 类别 | 条目数 | 核心限制逻辑 |
|---|---|---|
| **全国外资准入负面清单** | 29 条 | 清单之外的行业,内资外资一视同仁(国民待遇) |
| **自贸试验区负面清单** | 27 条 | 较全国版更短 |

### 3.2 禁止外商投资领域(重点)

| 行业 | 限制内容 |
|---|---|
| 稀土/放射性矿产/钨矿等 | **禁止**外商投资勘查开采 |
| 邮政/国内信件快递 | **禁止**外商投资 |
| 新闻机构(通讯社/报纸出版等) | **禁止**外商投资 |
| 广播电视播出/传输 | **禁止**外商投资 |
| 互联网新闻信息服务/网络出版/视听节目/文化经营 | **禁止**外商投资(个别除外) |
| 义务教育/宗教教育 | **禁止**外商投资 |

### 3.3 限制外商投资领域(须中方控股或特定条件)

| 行业 | 限制内容 |
|---|---|
| 农作物新品种选育/种子生产 | 中方**持股 ≥ 34%** |
| 汽车整车制造 | 取消外资股比限制(2022年已全部放开),保留**境外投资者投资准入不超过 2 家合资企业**的限制(逐步放开) |
| 证券公司/基金公司 | 外资股比放开(2020年后取消限制),但仍须符合**股东资质条件** |
| 增值电信业务 | 外资股比不超过 **50%**(部分放开) |

### 3.4 外商投资安全审查
2020 年《外商投资安全审查办法》建立独立审查机制：

| 须主动申报的情形 | 审查范围 |
|---|---|
| 投资**军工**及军工配套领域 | 在所投资企业中所持股权或控制权 |
| 投资**关系国家安全的重要农产品/能源资源/重大装备制造/基础设施/运输服务/文化/信息技术/金融服务/关键技术**等领域,并取得所投资企业的**实际控制权** | 取得实际控制权 |

> 安全审查决定分为三类：通过 / 附条件通过 / 禁止。安全审查决定为最终决定,不可复议。

## 四、投行实务提示

### 4.1 国资并购时间预算
- 审计+评估:1-3 个月
- 内部决策+审批:1-2 个月
- 挂牌+成交:1-3 个月
- 反垄断审查(如需):1-3 个月(简易)
- 外资安全审查(如需):45 个工作日(可延长)
- **总时间预算**：至少 4-8 个月(不含交易谈判)

### 4.2 转持划转/无偿划转
国有产权在国有单位之间**无偿划转**的,适用专门规定,可不进场交易,但仍须履行审计和国资监管部门审批程序。

### 4.3 跨境并购中的国资要素
- 中国企业海外并购如涉及国有资产的,须关注《中央企业境外投资监督管理办法》
- 境外反垄断 + 中国外资安全审查可能双向触发
- ODI(对外直接投资)须在发改委/商务部/外管局完成备案或核准
