# 《上市公司重大资产重组管理办法》2023年修订要点

## 文件信息
- **文件全名**：《上市公司重大资产重组管理办法》(证监会令第214号)
- **发布日期**：2023年2月17日(修订发布)
- **生效日期**：2023年2月17日
- **发布机构**：中国证券监督管理委员会(CSRC)
- **历史沿革**：2008年首发 → 2014年全面修订 → 2016、2019、2020、2023年修订
- **出处链接**：http://www.csrc.gov.cn/csrc/c101953/c7121862/content.shtml

## 一、重大资产重组认定标准(第十二条)

购买/出售资产达到以下任一标准，构成重大资产重组：

| 指标 | 阈值 | 计算基准 |
|------|------|----------|
| **总资产** | ≥50% | 标的资产总额 / 上市公司最近一期经审计合并报表期末资产总额 |
| **营业收入** | ≥50% | 标的最近会计年度营业收入 / 上市公司同期营业收入 |
| **净资产** | ≥50% 且 >5000万元 | 标的资产净额 / 上市公司最近一期经审计合并报表期末净资产额 |

> **多次交易判断**：12个月内对同一或相关资产分期交易的，累计计算。

### 重组上市(借壳)认定标准(第十三条)

同时满足以下条件构成重组上市(等同IPO审核)：

1. 上市公司控制权发生变更(自变更之日起**36个月**内)
2. 向收购人及其关联人购买资产的任一指标 ≥ **100%**
3. 或购买资产导致主营业务发生根本变化

## 二、审核流程(根据重组类型分三类)

### 1. 不涉及发行股份 → 无需证监会审核

- 仅需董事会/股东大会审议 + 信息披露
- 标的如为国有资产需额外国资审批

### 2. 发行股份购买资产 → 交易所审核 + 证监会注册

| 步骤 | 主体 | 时限 |
|------|------|------|
| 董事会决议 + 预案公告 | 上市公司 | — |
| 交易所问询(如触发) | 交易所 | 10个工作日 |
| 股东大会审议(2/3通过) | 上市公司 | 董事会后尽快 |
| **交易所重组委审核** | 交易所 | 受理后3个月内 |
| **证监会注册** | 证监会 | 交易所审核意见后15个工作日内 |
| 注册后实施 | 上市公司 | 12个月内有效 |

### 3. 重组上市(借壳) → 严格IPO等同审核

- 除上述流程外，还需满足IPO发行条件
- 交易所重组委审核 + 证监会注册(等同于IPO程序)

## 三、信息披露核心要点

| 阶段 | 必须披露内容 |
|------|-------------|
| **预案阶段** | 交易概况、标的资产基本情况、交易方式、定价依据及合理性 |
| **草案阶段** | 完整重组报告书 + 独立财务顾问报告 + 法律意见书 + 审计/评估报告 |
| **实施阶段** | 实施进展公告(至少每30日)、过户公告、股本变动公告 |
| **持续督导** | 独立财务顾问持续督导 ≥1个完整会计年度 |

---

## 四、发行股份购买资产特别规定

| 项目 | 要求 |
|------|------|
| **定价基准日** | 董事会决议公告日/股东大会决议公告日/发行期首日(三选一) |
| **股份定价** | 不低于定价基准日前20/60/120个交易日均价 **80%** |
| **锁定期** | 一般12个月；控股股东/实际控制人及其控制人认购的36个月 |
| **重组上市** | 存量老股及新股锁定期24个月(特定情况36个月) |

---

## 五、业绩承诺与补偿

| 情形 | 是否强制业绩承诺 |
|------|:---:|
| 向控股股东/实控人购买资产,且基于未来收益预期估值 | **强制** |
| 向非关联方购买资产 | 市场自主协商 |
| 重组上市(借壳) | **强制**(股份+现金补偿) |

---

## 六、2023年修订要点总结

1. **全面对接注册制**：审核主体从证监会发审委调整为交易所重组委,证监会职能转为"注册制"
2. **完善认定标准**：明确"主营业务根本变化"等定量+定性标准
3. **强化信息披露**：由"事前审批"转向"事中事后监管"
4. **优化定价机制**：定价基准日三选一,定价不低于80%
5. **加强投资者保护**：业绩承诺、锁定期、独立董事发表意见等要求

---

> **红线提醒：** 以上要点为依据证监会令第214号整理。实务操作中需结合沪深北交易所各自的重组审核规则(上交所上市规则/深交所上市规则/北交所上市规则)及配套信息披露准则使用。具体法律适用以官方原文和律师意见为准,不可机械套用。
