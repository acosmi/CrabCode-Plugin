## 一、重大资产重组认定标准(第十二条)

购买/出售资产达到以下任一标准，构成重大资产重组：

| 指标 | 阈值 | 计算基准 |
|------|------|----------|
| **总资产** | ≥50% | 标的资产总额 / 上市公司最近一期经审计合并报表期末资产总额 |
| **营业收入** | ≥50% | 标的最近会计年度营业收入 / 上市公司同期营业收入 |
| **净资产** | ≥50% 且 >5000万元 | 标的资产净额 / 上市公司最近一期经审计合并报表期末净资产额 |

> **多次交易判断**：12个月内对同一或相关资产分期交易的，累计计算。

### 重组上市(借壳)认定标准(第十三条)

同时满足以下条件构成重组上市(等同IPO审核)：

1. 上市公司控制权发生变更(自变更之日起**36个月**内)
2. 向收购人及其关联人购买资产的任一指标 ≥ **100%**
3. 或购买资产导致主营业务发生根本变化

## 二、审核流程(根据重组类型分三类)

### 1. 不涉及发行股份 → 无需证监会审核

- 仅需董事会/股东大会审议 + 信息披露
- 标的如为国有资产需额外国资审批

### 2. 发行股份购买资产 → 交易所审核 + 证监会注册

| 步骤 | 主体 | 时限 |
|------|------|------|
| 董事会决议 + 预案公告 | 上市公司 | — |
| 交易所问询(如触发) | 交易所 | 10个工作日 |
| 股东大会审议(2/3通过) | 上市公司 | 董事会后尽快 |
| **交易所重组委审核** | 交易所 | 受理后3个月内 |
| **证监会注册(常规程序)** | 证监会 | 交易所审核意见后15个工作日内 |
| **证监会注册(简易审核程序)** | 证监会 | 5个工作日内(2025年第230号新增) |
| 注册后实施 | 上市公司 | 12个月内有效 |

### 3. 重组上市(借壳) → 严格IPO等同审核

- 除上述流程外，还需满足IPO发行条件
- 交易所重组委审核 + 证监会注册(等同于IPO程序)

## 三、信息披露核心要点

| 阶段 | 必须披露内容 |
|------|-------------|
| **预案阶段** | 交易概况、标的资产基本情况、交易方式、定价依据及合理性 |
| **草案阶段** | 完整重组报告书 + 独立财务顾问报告 + 法律意见书 + 审计/评估报告 |
| **实施阶段** | 实施进展公告(至少每30日)、过户公告、股本变动公告 |
| **持续督导** | 独立财务顾问持续督导 ≥1个完整会计年度 |

---

## 四、发行股份购买资产特别规定

| 项目 | 要求 |
|------|------|
| **定价基准日** | 董事会决议公告日/股东大会决议公告日/发行期首日(三选一) |
| **股份定价** | 不低于定价基准日前20/60/120个交易日均价 **80%** |
| **锁定期** | 一般12个月；控股股东/实际控制人及其控制人认购的36个月 |
| **重组上市** | 存量老股及新股锁定期24个月(特定情况36个月) |

---

## 五、业绩承诺与补偿

| 情形 | 是否强制业绩承诺 |
|------|:---:|
| 向控股股东/实控人购买资产,且基于未来收益预期估值 | **强制** |
| 向非关联方购买资产 | 市场自主协商 |
| 重组上市(借壳) | **强制**(股份+现金补偿) |

---

## 六、2025年修订(第230号)要点总结

1. **建立重组股份对价分期支付机制**：申请一次注册、分期发行股份购买资产的,注册决定有效期延长至48个月
2. **提高监管包容度**：对财务状况变化、同业竞争和关联交易的监管包容度提高
3. **新设重组简易审核程序**：符合条件者证监会5个工作日内作出注册决定
4. **明确吸收合并锁定期**：上市公司之间吸收合并的锁定期要求
5. **鼓励私募基金参与**：支持私募基金参与上市公司并购重组
6. **适应新《公司法》**：根据2024-07-01施行的新《公司法》对有关条文表述作适应性调整

> 历史背景(2023年修订要点,供沿革参考):全面对接注册制(审核主体调整为交易所重组委)、完善"主营业务根本变化"认定、定价基准日三选一/不低于80%、业绩承诺与锁定期等投资者保护。
