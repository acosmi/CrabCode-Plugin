# 催化剂日历 — A股/港股事件追踪

为A股/港股构建事件驱动日历，覆盖定期报告、业绩预告、业绩快报、股东大会、分红、限售解禁、指数调整、ST/退市等中国市场特有的催化剂事件。

> **核心素材**：`sources/laws/沪深北港交所信息披露规则要点.md`（所有披露时限与量化触发标准）
> ⚠️ 红线：不可套用美股 earnings season 概念。A股有独立的业绩预告触发体系，港股不强制季度报告。

## 操作步骤

### Step 1 — 定期报告披露日程

**输入**：公司代码/简称 + 财年
**输出**：根据 `sources/laws/沪深北港交所信息披露规则要点.md` 的时限规则推算：

| 报告类型 | 披露截止 | 审计要求 | 适用板块 |
|---------|---------|---------|---------|
| 年报 | 4月30日前 | 必须审计 | 沪/深/北/港 |
| 一季报 | 4月30日前 | 无需审计 | 沪/深/北 |
| 半年报 | 8月31日前 | 一般无需审计（除非拟分红/转增或另被要求） | 沪/深/北/港 |
| 三季报 | 10月31日前 | 无需审计 | 沪/深/北 |

**港股特别说明**：港股年报须在 **3个月** 内披露（非4个月）；港股不强制季度报告（创业板已取消）。

**数据不可得fallback**：若公司尚未公布具体日期，按法定期限的最晚日期标注，并注明"法定期限截止日，实际日期以公司公告为准"。

### Step 2 — 业绩预告触发判断

**输入**：公司当前预计的业绩状况
**输出**：对照 `sources/laws/沪深北港交所信息披露规则要点.md` 第二章的五大触发条件：

| 触发条件 | 判定标准 | 是否触发 | 预告时限 |
|---------|---------|---------|---------|
| 净利润为负 | 归母净利润 < 0 | | 年报：1月31日前 / 半年报：7月15日前 |
| 扭亏为盈 | 上年同期亏损 → 本期盈利 | | 同上 |
| 净利润增减 ≥ 50% | YoY变动 ≥ ±50% | | 同上 |
| 退市警示 | 扣非前后孰低为负，且扣除后营收 < 1亿 | | 同上 |
| 净资产为负 | 期末净资产 < 0 | | 同上 |

**停止条件**：无法获取公司财务数据 → 标注"待数据确认"。

### Step 3 — 业绩快报与修正检查

**输入**：公司已发布预告/快报
**输出**：
- 是否需发布快报：定期报告公告前出现**业绩泄露** → 须快报
- 修正检查：快报与实际数据差异 **> 20%** → 须修正公告
- 差异方向改变（如盈利变亏损）→ 立即修正

### Step 4 — 其他催化剂事件梳理

**输入**：公司公告历史 + 股权结构
**输出**：

| 事件类型 | 中国口径要点 | 日历标注 |
|---------|------------|---------|
| 股东大会 | 年度股东会须在 **6月30日前** 召开，**至少20天前**公告 | 标注预计日期 |
| 分红 | 现金分红/股价 = 股息率；关注**除权除息日(XD)** | 标注除权除息日 |
| 限售解禁 | IPO/定增/股权激励 锁定期12-36个月不等 | 标注解禁日期+股数 |
| 指数调整 | 沪深300/中证500每半年调整(**6月/12月**) | 标注调整生效日 |
| ST/退市 | *ST公司年报披露后触发暂停上市/退市流程 | 高亮警示 |

**港股特有事件**：港股内幕消息须在**知悉后尽快**披露（《证券及期货条例》第XIVA部），不设固定日期间隔。

### Step 5 — 输出催化剂日历

**输出模板**：

```markdown
# [股票简称](代码) — 催化剂日历(YYYY/MM月起)

| 日期 | 事件类型 | 内容 | 影响方向 | 置信度 |
|------|---------|------|---------|--------|
| 2025-01-31 | 业绩预告 | 2024年年报预告 | 待定 | 高 |
| 2025-03-25 | 年报披露 | 2024年年报正式发布 | — | 高 |
| 2025-06-20 | 限售解禁 | IPO战投解禁X万股 | 偏负面 | 高 |
```

每项事件附 1-2 句影响分析。

**置信度标注**：
- **高**：法定期限/公司已公告日期
- **中**：基于历史惯例推算
- **低**：基于行业惯例/主观判断

### Step 6 — 风险揭示

```
⚠️ 免责声明：本催化剂日历仅供研究参考，不构成投资建议。事件日期基于公开信息推算，
实际日期以公司公告为准。业绩预告触发条件基于中国证监会/交易所现行规则，
规则修订可能导致触发标准变化。

信息来源：
1. 上交所(www.sse.com.cn) / 深交所(www.szse.cn) / 北交所(www.bse.cn) 公告
2. 巨潮资讯网(www.cninfo.com.cn) — A股法定披露平台
3. HKEXnews(www.hkexnews.hk) — 港股法定披露平台
```
