# 行业概览 — 申万行业分类驱动

基于申万行业分类体系（28个一级行业/100+二级行业/200+三级行业），对A股/港股特定行业进行结构化分析，输出包含估值水位、竞争格局、政策催化的完整行业研判报告。

> **核心素材**：`sources/mappings/申万行业分类_2021.csv`（行业分层） / `sources/benchmarks/A股港股估值惯例与基准倍数.md`（估值基准）
> **协作技能**：需要个股深度时调用 `initiating-coverage` / `earnings-analysis`；政策窗口跟踪时调用 `catalyst-calendar`

## 操作步骤

### Step 1 — 行业定位与分类确认

**输入**：用户提及的行业名称/关键词
**输出**：`申万一级 | 申万二级 | 申万三级 | 证监会门类 | 成分股数量` 表格

从 `sources/mappings/申万行业分类_2021.csv` 中匹配行业归属。若用户指定的行业名与申万体系不直接对应，做模糊匹配并提示映射关系。

**申万一级行业代码速查**（高频行业）：
- 801730 电力设备（光伏/锂电/风电）| 801780 电子（半导体/消费电子）
- 801120 食品饮料（白酒/调味品）| 801150 医药生物（创新药/CXO/器械）
- 801750 计算机（信创/AI应用）| 801220 银行（高股息防御）
- 801230 非银金融（券商/保险）| 801180 房地产

**停止条件**：无法匹配到申万三级行业 → 告知用户并请其提供更具体的行业描述。

### Step 2 — 产业链图谱

**输入**：Step 1 确定的行业定位
**输出**：`上游(原材料/核心部件) → 中游(制造/组装) → 下游(应用/终端)` 链式图，各环节标注A股代表性公司及市值

**数据不可得fallback**：若无法识别具体公司，标注产业链环节和典型公司类型（如"电池片制造环节，代表企业类型：通威/隆基/爱旭等"），不编造具体市值数据。

### Step 3 — 市场规模与增长

**输入**：Step 2 的产业链范围
**输出**：

| 维度 | 数据 | 来源 |
|------|------|------|
| 市场规模(亿元) | | 行业协会/统计局 |
| 近3年CAGR | | |
| 未来3年预期CAGR | | 券商一致预期 |

**数据不可得fallback**：标注"暂无公开数据"并说明所需数据出处；不可编造行业规模数字。

### Step 4 — 竞争格局与壁垒

**输入**：Step 2 的产业链各环节
**输出**：

| 梯队 | 公司(代码) | 份额 | 核心壁垒 |
|------|-----------|------|---------|
| 第一梯队 | | | |
| 第二梯队 | | | |

分析壁垒时需区分：牌照壁垒（金融/电信）、规模壁垒（制造业）、技术壁垒（半导体/医药）、品牌壁垒（消费品）。

### Step 5 — 行业估值水位 ⚠️ 核心步骤

**输入**：Step 1 的申万一级行业代码
**输出**：

| 指标 | 当前值 | 5年分位 | 历史中枢 | 评级 |
|------|--------|---------|---------|------|
| PE-TTM中位数 | | | | |
| PB中位数 | | | | |

**估值分位数判断标准**（引用自 `sources/benchmarks/A股港股估值惯例与基准倍数.md`）：

| 水位 | 分位范围 | 含义 | 输出口径 |
|------|---------|------|---------|
| 低估 | < 25% | 行业PE处于近5年低点 | "历史低位，关注政策/盈利拐点" |
| 合理偏低 | 25%-40% | 略低于中枢 | "估值有一定安全边际" |
| 中枢 | 40%-60% | 靠近历史均值 | "估值水平正常" |
| 合理偏高 | 60%-75% | 略高于中枢 | "估值偏高，需盈利支撑" |
| 高估 | > 75% | 处于近5年高位 | "估值处于历史高位，警惕回调风险" |

**中国特色估值调整项**（必须说明）：
- **银行/地产破净**：PB长期<1x（银行~0.55x，地产~0.85x），属于系统性折价，不能简单对标境外市场
- **A/H溢价**：A股较H股均值区间120-145（贵20-45%），原因：资本管制/分红所得税差异/流动性差异
- **国企分红权重**：大型国企PE估值中须考虑现金分红率对估值的影响，可用股息率作为防御性估值锚
- **扣非PE**：消费/医药/制造行业的PE须注明是否剔除政府补贴等非经常性损益

**数据不可得fallback**：若无实时数据接入，标注"参考期间：2024Q4"及数据来源（Wind/申万研究所），提示"估值分位数随行情变化，建议接入实时数据接口"。

### Step 6 — 景气度跟踪

**输入**：Step 1 的行业范围
**输出**：

| 指标 | 当前 | 环比 | 同比 | 趋势 |
|------|------|------|------|------|
| 行业营收增速中位数 | | | | |
| 行业净利润增速中位数 | | | | |
| 行业毛利率中位数 | | | | |
| 行业应收账款周转天数 | | | | |

**数据不可得fallback**：如无法获取行业汇总数据，可列出该行业重点公司的上述指标（标注"样本公司口径，非行业全量"）。

### Step 7 — 政策与催化剂

**输入**：行业相关政策/产业周期信息
**输出**：
- 已出台政策及影响评估
- 待出台政策预期（两会/国常会/部委文件）
- 产业周期位置：导入/成长/成熟/衰退/反转

**A股特有政策关注点**：
- 集采政策（医药/医疗器械）→ 直接影响板块估值中枢
- 补贴退坡（新能源/光伏）→ 关注退坡节奏对盈利的影响
- 国产替代/信创（半导体/计算机）→ 政策加码意味着需求侧支撑
- 双碳/能耗双控（化工/钢铁/有色）→ 供给侧约束

### Step 8 — 投资策略与标的建议

**输入**：Step 1-7 的全部分析结论
**输出**：

| 策略 | 适合环境 | 推荐标的 | 逻辑 |
|------|---------|---------|------|
| 龙头策略 | | | |
| 弹性标的 | | | |
| 困境反转 | | | |

**标的匹配**：推荐标的须标注申万三级行业归属（从 `sources/mappings/申万行业分类_2021.csv` 校验代码归属）。

### Step 9 — 风险揭示与合规声明

**必须输出的合规声明**：
```
⚠️ 免责声明：本行业分析仅供内部研究参考，不构成投资建议。
估值数据基于公开信息和历史统计（数据截止：YYYY-MM），行业惯例区间非定价依据。
投资决策须结合个人风险承受能力和专业顾问意见。
```

**行业风险提示要点**（按行业特征逐项说明）：
- 政策风险（如集采/反垄断/能耗双控）
- 周期风险（如猪周期/半导体周期/大宗商品价格波动）
- 技术替代风险（如新能源技术路线变更）
- 估值风险（当前分位数水平下的回撤风险）

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `initiating-coverage` | 需要深入分析某只个股时 | 个股估值/盈利预测/风险评级 |
| `earnings-analysis` | 需要验证行业龙头的盈利质量时 | 扣非净利润/现金流质量/商誉风险 |
| `catalyst-calendar` | 需要确认行业政策催化时点 | 业绩披露窗口/政策会议日程 |
| `idea-generation` | 需要生成具体投资建议时 | 标的池/仓位建议/止损位 |

> ⚠️ 红线：本技能不直接输出个股的买入/卖出/持有评级。若用户要求，指引其调用 `initiating-coverage` 或 `idea-generation`。
