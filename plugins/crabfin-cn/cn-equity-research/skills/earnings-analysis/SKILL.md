# 财报分析 — CAS口径深度解读

对中国上市公司（A股为主，兼顾港股）财报进行结构化分析，核心围绕**扣非归母净利润**（A股特有指标）与**CAS科目映射**进行深度解读，区分CAS与美国GAAP/IFRS的准则差异。

> **CAS科目映射**：`sources/mappings/CAS科目映射_v1.csv`（60+核心CAS科目到三大报表行项的精确映射）
> **信披规则参考**：`sources/laws/沪深北港交所信息披露规则要点.md`（业绩预告触发条件/退市新规红线）
> **估值基准**：`sources/benchmarks/A股港股估值惯例与基准倍数.md`（行业估值分位数对标）
> **协作技能**：首次覆盖调用 `initiating-coverage`；盈利预测调用 `model-update`；业绩前瞻调用 `earnings-preview`

## 操作步骤

### Step 1 — 核心财务数据提取

**输入**：指定公司的财报期数（年报/半年报/季报）
**输出**：

```markdown
| 指标 | 报告期 | 上年同期 | YoY |
|------|--------|---------|-----|
| 营业收入(亿元) | | | ±XX% |
| 归母净利润 | | | ±XX% |
| 扣非归母净利润 | | | ±XX% |
| 毛利率 | | | ±XXpp |
| 净利率 | | | ±XXpp |
| 经营现金流 | | | |
```

**CAS科目映射规则**（引用自 `sources/mappings/CAS科目映射_v1.csv`）：
- 营业收入 = 6001主营业务收入 + 6051其他业务收入；确认时点为"控制权转移"（CAS 14号—收入）
- 营业成本 = 6401主营业务成本 + 6402其他业务成本
- 税金及附加 = 6403（含城建税、教育费附加等；**增值税为价外税**，不在此科目）
- 管理费用 = 6602（**含研发费用未资本化部分**；科创板/创业板公司须单独拆出研发费用率）
- 信用减值损失 = 6701（CAS 22号—预期信用损失模型，**已替代旧准则的已发生损失模型**；A股须单独列示，这是与IFRS的重大差异）
- 资产减值损失 = 6701（含商誉减值，**商誉减值信号是A股重大负面催化剂**）

**计算过程中必须使用的CAS科目代码**：
| 科目 | CAS代码 | 用途 |
|------|--------|------|
| 主营业务收入 | 6001 | 营收核心构成 |
| 主营业务成本 | 6401 | 毛利计算 |
| 管理费用 | 6602 | 含研发未资本化部分 |
| 资产减值损失 | 6701 | 减值风险识别 |
| 公允价值变动损益 | 6101 | 非经常性损益识别 |
| 投资收益 | 6111 | 区分经常性与非经常性 |

**停止条件**：无法获取公司财报原文 → 告知用户前往巨潮资讯网(cninfo.com.cn)或HKEXnews获取，不可基于摘要信息编造财务数据。

### Step 2 — 盈利质量分析（核心步骤）

**输入**：Step 1 的利润表数据
**输出**：

**2.1 扣非/净利润偏离度**
```
扣非净利润 / 归母净利润 = XX%
```
判断标准：
- ≥ 85%：盈利质量良好（利润主要来自主营业务）
- 70%-85%：关注非经常性损益构成
- < 70%：盈利质量偏低，利润依赖非经常性项目
- **持续2个季度以上偏离 > 20%（即扣非/净利润 < 80%）→ 警示盈利质量**

非经常性损益判定清单（CAS口径）：
- 非流动性资产处置损益
- 政府补助（计入当期损益部分，**与正常经营密切相关且符合国家政策、持续享受的除外**）
- 公允价值变动损益（CAS 6101）+ 投资收益（CAS 6111）中属于非经常性的部分
- 非货币性资产交换损益
- 同一控制下企业合并产生的子公司期初至合并日的当期净损益

**2.2 经营现金流含金量**
```
经营现金流 / 归母净利润 = XX
```
- ≥ 1.0：现金流充沛，利润含金量高
- 0.7-1.0：基本正常
- **< 0.7：利润含金量警示** — 利润可能来自应收而非真实现金流入
- < 0：严重警示

**2.3 政府补助依赖度**
```
政府补助 / 归母净利润 = XX%
```
- > 30%：高度依赖政府补助，关注补贴退坡风险（新能源/光伏行业尤为关键）

### Step 3 — 资产负债表风险扫描

**输入**：公司资产负债表
**输出**：

**3.1 商誉减值风险**（证监会重点监管领域）
```
商誉 / 净资产 = XX%
```
- **> 30%：减值高风险**，必须逐笔核查商誉对应的被收购标的业绩完成情况
- 10%-30%：中等风险，关注
- < 10%：低风险

**3.2 应收账款质量**
```
应收票据及应收账款(CAS 1121+1122) / 营业收入 = XX%
坏账准备(CAS 1231) / 应收账款原值 = XX%
```
- 应收账款/营收：同比大幅上升 → 回款恶化信号
- 坏账准备率：同比下降 → 可能计提不足

**3.3 合同负债**（新准则预收款 — CAS 14号）
- 合同负债(CAS 2203 预收账款 → 合同负债)增长 → 未来营收增长先行指标
- 合同负债下降 → 在手订单减少信号

**3.4 应收账款融资/保理出表**（CAS 23号—金融资产转移）
- 关注附注中应收账款的保理／资产证券化安排
- 若带有追索权（回购义务）但仍已出表 → **潜在表外负债风险**

### Step 4 — 业绩预告合规性检查

**输入**：公司最新财报 V.S. 已发布的业绩预告
**输出**：对照 `sources/laws/沪深北港交所信息披露规则要点.md` 的触发条件检查：

| 检查项 | 触发条件 | 实际状况 | 是否须发预告 |
|--------|---------|---------|------------|
| 净利润为负 | 净利润 < 0 | | |
| 扭亏为盈 | 上年同期亏损 → 本期盈利 | | |
| 净利润增减 ≥ 50% | |YoY变动| ≥ 50%? | |
| 退市红线 | 扣非前后孰低为负，且扣除后营收 < 1亿 | | |
| 净资产为负 | 期末净资产 < 0 | | |

**若公司未按规则发业绩预告** → 在分析中标注为合规风险信号。

**业绩快报修正**：若已发业绩快报，检查与定期报告差异是否 > 20%（> 20%须修正公告），若未修正 → 标注。

### Step 5 — 输出完整分析报告

**输出模板**（必须使用此结构）：

```markdown

| 扣非归母净利润 | | | |
| 毛利率 | | | |
| 净利率 | | | |
| 经营现金流 | | | |

## 2. 盈利质量
- 扣非/净利润比率：XX%（正常/偏低）
- 经营现金流/净利润：XX（良好/警惕/严重警示）
- 政府补助/净利润：XX%（正常/依赖度高）

## 3. 关键发现
[逐项分析各科目变动原因]

- 成本端：原材料/人工/制造费用变动

财务数据来源于公开披露的定期报告，分析结论基于CAS准则口径。
上市公司存在财务舞弊可能，本分析不替代独立审计。

数据来源：
1. 巨潮资讯网(cninfo.com.cn) — A股财报原文及问询函
2. HKEXnews — 港股财报
3. 交易所问询函 — 年报问询函中的监管关切点（重点关注收入确认/商誉减值/关联交易）
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `initiating-coverage` | 首次分析该公司时，需要完整估值模型 | DCF/可比估值/目标价区间 |
| `model-update` | 新财报数据需要更新盈利模型 | 更新后的三表预测 |
| `earnings-preview` | 财报发布前，需对比实际数据与一致预期 | 一致预期/超预期/低预期判断 |
| `catalyst-calendar` | 判断业绩披露是否在公司重要催化剂窗口 | 业绩披露日程/政策事件 |
| `sector-overview` | 需要将公司放在行业估值水位中比较 | 行业PE/PB分位数 |

> ⚠️ 红线：本技能不就股票投资价值给出买入/卖出建议。分析中发现退市风险信号（扣非前后孰低为负 + 营收 < 1亿 / 净资产为负）时，必须在报告开头以醒目方式标注。
