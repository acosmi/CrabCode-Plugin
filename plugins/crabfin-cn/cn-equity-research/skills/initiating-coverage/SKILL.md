# 首次覆盖报告 — A股卖方研报格式

按中国卖方研究惯例编写首次覆盖报告，包含投资摘要、公司概况、行业分析、盈利预测、估值定价和风险提示。

> **报告模板**：`sources/templates/首次覆盖研报_脱敏模板.md`（完整脱敏模板，严格按此结构输出）
> **估值基准**：`sources/benchmarks/A股港股估值惯例与基准倍数.md`（行业PE/PB分位数+估值方法选择指引）
> **CAS映射**：`sources/mappings/CAS科目映射_v1.csv`（60+科目到三表行项映射）
> ⚠️ 红线：评级和目标价必须基于公开数据和合理假设推导，不可编造。

## 操作步骤

### Step 1 — 投资评级确定

**输入**：当前股价 + 目标价预估区间
**输出**：五级评级（中国卖方标准口径）：

| 评级 | 含义 | 超额收益要求 |
|------|------|------------|
| 买入 | 未来6-12个月相对沪深300超额收益 | ≥ 20% |
| 增持 | 超额收益 | 5%-20% |
| 中性 | 超额收益 | -5% ~ +5% |
| 减持 | 超额收益 | ≤ -5% |
| 卖出 | 超额收益 | ≤ -20% |

**跨市场调整**：
- **A股中小盘**：普遍享有流动性溢价，可比PE可向上微调
- **港股中小盘**：普遍存在流动性折价，可比PE须向下调整
- **A/H股**：AH溢价指数均值120-145（A较H贵20-45%），跨市场比较时必须注明

### Step 2 — 公司概况梳理

**输入**：公司公告 + 招股书 + 年报
**输出**（按 `sources/templates/首次覆盖研报_脱敏模板.md` 结构）：

1. **发展历程**：创业期→成长期→转型期 三个阶段的里程牌事件
2. **股权结构**：实控人及持股比例（直接/间接），前三大股东，机构投资者占比（含北向资金/社保/公募/险资）
3. **业务构成**：按CAS收入确认口径拆分（主营业务收入6001 vs 其他业务收入6051），含各板块毛利率和增速

**停止条件**：无法获取公司股权结构 → 标注"基于公开公告信息整理，非完整版"。

### Step 3 — 行业分析

**输入**：公司所属申万行业 → 调用 `sector-overview` 获取行业数据
**输出**：
- 行业空间：中国市场规模（亿元）、历史CAGR、未来预期CAGR
- 竞争格局对标表：| 维度 | 本公司 | 竞对A | 竞对B |
- 行业结构性趋势（2-3条）

**协作**：本步骤直接调用 `sector-overview` 获取行业PE/PB分位数和景气度数据。

### Step 4 — 核心投资逻辑（3条）

**输入**：Step 2+3 的公司和行业分析
**输出**：3条可验证的核心投资逻辑，每条须含：
- 核心论点
- 数据支撑（引用具体数字/行业地位/管理层指引/第三方验证）
- 验证时点（季报/半年报/年报/具体事件）

### Step 5 — 盈利预测（CAS科目映射）

**输入**：公司历史财报（A股→巨潮资讯网，港股→HKEXnews）
**输出**：

| 科目 | CAS代码 | 2023A | 2024A | 2025E | 2026E | 2027E |
|------|--------|-------|-------|-------|-------|-------|
| 营业收入 | 6001+6051 | | | | | |
| 营业成本 | 6401+6402 | | | | | |
| 销售费用 | 6601 | | | | | |
| 管理费用 | 6602 | | | | | |
| 研发费用 | 6602(未资本化) | | | | | |
| 财务费用 | 6603 | | | | | |
| 信用减值损失 | 6701(A股单独列示) | | | | | |
| 资产减值损失 | 6701 | | | | | |
| **归母净利润** | | | | | | |
| **扣非净利润** | 归母−非经常损益(税后) | | | | | |

**关键假设表**：

| 假设项 | 2025E | 2026E | 依据 |
|--------|-------|-------|------|
| 营收增速 | | | 行业增速 × 公司份额 |
| 毛利率 | | | 原材料趋势 + 产品结构 |
| 研发费用率 | | | 历史均值 ± 战略变动 |
| 所得税率 | | | 高新15% / 一般25%（关注证书有效期） |

**数据不可得fallback**：若无法获取分业务板块收入，按总量预测，标注"未拆分业务线"。

### Step 6 — 估值定价

**输入**：Step 5 的盈利预测 + `sources/benchmarks/A股港股估值惯例与基准倍数.md`

**估值方法选择**（按行业适配）：

| 估值方法 | 适用行业 | 中国调整项 |
|---------|---------|-----------|
| PE(TTM/Forward) | 消费/医药/制造/金融 | 扣非PE；国企考虑分红 |
| PB | 银行/地产/重资产 | 银行~0.55x破净是系统性折价 |
| EV/EBITDA | 公用事业/通信/重工业 | 高负债率行业适用 |
| DCF | 补充验证 | 调用 fin-core `dcf-model` |

**目标价推导**：
```
目标价 = 预测EPS(Forward 12M) × 目标PE倍数
上涨空间 = (目标价 − 当前价) / 当前价
```

**同业估值对标表**：

| 可比公司 | 代码 | PE(2025E) | PEG | 备注 |
|---------|------|-----------|-----|------|
| 估值中枢 | — | XX.Xx | — | |

目标PE取可比公司均值±1个标准差。

### Step 7 — 输出完整报告

**输出模板**（严格按 `sources/templates/首次覆盖研报_脱敏模板.md` 结构）：

```markdown
# [公司简称](代码.SZ/SH) — 首次覆盖报告
**评级**：[买入/增持/中性/减持] | **目标价**：[XX.XX]元 | **当前价**：[XX.XX]元 | **上涨空间**：[XX]%

## 投资摘要
[3-4句话总结 + 关键指标表(营收/EPS/PE/PB/ROE)]

## 一、公司概况
- 发展历程（三阶段）
- 股权结构（实控人+前三大股东+机构占比）
- 业务构成（按CAS收入口径）

## 二、行业分析
- 行业空间（CAGR+预期规模）
- 竞争格局对标
- 结构性趋势

## 三、核心投资逻辑
- 逻辑一/二/三（各含数据支撑+验证时点）

## 四、盈利预测与估值
- 关键假设表
- 盈利预测表（含CAS科目映射）
- 估值（PE主+DCF辅+同业对标）

## 五、风险提示
[5条风险，含影响幅度/概率]

## 免责声明
本报告仅供内部研究参考，不构成投资建议。评级基于公开信息和合理假设，实际走势受多种因素影响。分析师与报告提及公司不存在利益关系。
```

### Step 8 — 风险揭示与合规声明

```markdown
⚠️ 免责声明：本报告仅供参考，不构成投资建议。投资者据此操作，风险自负。
分析基于公开数据（巨潮资讯网/Wind/公司公告），对信息的准确性和完整性不做保证。
估值假设包含主观判断，实际结果可能存在重大偏差。

评级定义：
- 买入：未来6-12个月相对沪深300超额收益 ≥20%
- 增持：超额收益 5%-20%
- 中性：超额收益 ±5%以内
- 减持：超额收益 ≤-5%
- 卖出：超额收益 ≤-20%
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `sector-overview` | Step 3 行业分析 | 行业PE/PB分位数/景气度/竞争格局 |
| `model-update` | Step 5 盈利预测需要 | 三表预测框架 |
| `earnings-analysis` | 验证公司历史盈利质量 | 扣非/现金流/商誉分析 |
| `dcf-model` (fin-core) | Step 6 DCF交叉验证 | 绝对估值 |
| `thesis-tracker` | 报告完成后，建立投资逻辑跟踪 | 核心假设跟踪表 |
