# 投资逻辑跟踪 — A股证伪信号监控

为A股/港股持仓构建投资逻辑跟踪体系，将投资判断拆解为可验证的核心假设，持续监控公司动态和外部变化，及时发现证伪信号。

> **信披规则**：`sources/laws/沪深北港交所信息披露规则要点.md`（重大事项临时披露标准/ST规则/退市红线）
> **估值基准**：`sources/benchmarks/A股港股估值惯例与基准倍数.md`（盈利偏离时，评估估值水位变化）

## 操作步骤

### Step 1 — 投资逻辑拆解

**输入**：用户的核心投资逻辑（可从 `initiating-coverage` 报告或 `idea-generation` 输出继承）
**输出**：核心假设拆解表

```
投资逻辑 = 假设(H1) × 假设(H2) × ... × 假设(HN) → 预期结果
```

每个假设必须满足：
- **可验证性**：能通过公开信息判断是否成立
- **可量化性**：有数字化的验证标准（而非"公司会变好"这种模糊表述）
- **时间表**：明确在什么时点可以验证（季报/年报/特定事件）

**协作**：若用户未提供投资逻辑，调用 `initiating-coverage` → 从中提取核心假设。

### Step 2 — 建立跟踪表

**输入**：Step 1 的假设拆解
**输出**：

```markdown
# [股票简称](代码) — 投资逻辑跟踪表

## 基本信息
| 项目 | 内容 |
|------|------|
| 建仓日期(如有) | |
| 核心投资逻辑 | 一句话概括 |

## 逻辑拆解与跟踪
| 编号 | 核心假设 | 验证指标 | 目标值 | 当前值 | 状态 | 下次验证 |
|------|---------|---------|--------|--------|------|---------|
| H1 | | | | | ✅/⚠️/❌ | YYYY-QQ |
| H2 | | | | | | |
| H3 | | | | | | |

## 催化剂跟踪
| 日期 | 催化剂 | 预期影响 | 实际结果 | 逻辑影响 |
|------|--------|---------|---------|---------|
```

**状态含义**：
- ✅ 假设成立 / 符合预期
- ⚠️ 有疑虑 / 需要进一步观察（标注疑虑来源）
- ❌ 假设已证伪 / 显著低于预期（标注偏离幅度）

**停止条件**：用户无法提供任何可验证的投资假设 → 指引调用 `initiating-coverage` 先建立研究框架。

### Step 3 — 监控中国特有的证伪信号 ⚠️ 核心步骤

**输入**：最新公司公告/交易所信息
**输出**：以下五类信号的逐条检查：

#### 3.1 业绩信号

| 信号 | 判定标准 | 严重程度 |
|------|---------|---------|
| 扣非/净利润持续偏离 | 扣非远小于净利润，持续 **2个季度以上** | ⚠️ 中 — 主营承压 |
| 毛利率与行业趋势背离 | 行业毛利率下行但公司上行，无合理解释 | ⚠️ 中 — 财务调节风险 |
| 经营现金流/净利润 | **< 0.7 持续** | ⚠️ 中高 — 利润含金量存疑 |
| 业绩预告修正 | 方向改变（盈利→亏损），或幅度 > 20% | ❌ 高 — 管理层判断失误 |

#### 3.2 监管信号（引用自 `sources/laws/沪深北港交所信息披露规则要点.md`）

| 信号 | 严重程度 | 对应行动 |
|------|---------|---------|
| 交易所年报问询函 | 中 | 逐条分析问询事项，关注收入确认/商誉/关联交易 |
| 证监会立案调查 | **高** | 立即评估是否清仓 |
| ST/\*ST 警示 | **极高** | **退市风险 — 一般应立即清仓** |
| 监管层公开谴责/罚款 | 中高 | 关注是否涉及核心管理层 |

**退市新规红线**（来自 `sources/laws/沪深北港交所信息披露规则要点.md`）：
- 扣非前后孰低为负 + 扣除后营收 < **1亿元**
- 净资产为负
- 审计报告出具无法表示意见/否定意见

#### 3.3 公司治理信号

| 信号 | 影响方向 | 依据 |
|------|---------|------|
| 独立董事对议案投反对/弃权票 | ↓ 负面 | 内部治理矛盾公开化 |
| 审计师出具非标意见 | ↓↓ **极负面** | 财务可信度受损 |
| 董事长/总经理/财务总监离职 | ↓ 关注 | 核心管理层不稳定 |
| 大股东高比例质押 | ↓ **平仓风险** | 占总股本 > 50%为高危 |
| 大股东股份冻结 | ↓↓ **控制权风险** | 查冻结原因 |
| ST公司关联交易异常增加 | ↓ 警示 | 保壳操纵嫌疑 |

#### 3.4 行业与政策信号

| 信号 | 影响 | 适用行业 |
|------|------|---------|
| 行业补贴退坡 | 行业利润率可能下移 | 新能源/光伏 |
| 集采/谈判降价 | **医药行业核心杀逻辑** | 医药/医疗器械 |
| 反垄断调查 | 互联网/平台经济 | 互联网 |
| 产业政策加码 | 正面催化 | 半导体/信创/新能源 |

#### 3.5 信披合规信号

| 信号 | 含义 | 对应规则 |
|------|------|---------|
| 触发强制预告但未公告 | 信披违规 | `信披规则` 2.1 |
| 业绩快报与实际差异 > 20%未修正 | 信披违规 | `信披规则` 2.2 |
| 重大交易未在2个交易日内披露 | 信披违规 | `信披规则` 3.1 |

### Step 4 — 退出条件判定

**输入**：Step 3 的监控结果
**输出**：

| 退出类型 | 条件 | 行动 |
|---------|------|------|
| 目标达成 | 股价达到目标价 | 按计划减仓/清仓 |
| 逻辑证伪 | ≥ 2个核心假设被 ❌ | 择机清仓 |
| 结构变化 | ST/\*ST / 立案调查 / 控制权变更 | **立即评估是否清仓** |
| 时间止损 | 持仓超预期周期 + 逻辑无进展 | 重新评估 |

**数据不可得fallback**：若无法获取监管/治理类信息（如公司公告检索困难），标注"信息来源不足，部分信号未校验"并列出未校验项。

### Step 5 — 更新跟踪表并输出

```markdown
# [股票简称](代码) — 投资逻辑跟踪表（更新至 YYYY-MM-DD）

## 最新进展
[本监测周期内的关键变化，1-2句]

## 逻辑状态总览
| 编号 | 核心假设 | 状态 | 变化说明 |
|------|---------|------|---------|

## 证伪信号扫描
| 类别 | 信号 | 触发？ | 影响 |
|------|------|--------|------|
| 业绩 | 扣非/净利润偏离 | | |
| 监管 | 问询函/立案 | | |
| 治理 | 管理层变动/质押 | | |
| 政策 | 行业政策变化 | | |

## 行动建议
[基于以上分析，建议：继续持有/减仓/清仓/观望 — 不构成投资建议]
```

### Step 6 — 风险揭示与合规声明

```
⚠️ 免责声明：本投资逻辑跟踪仅供研究参考，不构成"持有/卖出/买入"等投资建议。
证伪信号基于公开信息整理，可能不完整或存在时滞。
持仓决策须结合个人风险承受能力和独立研究判断。

信息来源：
- 巨潮资讯网(cninfo.com.cn) — 公司公告原文/问询函
- 交易所问询函专区
- Wind/Choice — 财务数据
- 内部模型预测 vs 实际数据对比
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `initiating-coverage` | 尚无投资逻辑 → 需要先建研究框架 | 核心假设/目标价/盈利预测 |
| `earnings-analysis` | 新财报发布 → 验证假设 | 盈利质量分析 |
| `catalyst-calendar` | 跟踪验证时点 | 催化剂日历 |
| `model-update` | 盈利数据偏离 → 更新模型 | 更新后的三表预测 |
| `idea-generation` | 逻辑证伪 → 寻找替代标的 | 基于新逻辑的选股池 |
