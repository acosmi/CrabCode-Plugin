# 选股思路生成 — 申万/证监会分类驱动

基于申万28个一级行业和证监会行业分类体系，提供三种选股范式：自上而下（行业轮动）、自下而上（因子筛选）、主题/事件驱动。

> **行业分类**：`sources/mappings/申万行业分类_2021.csv`（28一级/100+二级/200+三级行业完整代码）
> **估值基准**：`sources/benchmarks/A股港股估值惯例与基准倍数.md`（PE/PB分位数+行业估值中枢）
> ⚠️ 红线：本技能不给出个股的"买入/卖出/持有"操作建议。选股结果为研究参考，非投资建议。

## 操作步骤

### Step 1 — 确定选股策略类型

**输入**：用户的选股意图
**输出**：策略类型判定

| 用户表述 | 策略类型 |
|---------|---------|
| "这个行业有什么好票""帮我筛XX板块" | 自上而下(Top-Down) |
| "ROE高/现金流好/估值低的票""按照XX指标筛选" | 自下而上(Bottom-Up) |
| "国产替代受益""新质生产力标的""国企改革标的" | 主题/事件驱动 |

### Step 2 — 自上而下(Top-Down)选股

**输入**：目标行业（申万一级）
**输出**：

```
宏观判断 → 行业配置 → 选股
```

**行业筛选维度**：

| 维度 | 指标 | 判断标准 | 来源 |
|------|------|---------|------|
| 景气度 | 营收增速中位数 | 正增长且环比提速 | Wind/Choice |
| 估值 | PE-TTM分位 | < 25% 低估 | `benchmarks/估值惯例` |
| 资金面 | 北向资金净买入 | 连续N日净流入 | 交易所数据 |
| 政策面 | 产业政策边际变化 | 定性评估 | 国务院/发改委/工信部 |

**估值水位判断**（来自 `sources/benchmarks/A股港股估值惯例与基准倍数.md`）：
- 低估(< 25%分位) → 关注机会
- 合理偏低(25%-40%) → 有安全边际
- 中枢(40%-60%) → 正常
- 合理偏高(60%-75%) → 需盈利支撑
- 高估(> 75%) → 警惕回调

**停止条件**：所选行业在申万分类表中找不到匹配 → 提示用户提供更精确的行业名或使用证监会分类。

### Step 3 — 自下而上(Bottom-Up)选股

**输入**：用户指定的筛选条件（或使用默认条件）
**输出**：筛选结果池

**A股推荐筛选条件（默认）**：

| 筛选维度 | 条件 | 排除目的 |
|---------|------|---------|
| 市值 | > 50亿元 | 剔除微盘流动性风险 |
| 日均成交额 | > 5000万元 | 保证流动性 |
| ROE (连续3年) | > 10% | 盈利持续性 |
| 经营现金流/净利润 | > 0.8 | 利润含金量 |
| 扣非净利润增速 | > 营收增速 | 效率提升信号 |
| ST/*ST | **排除** | 退市风险 |
| 商誉/净资产 | < 30% | 商誉减值风险（证监会重点监管领域） |

**银行/金融行业特殊调整**：
- PB长期破净（银行 ~0.55x）是系统性特征，PB条件放宽
- 须关注拨备覆盖率、不良率、净息差替代ROE
- 关注国企现金分红率（股息率型估值）

**数据不可得fallback**：若无法批量获取全市场数据，请用户提供候选股票列表（5-10只），基于手工数据做筛选验证。

### Step 4 — 主题/事件驱动选股

**输入**：投资主题/事件
**输出**：主题映射 + 受益标的

**A股常见投资主题**：

| 主题 | 映射的申万行业 | 政策出处 |
|------|-------------|---------|
| 国企改革/估值重塑 | 801220银行/801230非银/801010农林牧渔（央企） | 国资委/证监会 |
| 碳中和/新能源 | 801730电力设备/801160公用事业 | 发改委/能源局 |
| 数字经济/AI | 801750计算机/801780电子/801760传媒 | 工信部/网信办 |
| 新质生产力 | 801780半导体/801150创新药/801750软件 | 政府工作报告 |
| 国产替代 | 801780半导体/801090机械设备/801750基础软件 | 工信部/科技部 |
| 消费复苏 | 801120食品饮料/801200休闲服务/801110汽车 | 商务部 |

### Step 5 — 输出选股报告

```markdown
# 选股思路 — [策略名称]([日期])

## 策略逻辑
[为什么这个策略在当前市场有效 — 2-3句话]

## 筛选标准
| 维度 | 标准 |
|------|------|
| 行业(申万一级) | 801730 电力设备 |
| 市值 | > 50亿 |
| ROE | 连续3年 > 10% |
| 额外条件 | 研发费用率 > 5% |

## 候选标的（5-10只）
| 代码 | 简称 | 申万三级 | 市值(亿) | PE-TTM | ROE | 核心逻辑 |
|------|------|---------|---------|--------|-----|---------|

## 估值对标
- 行业PE中枢：XXx（`benchmarks/估值惯例`）
- 行业PE当前分位：XX%
- 候选标的PE均值：XXx

## 风险提示
- 策略失效条件：[什么情况下这个选股逻辑会失效]
- 系统性风险：[宏观/政策黑天鹅]
- 个股风险：[解禁/商誉/大股东质押]
```

### Step 6 — 风险揭示与合规声明

```
⚠️ 免责声明：本选股思路仅供研究参考，不构成投资建议。所选标的基于公开数据和量化筛选条件，
不代表对未来收益的任何保证。投资者须结合自身风险承受能力和独立研究进行判断。

重要提示：
- 本技能不提供个股的"买入/卖出/持有"评级
- 历史ROE持续性不代表未来表现
- 筛选条件为学术/经验性参考，无预测能力
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `sector-overview` | 自上而下选股时，需行业景气度和估值水位 | 行业PE/PB分位数/景气度指标 |
| `initiating-coverage` | 筛选出的标的需深度分析 | 个股估值模型 |
| `earnings-analysis` | 候选标的需盈利质量验证 | 扣非/现金流/商誉分析 |
| `catalyst-calendar` | 事件驱动型选股时，需确认催化剂时点 | 事件日历 |
| `thesis-tracker` | 选股思路需持续跟踪验证 | 投资逻辑状态更新 |
