# 晨报 — A股每日速递

按中国卖方研究惯例编写每日晨报，汇总隔夜外盘、A股/港股盘前信号、重要公告、研究观点及宏观日历。

> **格式模板**：`sources/templates/晨报_脱敏模板.md`（完整脱敏模板，按此结构输出）
> **信息来源规则**：`sources/laws/沪深北港交所信息披露规则要点.md`（业绩预告/重大事项公告的触发时点与量化标准）
> **协作技能**：公司深度点评调用 `initiating-coverage`；业绩预测调用 `earnings-preview` / `model-update`

## 操作步骤

### Step 1 — 市场回顾

**输入**：最新交易日行情数据（隔夜外盘+昨日A股/港股）
**输出**：

**隔夜外盘**：
| 指数 | 收盘价 | 涨跌幅 |
|------|--------|--------|
| 标普500 | | |
| 纳斯达克 | | |
| 恒生指数夜期 | | |
| 富时A50期货 | | |

**昨日A股**（按 `sources/templates/晨报_脱敏模板.md` 的指标格式）：
| 指数 | 收盘 | 涨跌幅 | 成交额(亿) |
|------|------|--------|-----------|
| 上证指数 | | | |
| 深证成指 | | | |
| 创业板指 | | | |
| 科创50 | | | |

**必填的A股特有信号**：
- **北向资金（陆股通）**：净流入/净流出金额（亿），注明是否连续N日流入/流出
- **两融（融资融券）余额**：前日余额及变动方向
- **成交额分水岭判断**：两市合计成交额是否突破万亿（万亿为缩量/放量分水岭）
- **领涨/领跌行业**：按申万一级行业列出前3，含涨跌幅

**数据不可得fallback**：若无法获取实时行情，标注"截至 [日期] 收盘数据"并提示"实时数据建议接入Wind/Choice终端"。

### Step 2 — 重要公告速览

**输入**：当日重大公告（来自巨潮资讯网/HKEXnews/交易所公告）

**输出**（按事件类型分类）：

| 类型 | 公司(代码) | 内容 | 影响方向 |
|------|-----------|------|---------|
| 业绩预告 | | | ↑/↓ |
| 重大合同 | | | |
| 重组停牌 | | | |
| 股东增减持 | | | |
| 监管问询 | | | |

**中国特有公告筛选规则**（引用自 `sources/laws/沪深北港交所信息披露规则要点.md`）：

| 公告类型 | 量化触发标准 | 晨报中的处理方式 |
|---------|-------------|----------------|
| 业绩预告 — 净利润为负 | 净利润负值 | 标红警示，注明是否触及退市新规红线（扣非前后孰低为负，且扣除后营收 < 1亿） |
| 业绩预告 — 增减 > 50% | 净利润同比变动 ≥ 50% | 标注变动幅度和核心原因 |
| 业绩预告 — 扭亏为盈 | 净利润由负转正 | 标注"扭亏" |
| ST/*ST 风险警示 | 退市风险警示相关公告 | 必须在晨报中重点标注 |
| 重大交易 | 交易金额占最近一期经审计总资产/净资产/净利润 ≥ 10%（且 > 100万） | 简要说明交易标的和金额 |
| 关联交易 | 与关联法人 > 300万（占净资产0.5%+），与关联自然人 > 30万 | 标注关联方和交易性质 |
| 重大诉讼/仲裁 | 涉案金额 > 1000万 且占净资产 ≥ 10% | 标注诉讼性质和涉案金额 |

**业绩快报修正提醒**：若最新公告的业绩与已披露预告差异 > 20%（方向改变或幅度重大），必须在晨报中标注"业绩修正"。

### Step 3 — 今日关注

**输入**：当日事件日历
**输出**：
1. **今日披露**：X家公司发布年报/季报，按披露规则（年报4个月内/半年报2个月内/季报1个月内 — 来自 `sources/laws/沪深北港交所信息披露规则要点.md`）判断是否为法定披露密集期
2. **今日解禁**：限售股解禁数量和合计市值
3. **宏观数据**：当日发布的经济数据（CPI/PPI/PMI等），注明预期值和前值

### Step 4 — 研究观点速递

**输入**：当日推荐关注标的
**输出**：1-3只推荐标的，格式：`[公司简称](代码) | 评级 | 目标价 | 一句话逻辑`

涉及港股标的时，标注A/H溢价水平（参考 `sources/benchmarks/A股港股估值惯例与基准倍数.md` 中的 A/H 溢价惯例：120-145 区间）。

**协作**：若有需要深度论证的标的，调用 `initiating-coverage` 获取详细研报口径。

### Step 5 — 新股/再融资

**输入**：当日新股申购/上市信息
**输出**：

| 代码 | 简称 | 类型(IPO/增发/配股/可转债) | 发行价 | 申购/上市日期 |
|------|------|--------------------------|--------|-------------|

### Step 6 — 宏观日历（本周）

**输入**：当周宏观事件
**输出**：

| 日期 | 事件 | 涉及标的 | 关注级别 |
|------|------|---------|---------|
| | | | ⭐⭐⭐/⭐⭐/⭐ |

### Step 7 — 风险揭示与合规声明

**必须附加的声明**（引用自 `sources/templates/晨报_脱敏模板.md` 声明段）：
```
⚠️ 声明：本晨报仅供内部交流使用，不构成投资建议。所含信息来源于公开资料，
对信息的准确性和完整性不做保证。市场数据可能因数据源差异存在偏差。

信息来源优先级：
1. 交易所公告（巨潮资讯网/HKEXnews）
2. Wind/Choice终端数据（如有接入）
3. 财新/上海证券报/证券时报等权威财经媒体
4. 不得引用来源不明的社交媒体信息、小道消息、传闻
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `initiating-coverage` | 需要对某只个股做深入点评时 | 估值区间/盈利预测/风险评级 |
| `earnings-preview` | 当日有重点公司发布业绩时 | 业绩预览数据/一致预期 |
| `catalyst-calendar` | 编制"今日关注"和"宏观日历"时 | 未来一周事件日历 |
| `model-update` | 重大公告后需要调整盈利模型时 | 更新后的盈利预测 |

> ⚠️ 红线：本技能不产生个股的买入/卖出/持有建议。若盘中出现重大异动，应提示用户调用 `idea-generation` 或关注 `thesis-tracker` 中的投资逻辑是否触发。
