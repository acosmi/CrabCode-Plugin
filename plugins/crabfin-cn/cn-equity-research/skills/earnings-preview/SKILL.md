# 财报前瞻 — 业绩预览与预告分析

在财报正式披露前，基于A股独有的业绩预告触发体系、行业趋势、历史规律，对即将发布的财报进行前瞻分析和预期差判断。

> **预告规则**：`sources/laws/沪深北港交所信息披露规则要点.md` 第二章（五大强制触发条件 + 预告时限）
> **估值基准**：`sources/benchmarks/A股港股估值惯例与基准倍数.md`（盈利预测对标行业估值中枢）
> ⚠️ 红线：不可套用美股"earnings whisper"概念。A股未预告本身即包含了"变动在 ±50% 以内"的信号含义。

## 操作步骤

### Step 1 — 预告状态判定（决策树）

**输入**：公司代码 + 即将到来的财报期数
**输出**：

```
已公告预告？
├─ 是 → Step 2（基于预告区间预测）
└─ 否 → 检查是否触发强制预告条件：
         ├─ 触发但未公告 → 标注"信披违规风险"（重大负面信号）
         └─ 未触发 → 推定变动幅度在 ±50% 以内（信号本身有价值）
```

**五大强制预告条件**（引用自 `sources/laws/沪深北港交所信息披露规则要点.md`）：
1. 净利润为**负值**
2. 净利润**扭亏为盈**
3. 净利润同比变动 **≥ ±50%**
4. 扣非前后孰低为负，且扣除后营收 **< 1亿元**（退市新规红线）
5. 期末净资产为**负值**

**预告时限**：年报预告→1月31日前 / 半年报→7月15日前

**停止条件**：无法确定公司当前业绩区间 → 标注"数据不足，无法判断预告触发状态"，不可编造。

### Step 2 — 关键指标预测

**输入**：Step 1 的预告状态 + 行业数据
**输出**：

| 指标 | 预测方法 | 预测值 |
|------|---------|--------|
| 营业收入 | 行业增速 × 公司市场份额 + 新业务增量 | |
| 毛利率 | 原材料价格趋势 + 产品结构变动 + 产能利用率 | |
| 扣非净利润 | (营收 × 净利润率) − 非经常性损益预期 | |
| 经营现金流 | 营收 × 历史现金流/营收比率 ± 应收账款变动 | |

**数据不可得fallback**：
- 无行业增速数据 → 使用公司近3年自身营收CAGR，标注"基于历史增速推算"
- 无原材料价格数据 → 标注"原材料价格假设基于公开大宗商品指数"
- 无产能利用率 → 使用历史均值，标注"假设产能利用率不变"

### Step 3 — 一致性对标

**输入**：Step 2 的预测值
**输出**：

| 指标 | 预测值 | 一致预期(Wind/Choice) | 偏离度 | 方向 |
|------|--------|---------------------|--------|------|
| 营收 | | | | 超/符/低预期 |
| 净利润 | | | | |
| 扣非净利润 | | | | |

- 偏离度 < 5% → 符合预期
- 偏离度 5%-10% → 轻度超/低预期
- 偏离度 > 10% → **需要重点解释差异原因**

**数据不可得fallback**：若无一致预期数据，标注"暂无一致预期可对比"，但仍给出独立预测。

### Step 4 — 催化剂打分

**输入**：所有可预见的催化剂事件
**输出**：

| 催化剂 | 方向 | 强度 | 触发概率 | 影响量级 |
|--------|------|------|---------|---------|
| 业绩预告超预期 | ↑ | 强 | | |
| 业务拆分/重组 | ↑↓ | 中 | | |
| 监管问询 | ↓ | 强 | | |
| 行业政策变化 | ↑↓ | 按政策力度 | | |

**强度判断标准**：
- **强**：可能导致分析师大幅调整预测/评级变化
- **中**：影响盈利 ≤ 10%
- **弱**：情绪影响为主，基本面影响有限

### Step 5 — 输出业绩前瞻报告

```markdown
# [股票简称](代码) — [报告期]业绩前瞻

## 预告状态
- 状态：已公告 / 未公告（推定 ±50%以内 / 触发未公告→信披风险）
- 预告区间(如有)：归母净利润 XX~XX 亿元(±XX%)

## 核心预测
| 指标 | 预测值 | 上年同期 | YoY | 一致预期 | 偏离度 |
|------|--------|---------|-----|---------|--------|
| 营收(亿元) | | | | | |
| 归母净利润 | | | | | |
| 扣非净利润 | | | | | |
| 毛利率 | | | | | |
| 净利率 | | | | | |

## 预期差分析
[超预期/不及预期的关键驱动因素，逐条说明]

## 盈利质量预判
- 扣非/净利润比率预估：XX%（正常/偏低）
- 经营现金流/净利润预估：XX
- 关注点：[是否存在可能被调节的科目]

## 催化剂前瞻
| 催化剂 | 方向 | 强度 | 触发概率 |
|--------|------|------|---------|

## 风险提示
[业绩低于预期的主要风险来源 / 业绩预告修正风险 / 退市警示风险]
```

### Step 6 — 风险揭示与合规声明

```
⚠️ 免责声明：本业绩前瞻仅供内部研究参考，不构成投资建议。
预测基于公开信息、历史规律和行业趋势推断，存在实际值与预测值偏离的风险。
业绩预告规则以中国证监会/交易所现行规定为准，规则修订可能导致判断标准变化。

信披合规警示：
- 若公司触发强制预告条件但未公告，此为信披违规信号，须特别标注
- 业绩快报与实际数据差异 > 20% 须修正公告
- 不可基于内幕信息或未公开数据进行预测
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `earnings-analysis` | 财报正式发布后，对比预测 V.S. 实际 | CAS口径深度分析 |
| `model-update` | 财报发布后需更新三表预测 | 更新后的盈利模型 |
| `catalyst-calendar` | 判断预告/披露日期时 | 完整催化剂日历 |
| `sector-overview` | 需要行业景气度数据作为营收预测基础 | 行业营收增速中位数/毛利率中位数 |
| `initiating-coverage` | 首次覆盖时需完整估值框架 | DCF/可比估值 |
