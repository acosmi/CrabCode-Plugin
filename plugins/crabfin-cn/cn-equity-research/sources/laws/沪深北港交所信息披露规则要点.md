# 沪深北港交所信息披露规则要点

> **文件全名**：《上市公司信息披露管理办法》(证监会令第182号，2021年修订) / 沪深北三地《股票上市规则》/ 港交所《上市规则》
> **出处**：中国证监会 / 上交所 / 深交所 / 北交所 / 港交所(HKEX)
> **生效日期**：2021.5.1 (信披办法) / 持续更新 (交易所上市规则)
> **版本**：2024年编撰整合版
>

## 一、定期报告披露要求

### 1.1 年度报告
| 项目 | 要求 |
|---|---|
| 披露时限 | 会计年度结束之日起 **4个月** 内 |
| 主要板块 | 沪主板 / 深主板 / 创业板 / 科创板 / 北交所 |
| 审计要求 | 必须经符合《证券法》规定的会计师事务所审计 |
| 主要内容 | 公司基本情况、主要会计数据和财务指标、管理层讨论与分析、公司治理、重大事项等 |

### 1.2 半年度报告
| 项目 | 要求 |
|---|---|
| 披露时限 | 上半年结束之日起 **2个月** 内 |
| 审计要求 | 可不经审计(除非拟分配利润、资本公积金转增股本、或证监会/交易所另有要求) |
| 特殊规定 | 北交所挂牌公司应在会计年度上半年结束之日起 2 个月内编制并披露 |

### 1.3 季度报告
| 项目 | 要求 |
|---|---|
| 披露时限 | 季度结束后 **1个月** 内(北交所同理) |
| 审计要求 | 无需审计 |
| 豁免情形 | 年度报告提前披露的,可不单独披露第四季度报告 |

## 二、业绩预告与业绩快报

### 2.1 业绩预告触发条件

**沪深主板**——出现下列情形之一时应披露：

| 维度 | 触发条件 |
|---|---|
| 净利润 | 净利润为**负值** |
| 净利润 | 净利润实现**扭亏为盈** |
| 净利润 | 实现盈利,且净利润与上年同期相比**上升或下降 50% 以上** |
| 营业收入+净利润 | 利润总额/净利润/扣非净利润三者孰低为负值,且扣除后营业收入**低于 1 亿元**(退市新规红线) |
| 净资产 | 期末净资产为**负值** |

**创业板 / 科创板**：在上述基础上,增加"扣非前后孰低值为负、且扣除后营收低于 1 亿元"的明确退市警示信号。

业绩预告时限：**会计年度结束后 1 个月内**(年报预告)；半年度预告时限为 7 月 15 日前。

### 2.2 业绩快报
主板/创业板/科创板在出现以下情形时须披露：
- 在定期报告公告前出现**业绩泄露**
- 业绩快报与定期报告主要财务数据差异超过 **20%** 的,应及时修正

### 2.3 业绩预告修正
已披露业绩预告的,如最新预计与已披露预告差异方向发生变化(盈利变为亏损等),或变动幅度较大(通常为 20%以上),应立即披露修正公告。

## 三、重大事项临时披露

### 3.1 量化触发标准
上市公司发生以下重大事项,应立即(2个交易日内)披露：

| 交易类型 | 量化标准(满足任一) |
|---|---|
| 重大交易 | 交易资产总额/成交金额/交易产生的利润,占最近一期经审计总资产/净资产/净利润的 **10%以上**（且绝对金额超过 100 万元） |
| 关联交易 | 与关联法人发生**300万元以上**(占净资产 0.5%以上)的交易；与关联自然人发生 30 万元以上的交易 |
| 重大诉讼/仲裁 | 涉案金额超过**1000万元**,且占最近一期经审计净资产绝对值 10%以上 |
| 担保 | 任何为非关联第三方提供担保(须董事会/股东会审议) |

### 3.2 其他须披露的重大事项
- 重大资产重组/合并分立
- 控制权变更
- 重大债务违约
- 重大亏损/重大赔偿责任
- 主要负责人无法履职
- 被立案调查/行政处罚

## 四、港交所披露特别规则(适用于 AH 股/纯 H 股)

### 4.1 内幕消息披露
港交所《上市规则》第 13.09 条 + 《证券及期货条例》第 XIVA 部:
- 上市公司须在**知悉**内幕消息后,在**合理切实可行范围内尽快**向公众披露
- "内幕消息"(inside information)定义：具体消息,如为公众所知,相当可能会对股价造成重大影响

### 4.2 业绩发布时限对比
| 报告类型 | A 股 | 港股 |
|---|---|---|
| 年度报告 | 4 个月 | **3 个月**(12月31日结→3月31日前) |
| 半年度/中期报告 | 2 个月 | **2 个月** |
| 季度报告 | 1 个月 | **不强制**(创业板已取消季度报告要求) |

### 4.3 港股业绩预告
港交所不强制要求业绩预告,但若公司自行发布盈利预警/盈喜,须遵守内幕消息披露的一般原则——在知悉后尽快披露。

## 五、北交所特色规则

| 规则项 | 核心要求 |
|---|---|
| 定期报告 | 年度报告(4个月)/ 半年度报告(2个月)/ 季度报告(1个月) |
| 业绩预告 | 净利润为负、扭亏为盈、净利润增减 50%以上、扣除后营收低于 1 亿元、净资产为负 (与深交所主板趋同) |
| 异常波动公告 | 连续 3 个交易日累计涨跌幅偏离 ±40% 须发公告 |
| 自愿披露 | 鼓励,但不允许选择性披露 |

---

## 六、违规后果

- **行政监管**：证监会责令改正/警告/罚款(信息披露义务人最高 1000 万元,直接责任人最高 500 万元)
- **自律监管**：交易所通报批评/公开谴责/认定不适合担任高管
- **民事赔偿**：虚假陈述民事赔偿责任(代表人诉讼制度)
- **刑事责任**：欺诈发行/违规披露重要信息罪(刑法第 160、161 条)
