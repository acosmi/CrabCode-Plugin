# KYC规则栅格评估

对客户执行多维合规评估：风险评级(低/中/高/禁止)+尽调等级(简化/标准/强化)+大额可疑交易报告。

> **反洗钱法**：`sources/laws/反洗钱法_客户身份识别要点.md`（2024修订, 2025.1.1生效）
> **尽调口径**：`sources/laws/客户尽调_可疑交易报告口径.md`（大额/可疑交易阈值）
> **UBO规则**：`sources/mappings/受益所有人识别规则_v1.md`（≥25%穿透标准）

## 操作步骤

### Step 1 — 客户风险评级

| 等级 | 判定条件 | 尽调要求 |
|------|---------|---------|
| 高风险 | FATF高风险地区/PEP/复杂离岸架构/负面洗钱报道/异常交易 | **强化尽调(EDD)** |
| 中风险 | 未触发高低条件 | **标准尽调(CDD)** |
| 低风险 | 政府机关/上市公司/低金额低频交易 | **简化尽调(SDD)** |

### Step 2 — 受益所有人识别（穿透≥25%）

**三标准**（满足任一即构成）：

| 标准 | 阈值 |
|------|------|
| 持股标准 | 直接/间接 ≥ **25%** 股权/表决权 |
| 协议控制 | 通过协议/代持等实质控制 ≥ 25% |
| 高管兜底 | 无法按前两项识别时，判定高级管理人员 |

穿透计算：多层持股按路径乘法（如 A→B 60% × B→目标 50% = 30% ≥ 25% → 构成）

**豁免**：A股/港股/美股等受严格监管上市公司可豁免穿透。

### Step 3 — 大额交易报告

| 交易类型 | 阈值 | 时限 |
|---------|------|------|
| 现金 | ≥ **5万元** | **5个工作日** |
| 对公转账 | ≥ **200万元** | 5个工作日 |
| 对私境内 | ≥ **50万元** | 5个工作日 |
| 对私跨境 | ≥ **20万元** | 5个工作日 |

### Step 4 — 可疑交易判定（18类指标）

- 拒绝提供身份信息/虚假证件
- 分散转入集中转出/频繁快进快出
- 交易金额与身份/职业/收入明显不符
- 刻意规避大额报告阈值（拆分交易/结构化）
- 复杂架构隐藏实际控制人

确认可疑后 **立即** 报告中国人民银行反洗钱监测分析中心。

### Step 5 — 输出评估矩阵

| 评估维度 | 结果 | 触发行动 |
|---------|------|---------|
| 客户风险评级 | 高/中/低 | EDD/CDD/SDD |
| 受益所有人识别 | 已识别/未 | 要求穿透 |
| 制裁名单 | 命中/未 | 拒绝/报备 |
| 大额交易 | 触发/未 | 5工作日报告 |
| 可疑交易 | 可疑/无 | 立即报告 |
| PEP状态 | 是/否 | EDD |
| **综合KYC结论** | | 通过/关注/补件/拒绝 |

### 合规声明

```
⚠️ 2024版反洗钱法(2025.1.1生效)：首次将特定非金融机构纳入反洗钱义务主体。
客户身份资料保存 ≥ 5年。交易记录保存 ≥ 5年。
受益所有人信息须报送人行受益所有人信息管理系统。
```
