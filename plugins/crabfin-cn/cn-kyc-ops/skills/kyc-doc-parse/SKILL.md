# 开户文档解析

依据反洗钱法(2025.1.1生效)自动解析开户文档，提取KYC核心信息。

> **身份识别依据**：`sources/laws/反洗钱法_客户身份识别要点.md`
> **字段清单**：`sources/mappings/开户文档字段清单_v1.csv`

## 操作步骤

### Step 1 — 客户身份识别（7类信息）

| # | 信息类别 | 自然人 | 法人/组织 |
|---|---------|--------|----------|
| 1 | 姓名/名称 | 身份证姓名 | 营业执照全称 |
| 2 | 证件号码 | 18位身份证号 | 18位统一社会信用代码 |
| 3 | 证件有效期 | 起止日期 | 营业执照有效期 |
| 4 | 国籍/注册地 | 国籍 | 注册地 |
| 5 | 职业/行业 | 职业 | 行业分类 |
| 6 | 联系方式 | 手机+地址 | 电话+注册/办公地址 |
| 7 | 受益所有人 | — | 穿透识别≥25%自然人 |

### Step 2 — 个人客户文档解析

| 文档 | 提取字段 | 验证规则 |
|------|---------|---------|
| 身份证 | 姓名/号码/有效期/签发机关/地址 | 18位校验码/有效期检查 |
| 手机号实名 | 手机号/运营商/入网时间 | 与身份信息一致 |
| 银行卡 | 卡号/开户行 | 一类户/二类户判断 |
| 收入资产证明 | 年收入/金融资产 | 按合格投资者标准校验 |
| 风险揭示书 | 签名/日期 | 不可为空 |

### Step 3 — 机构客户文档解析

| 文档 | 提取字段 | 验证规则 |
|------|---------|---------|
| 营业执照 | 名称/USCC/法定代表人/经营范围 | 国家企业信用信息公示系统核验 |
| 章程/合伙协议 | 股权结构/控制权 | 受益所有人穿透 |
| 受益所有人声明 | ≥25%自然人 | 见 `受益所有人识别规则_v1.md` |
| 法人代表身份证明 | 同个人KYC | |
| 授权委托书 | 经办人+授权范围 | |
| 经营场所证明 | 注册地址/经营地址 | |

### Step 4 — 自动校验

```
身份证：18位校验+出生日期合理性
统一信用代码：18位校验(含登记管理机关码)
证件有效期：是否过期(<30天警告)
年龄：满18周岁
风险测评：结果一致性
黑名单/制裁名单扫描
```

### Step 5 — 输出解析报告

```
客户类型/编号
必填字段完整性 ✅/⚠️（标注缺失项）
证件校验 ✅/⚠️（过期预警）
交叉校验 ✅/⚠️
受益所有人 ✅/⚠️
制裁扫描 ✅/❌
KYC结论：☐通过 ☐补件 ☐拒绝
```

### 合规声明

```
⚠️ 客户身份资料保存 ≥ 5年（业务关系结束后起算）。
解析结果须经人工复核，不可替代人工KYC审核。
```
