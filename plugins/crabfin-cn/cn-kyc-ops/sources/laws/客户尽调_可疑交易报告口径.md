# 客户尽职调查与可疑交易报告口径

> **文件全名**：《中华人民共和国反洗钱法》(2024年修订) / 《金融机构客户尽职调查和客户身份资料及交易记录保存管理办法》(中国人民银行/银保监会/证监会令2022年第1号) / 《金融机构大额交易和可疑交易报告管理办法》(中国人民银行令2016年第3号,2018年修正)
> **出处**：全国人大常委会 / 中国人民银行 / 国家金融监管总局
> **生效日期**：2025.1.1 (反洗钱法2024修订) / 2022.3.1 (尽调办法,部分条款暂缓)
> **版本**：2024年编撰版
>
> ⚠️ 红线：反洗钱为法定强制义务——漏报、迟报或应识别而未识别客户身份,机构和个人均可能面临行政处罚、刑事责任和声誉风险。使用前必须经合规/反洗钱部门确认口径。

---

## 一、客户尽职调查(CDD)核心要求

### 1.1 CDD 的三个等级

| 等级 | 触发场景 | 核心要求 |
|---|---|---|
| **简化尽职调查(SDD)** | 低风险客户(如政府机关、上市公司、受严格监管的金融机构) | 可简化部分身份识别措施,但不得免除身份识别的根本义务 |
| **标准尽职调查(CDD)** | 普通客户(大多数) | 识别客户身份 + 了解开户目的 + 了解资金来源/用途 |
| **强化尽职调查(EDD)** | 高风险客户(PEP、跨境代理行、来自FATF高风险/不合作司法辖区的客户、非面对面客户、复杂/异常交易) | 标准尽职调查 + 了解财富来源 + 了解资金来源 + 加强交易监控 + 提高审批层级 |

### 1.2 CDD 的五个时间节点

| 节点 | 义务 |
|---|---|
| **建立业务关系时** | 完成客户身份识别(不得先办业务后补身份) |
| **客户身份信息变更时** | 接收变更通知,核实新身份信息 |
| **客户行为/交易异常时** | 触发重新识别(如交易模式突变、涉及制裁名单) |
| **客户风险等级变动时** | 重新评估风险等级,触发对应级别的 CDD 措施 |
| **定期更新** | 高风险客户 **≤ 1 年**、中风险客户 **≤ 3 年**、低风险客户 **≤ 5 年**完成重新识别 |

### 1.3 识别要素清单(机构客户 vs 个人客户)

**个人客户必查项**：
- 姓名 + 性别 + 国籍 + 职业 + 住所地(经常居住地)或者工作单位地址
- 有效身份证件种类/号码/有效期限
- 联系方式(手机号码等)
- 开户目的(了解)
- 财富来源(高风险客户须核实)

**机构/非自然人客户必查项**：
- 名称 + 注册地址 + 经营范围
- 有效证件种类(营业执照)/统一社会信用代码/证照有效期
- 法定代表人/负责人/授权办理人员信息
- 控股股东/实际控制人/受益所有人(穿透至自然人)
- 董事会/高管层中外国政要(PEP)身份确认

### 1.4 不得免除此义务的情形

客户身份识别不得因以下情形免除：
- 委托第三方识别(最终责任仍在委托方)
- 客户来自境外(反而须更严格)
- 客户持有其他金融机构账户(须独立重新识别)
- 业务量小/金额低(仍须完成基础识别)

## 二、受益所有人识别(穿透标准)

> 详细内容见 `mappings/受益所有人识别规则_v1.md`

受益所有人识别口径简表：

| 识别标准 | 类型 | 判定 |
|---|---|---|
| 持股 ≥ 25% | 每个持股超25%的自然人 | 逐层穿透,直至自然人 |
| 协议/表决权控制 | 通过非持股方式实质控制 | 协议安排/一致行动/董事委派权等 |
| 兜底：高级管理人员 | 无法通过前两条识别时 | 法定代表人或主要负责人 |

## 三、大额交易报告口径

### 3.1 五档报告阈值

| 交易类型 | 报告标准 | 报告时限 |
|---|---|---|
| **现金收付** | 单笔或当日累计 ≥ **5 万元**人民币(或等值外币) | 交易发生后 **5 个工作日**内 |
| **对公账户转账** | 单笔或当日累计 ≥ **200 万元**人民币(或等值外币) | 交易发生后 5 个工作日内 |
| **对私账户转账(境内)** | 单笔或当日累计 ≥ **50 万元**人民币(或等值外币) | 交易发生后 5 个工作日内 |
| **对私账户转账(跨境)** | 单笔或当日累计 ≥ **20 万元**人民币(或等值外币) | 交易发生后 5 个工作日内 |
| **可疑交易(无论金额)** | 有**合理理由怀疑**资金来源或用途涉及洗钱/恐怖融资的,无论金额大小 | **立即**(一般不超过 24 小时,重大情况即时报送) |

### 3.2 大额交易报告责任主体
- 金融机构(含银行/证券/基金/信托/保险/支付机构等)
- 非银行支付机构
- 特定非金融机构(房地产/贵金属交易/会计师事务所/律师事务所等——依反洗钱法 2024 修订扩展范围)

## 四、可疑交易报告(STD)口径

### 4.1 "可疑"的判断逻辑(非机械规则)
与"大额交易"的自动化触发不同,**可疑交易的核心在于"怀疑"**,依赖人工经验+系统预警相结合：

| 判断维度 | 可疑信号(示例,非穷举) |
|---|---|
| **资金特征** | 资金快进快出、化整为零(分拆成多笔小金额规避报告)、与客户身份/职业/收入明显不符 |
| **交易模式** | 与过往交易行为明显背离；大量使用现金/预付卡等难以追踪的支付方式 |
| **地域特征** | 与来自 FATF 高风险国家/地区的对手方频繁往来；交易与客户注册地/经营范围明显不符 |
| **关联方** | 交易对手与制裁名单 / 负面新闻 / 犯罪嫌疑信息匹配 |
| **目的不明** | 无合理解释/不配合尽调/回避询问 |

### 4.2 报告流程
```
预警信号触发(系统+人工)
  → 初筛/研判(可疑交易分析岗,时限 T+3 个工作日,特殊情况可延)
  → 经反洗钱合规负责人批准
  → 向中国反洗钱监测分析中心(CAMLMAC)报告
  → 报告后继续监控(不得以已报告为由放松)
  → 保密义务——不得告知客户反洗钱报告(构成"tipping-off"违规)
```

## 五、客户身份资料与交易记录保存

| 资料类型 | 保存期限(最低) |
|---|---|
| 客户身份信息资料 | **业务关系结束后 5 年** |
| 客户身份识别过程性文件 | 业务关系结束后 5 年 |
| 交易记录(含交易指令/记录/凭证) | **交易发生后 5 年** |
| 大额交易/可疑交易报告留档 | 报告后 5 年 |

> 涉及反洗钱调查且在保存期限内的,保存至调查结束。

## 六、实务注意事项

### 6.1 对 CrabFin-CN `cn-kyc-ops` 技能的影响
- `kyc-doc-parse`：解析开户文档时,须提取上述"个人/机构客户必查项"全部字段,缺失字段须标记为待补
- `kyc-rules`：规则引擎须内置 CDD 分级(简/标/强)的触发条件,以及大额交易五档报告阈值的自动匹配
- 受益所有人识别：需支持持股 25% 的逐层穿透 + 表决权控制的联合判定
- 可疑交易：为辅助判断功能,系统可先覆盖大额交易自动筛查,可疑交易判定层提供"预警信号清单"供人工后续研判

### 6.2 新反洗钱法的核心变化(2024 修订 vs 旧法)
- **扩展义务主体**：明确覆盖特定非金融机构(房地产、贵金属、律师、会计师等)
- **加重法律责任**：大幅度提高行政罚款上限(旧法 500 万→新法可达数千万甚至上亿)
- **受益所有人透明度**：建立受益所有人信息备案制度,要求市场主体主动报备
- **虚拟资产**：专门提及虚拟资产洗钱风险管理
- **数据保护平衡**：反洗钱信息处理与个人信息保护法的协调
