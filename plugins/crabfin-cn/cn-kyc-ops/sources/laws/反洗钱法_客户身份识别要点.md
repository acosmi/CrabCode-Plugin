# 《反洗钱法》及客户身份识别管理办法要点

- **生效日期**：2025年1月1日

## 一、反洗钱法(2024修订)核心要点

### 1.1 适用范围(第二条/第三条)

| 适用主体 | 说明 |
|---------|------|
| **金融机构** | 银行、证券、期货、基金、保险、信托、支付机构等 |
| **特定非金融机构** | 房地产中介、贵金属/珠宝交易商、会计师、律师、公证处、公司注册代理等 |
| **单位和个人** | 任何单位和个人不得为洗钱活动提供便利 |

### 1.2 反洗钱义务

| 义务 | 核心要求 |
|------|---------|
| **客户身份识别/尽职调查** | 建立业务关系/单笔交易超限额/发现可疑交易时,须识别客户身份 |
| **大额交易报告** | 单笔或当日累计超过规定金额的,向反洗钱监测分析中心报告 |
| **可疑交易报告** | 对疑似洗钱/恐怖融资交易,及时向监测分析中心报告 |
| **资料保存** | 客户身份资料(业务关系结束后)+交易信息(交易结束后)≥ **10年**(《反洗钱法》2024修订第三十四条;旧法为5年,已上调) |
| **内部控制** | 建立健全反洗钱内控制度、设置反洗钱专岗、定期培训 |

### 1.3 2024年修订亮点

| 修订要点 | 说明 |
|---------|------|
| 扩大适用范围 | 新增**特定非金融机构**(律师、会计师、房地产中介等) |
| 强化受益所有人识别 | 明确法人/组织的**受益所有人**识别义务 |
| 加大处罚力度 | 大幅提高罚款上限 |
| 加强与反恐怖融资衔接 | 新增反恐怖融资专章 |
| 跨境资金监测 | 加强对跨境资金流动的监测和管控 |

## 二、客户身份识别/尽职调查(CDD)核心要求

### 2.1 必须进行客户身份识别的时机

| 时机 | 要求 |
|------|------|
| **建立业务关系时** | 必须识别、核对并登记客户身份信息 |
| **一次性交易超限额时** | 现金交易 ≥ 5万元人民币(或等值外币 **1万美元**) |
| **跨境汇款** | 任何金额均须登记汇款人/收款人完整信息 |
| **发现可疑交易时** | 不受金额和时间限制 |
| **客户身份信息变更** | 重新识别 |
| **定期复核** | 定期(通常年度)复核客户信息 |

### 2.2 自然人客户识别内容

| 必须登记的信息项 |
|-----------------|
| 姓名 |
| 性别 |
| 国籍 |
| 职业 |
| 住所地或工作单位地址 |
| 联系方式(电话/邮箱) |
| 有效身份证件的种类、号码和有效期限 |

### 2.3 法人/组织客户识别内容

| 必须登记的信息项 |
|-----------------|
| 名称 |
| 住所 |
| 经营范围 |
| 组织机构代码(统一社会信用代码) |
| 营业执照/登记证书信息(号码、有效期限) |
| **控股股东**或**实际控制人**信息 |
| **法定代表人**信息 |
| **受益所有人**信息(姓名、地址、身份证明、持股/控制比例) |

## 三、持续尽职调查与风险分类

### 3.1 客户风险等级分类(至少三级)

| 风险等级 | 描述 | 加强措施 |
|:---:|------|:---:|
| **高风险** | PEP(政治公众人物)、来自高风险国家、复杂结构、无实质经营 | **强化尽职调查**(EDD) |
| **中风险** | 一般企业客户、常规交易模式 | **标准尽职调查**(SDD) |
| **低风险** | 上市公司、政府实体、养老金/公积金类客户 | **简化尽职调查**(SDD-lite) |

### 3.2 强化尽职调查(EDD)措施

| 额外要求 |
|---------|
| 了解**交易目的**和**资金来源** |
| 获取**更高层级**的批准 |
| 加强交易的**持续监控** |
| 对资金来源进行**合理验证** |

## 四、大额交易报告标准

| 交易类型 | 报告阈值 |
|---------|:---:|
| 现金收付 | ≥ **5万元**人民币(或等值1万美元) |
| 法人间银行转账 | ≥ **200万元**人民币(或等值20万美元) |
| 个人间银行转账(境内) | ≥ **50万元**人民币(或等值10万美元) |
| 个人间银行转账(跨境) | ≥ **20万元**人民币(或等值1万美元) |
| 保险理赔/退保 | ≥ **5万元**人民币(或等值1万美元) |
