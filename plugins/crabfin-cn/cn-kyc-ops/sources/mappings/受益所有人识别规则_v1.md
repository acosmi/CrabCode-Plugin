# 受益所有人识别规则(穿透标准与判定阈值)

## 法规依据
- 《受益所有人信息管理办法》(中国人民银行令,2024年发布)
- 《金融机构客户尽职调查和客户身份资料及交易记录保存管理办法》(中国人民银行令〔2022〕第1号,2022年3月1日起施行)
- 《中华人民共和国反洗钱法》(2024年修订)
- FATF(反洗钱金融行动特别工作组)建议24：透明度和法人受益所有权

## 版本日期
2025-06-20

---

## 一、受益所有人定义(核心)

**受益所有人(Beneficial Owner)**是指最终拥有或**实际控制**法人/组织的自然人。

---

## 二、穿透判定标准(满足其一即可认定为受益所有人)

### 2.1 标准一：直接或间接持股阈值

| 持股方式 | 判定阈值 | 说明 |
|---------|:---:|------|
| 直接持股 | ≥ **25%** | 自然人直接持有法人股权/股份 ≥ 25% |
| 间接持股 | ≥ **25%** | 通过多层嵌套(多层公司)间接持有 ≥ 25% |
| 合计持股 | ≥ **25%** | 直接+间接合计 ≥ 25% |

> **穿透计算方法：** 多层持股按持股比例乘积计算，例如 A 持有 B 公司 60% 股权，B 持有 C 公司 50% 股权，则 A 对 C 的穿透持股 = 60% × 50% = 30% → 触发判定。

### 2.2 标准二：协议/其他方式实际控制

| 控制方式 | 说明 |
|---------|------|
| 协议控制(VIE) | 通过一致行动协议、表决权委托协议等对公司实施实际控制 |
| 人事控制 | 通过决定/否决董事会多数成员任免控制公司 |
| 重大事项一票否决权 | 对重大经营、投融资、资产处置、分红等决策有一票否决权 |
| 其他间接控制 | 通过亲属关系、一致行动人关系等实质上控制 |

### 2.3 标准三：高级管理人员(兜底条款)

| 条件 | 认定结果 |
|------|---------|
| 按前两个标准**均无法识别**受益所有人时 | 将法定代表人/执行事务合伙人/董事长/总经理等**主要负责人**认定为受益所有人 |

---

## 三、多层架构穿透规则

```
穿透义务：
┌─ 自然人A (实际拥有/控制)
│  └─ 中间层B (持股平台/控股公司)
│     └─ 中间层C (SPV/开曼主体)
│        └─ 运营实体D (在中国经营的WFOE)
│
识别结论：A → 受益所有人
```

**穿透要求：**
- 无论中间经过**多少层**法人/信托/合伙/基金，均需穿透至**最终的自然人**
- 不得以"境外上市公司""境外信托""代持"等作为不穿透的理由
- 对非上市公司、多层嵌套架构应**逐层排查**

---

## 四、受益所有人识别判定矩阵

| 场景 | 判定路径 | 示例 |
|------|---------|------|
| 单一自然人持股 ≥ 25% | 直接识别为该自然人 | 张三持股60%→张三 |
| 多名自然人各自持股 < 25% 但合计 ≥ 25% | 识别所有 ≥ 10%的自然人 + 实际控制判定 | 张三18%+李四15%→两人均为受益所有人 |
| 无任一自然人 ≥ 25%，但有协议控制 | 按协议控制方识别 | 张三持10%但有40%表决权→张三 |
| 无任何标准满足 | 识别主要负责人 | 法定代表人 |
| 信托/基金等 | 信托委托人/受益人/保护人 或 基金管理人/GP | 须识别有效控制者 |
| 复杂多层境外架构 | 穿透至最终自然人 | 离岸公司→BVI→HK→WFOE→识别最终自然人 |

---

## 五、豁免情形(免于识别受益所有人的实体)

| 可豁免的实体类型 | 条件 |
|----------------|------|
| 上市公司及其子公司 | 已在境内外主要证券交易所**上市**(含信息披露义务) |
| 政府机构/机关 | 依法设立的政府机关、事业单位 |
| 国有独资公司/全资子公司 | 国有独资/全资(不适用豁免场景须核实) |

> 注意：豁免仅适用于"免于识别"受益所有人，但其他客户身份识别义务(基本信息登记、交易监控等)仍需履行。

---

## 六、持续监控与更新

| 触发条件 | 要求 |
|---------|------|
| 客户股权结构发生变更(≥25%以上股东变更) | 重新识别受益所有人 |
| 法人代表/主要负责人变更 | 重新确认受益所有人 |
| 客户经营范围/业务模式重大变化 | 重新评估风险等级 |
| 定期复核 | ≥ 每年1次(高风险客户≥ 每半年) |

---

## 七、违规后果

| 违规情形 | 行政处罚 |
|---------|---------|
| 未识别受益所有人 | 责令改正 → 罚款(个人/机构) |
| 故意隐瞒/虚假申报 | 情节严重可吊销许可证 |
| 造成严重后果(洗钱等) | 移交司法机关追究刑事责任 |

---

> **红线提醒：**
> - 受益所有人识别是KYC/AML的核心义务之一，不可走过场。
> - 25%阈值以外的"以其他方式实际控制"标准容易被忽略(尤其是VIE、协议控制等)。
> - FATF第四轮互评估(2019年)已明确要求中国加强受益所有人透明度。2024年反洗钱法修订+受益所有人信息管理办法进一步强化了此义务。
> - 具体操作请以中国人民银行反洗钱局最新指导文件和合规官意见为准。
