# 理财产品销售管理规则与适当性匹配要求

> **文件全名**：《理财公司理财产品销售管理暂行办法》(银保监会令2021年第4号) / 《商业银行理财业务监督管理办法》(银保监会令2018年第6号) / 《证券期货投资者适当性管理办法》(证监会令第130号,2020年修正)
> **出处**：中国银保监会(现国家金融监管总局) / 中国证监会
> **生效日期**：2021.6.27 (理财销售新规,核心条款) / 2017.7.1+2020修正(适当性办法)
> **版本**：2024年编撰版
>
> ⚠️ 红线：销售适当性违规已是监管检查的重中之重。错配风险等级可能触发"卖者尽责"推翻→民事赔偿。使用前请经合规部门核对。

## 一、理财产品销售核心规则(银保监 2021年第4号令要点)

### 1.1 销售机构准入
仅限两类机构代销理财公司发行的理财产品：
- **吸收公众存款的银行业金融机构**(含商业银行、农村合作银行等)
- **理财公司**(即理财子公司,可销售本公司和母公司发行的产品)

> ⚠️ 非银金融机构(证券/基金/保险/信托)**不能代销理财公司产品**(但可销售各自监管体系下的资管产品)。

### 1.2 销售行为红线(禁止行为清单)

| 类别 | 禁止行为 |
|---|---|
| **误导销售** | 虚假宣传/承诺收益/夸大过往业绩/使用"无风险""保本"等表述 |
| **捆绑销售** | 强制搭售/将销售业绩与绩效直接挂钩 |
| **诱导销售** | 提供礼金/礼品/IP 合作等利益诱惑 |
| **"飞单"** | 销售非本机构发行或代销的金融产品 |
| **规避双录** | 规避录音录像要求的操作 |
| **私售** | 理财经理私下推荐、承诺回报 |

### 1.3 销售专区与"双录"

| 要求 | 内容 |
|---|---|
| 销售专区 | 理财销售须在银行网点**专门区域**进行,不得在普通柜面或非专属区域 |
| 录音录像(双录) | **全部**理财产品销售均须进行录音录像(含自助终端) |
| 双录保存 | 保存期限至**产品终止后 6 个月**(部分地方局要求更长) |
| 远程销售 | 须通过机构官方 APP/网上银行,且必须嵌入双录功能 |

### 1.4 冷静期(仅私募类产品)
- 私募理财产品的投资者有权在划款后 **24 小时**内解除合同
- 销售机构须无退还投资款项(期间不计收益)

## 二、适当性管理核心要求(证监会口径)

### 2.1 五级风险等级匹配体系

当前交叉领域存在两套并行的适当性体系(银保监 vs 证监会),但实务中正在趋同：

**投资者分类(C1-C5)**：

| 投资者等级 | 风险承受能力 | 匹配产品等级 |
|---|---|---|
| **C1**(保守型,含**最低风险承受能力**) | 极低 | **R1** (低风险) |
| **C2**(相对保守型) | 较低 | R1、**R2** (中低风险) |
| **C3**(稳健型) | 中等 | R1、R2、**R3** (中等风险) |
| **C4**(相对积极型) | 较高 | R1-R4 |
| **C5**(积极型) | 高 | R1-R5 |

**产品风险等级(R1-R5)**：

| 风险等级 | 产品特征 | 典型产品 |
|---|---|---|
| **R1**(低风险) | 保障本金不受损失,预期收益可货币实现 | 存款、国债、货币基金 |
| **R2**(中低风险) | 本金风险较低,收益有一定波动 | 短债基金、标准债券型理财 |
| **R3**(中等风险) | 有一定本金风险,收益波动可预期 | 混合型基金、"固收+"理财 |
| **R4**(中高风险) | 本金风险较高,收益波动较大 | 偏股型基金、权益类理财 |
| **R5**(高风险) | 本金风险极高,收益波动剧烈 | 衍生品基金、杠杆类产品 |

### 2.2 匹配规则红线

| 核心规则 | 说明 |
|---|---|
| **C1 最低风险承受者** | 只能购买 R1 产品,不得购买 R1 以上 |
| **其他投资者** | 可购买**不高于**其风险承受能力的产品 |
| **越级购买** | 如 C2 投资者要求购买 R3 产品,须特别书面警示,且须销售机构**高风险产品准入委员会/专人审批** |
| **禁止主动推介** | 禁止向普通投资者主动推介**风险等级高于**其承受能力或**不符合**其投资目标的产品 |

**证券期货经营机构须履行的适当性义务**：
1. 了解投资者(KYC)——身份、财务状况、投资经验、风险偏好
2. 了解产品(KYP)——产品结构、风险等级、流动性
3. 匹配(Matching)——将合适的产品销售给合适的投资者
4. 告知(Disclosure)——充分揭示风险
5. 留痕(Record-keeping)——保留全部适当性管理记录

## 三、实务要点与 CrabFin-CN 影响

### 3.1 适当性匹配矩阵(穿透到子板块)

| 理财类型 | 风险等级 | 典型适用场景 |
|---|---|---|
| 现金管理类理财 | R1 | `portfolio-rebalance` 现金端 |
| 纯债类理财 | R2 | `financial-plan` 固收部分 |
| "固收+" / 混合类理财 | R3 | `investment-proposal` 核心推荐 |
| 权益类理财 / 私募股票基金 | R4 | `cn-private-equity` 尽调阶段关注 |
| 场外衍生品 / 杠杆类 | R5 | `cn-wealth` 仅限 C5 客户 |

### 3.2 对辅助技能的设计影响
- `investment-proposal`：生成投资方案时必须嵌入适当性匹配检查——产品风险等级不得超过客户风险等级
- `client-review`：重点检查客户风险等级是否发生变化(生命周期/财务状况变动),触发重新测评
- `financial-plan`：资产配置方案须在适当性边界内,出现越级配置须有揭示语

---

> **核对提示**：适当性管理在两套监管框架(银保监/证监会)下存在用语与技术口径差异(如投资者分类命名方式不同),但核心匹配逻辑趋同。使用前请以最新监管规则为准,建议与合规部门核对具体产品分类。
