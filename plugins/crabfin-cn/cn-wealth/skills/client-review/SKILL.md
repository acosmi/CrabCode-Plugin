# 客户回顾/复盘

对客户进行定期回顾，涵盖风险画像复核、持仓适当性扫描和产品合规检查。

> **适当性依据**：`sources/laws/投资者适当性管理办法要点.md`（第六条KYC/第十条重评/第十九条匹配/第二十二条禁止主动推介）
> **资管新规**：`sources/laws/资管新规_核心条款.md`（第十五条资金池/第十八条净值化/第十九条期限匹配/第二十一条嵌套）
> **销售规则**：`sources/laws/理财销售管理规则.md`（双录/冷静期/禁止行为）

## 操作步骤

### Step 1 — 客户画像复核（KYC Refresh）

**输入**：客户最新信息
**输出**（依据《证券期货投资者适当性管理办法》第六条）：

| 维度 | 核验要点 | 变化触发标准 |
|------|---------|------------|
| 基本信息 | 年龄/职业/婚姻 | 跨人生阶段（如退休） |
| 收入资产 | 收入来源/数额/金融净资产 | 年收入变化 > 30% 或跌破合格投资者标准 |
| 负债状况 | 贷款余额/月供 | 月供/收入 > 50% |
| 投资经验 | 年限/品种覆盖 | — |
| 风险偏好 | 主观意愿 | 客户主动申请调整 |
| 诚信记录 | 失信/违规 | 出现负面记录 |

### Step 2 — 风险等级重新评定

**输入**：Step 1 复核结果
**输出**：C1-C5 五级评定

| 等级 | 可匹配产品 |
|------|-----------|
| C1 保守型（含最低风险承受者） | 仅 R1 |
| C2 谨慎型 | R1, R2 |
| C3 稳健型 | R1, R2, R3 |
| C4 进取型 | R1-R4 |
| C5 激进型 | R1-R5 |

**特别规则**：
- **C1 最低风险承受者**：绝对不得购买 > R1 产品（法律法规强制性禁止）
- 普通投资者（除C5外）购买高于其风险等级产品 → 特别风险警示 + 书面确认
- **禁止**向普通投资者主动推介高于其风险等级的产品（第二十二条）

### Step 3 — 持仓适当性扫描

**输入**：客户现有持仓 × 新版C级
**输出**：

1. **风险等级匹配检查**：逐产品 R级 ≤ 客户 C级匹配上限
2. **投资期限检查**：封闭期 ≤ 客户投资期限
3. **集中度检查**：单一产品 > 20% / 单一管理人 > 30% 须标注
4. **越级处理**：R级 > C级 → **高风险行动项**，建议赎回或降仓

**越级调整时限**：触发重评且新等级低于原等级 → 越级产品须在 **30个交易日**内调整。

### Step 4 — 产品合规检查（资管新规视角）

**输入**：持仓产品元数据
**输出**（来自 `sources/laws/资管新规_核心条款.md`）：

| 合规维度 | 法规 | 检查要点 |
|---------|------|---------|
| 净值化管理 | 第十八条 | 是否净值化，公允价值计量 |
| 嵌套层数 | 第二十一条 | 不超过一层（公募基金除外） |
| 资金池禁止 | 第十五条 | 三单管理/无滚动发行分离定价 |
| 期限匹配 | 第十九条 | 非标到期 ≤ 产品到期/最近开放日 |
| 刚兑禁止 | 第十八条 | 无保本保收益承诺/滚动发行 |

### Step 5 — 销售适当性回检

**输入**：销售记录
**输出**（来自 `sources/laws/理财销售管理规则.md`）：

| 检查项 | 标准 |
|--------|------|
| 销售双录（录音录像） | 已完成 |
| 双录保存期限 | ≥ 产品终止后 6个月 |
| 私募产品冷静期 | **24小时** 冷静期已执行 |
| 捆绑/诱导销售 | 禁止 |
| C1最低风险承受者推介 > R1 | 红线禁止 |

### Step 6 — 输出回顾报告

**输出结构**：
1. 客户画像变更摘要
2. 风险等级评定结果（原→新）
3. 适当性扫描结果（匹配/越级产品清单）
4. 资管新规合规检查
5. 行动计划（重评/产品调整/双录补录）

### Step 7 — 合规声明

```
⚠️ 客户风险测评有效期不超过 2年，到期须强制重测。
适当性匹配资料保存期限 ≥ 20年（《适当性管理办法》第二十五条）。
越级产品须在触发重评后 30个交易日内调整。
各地银保监/证监局对双录保存期限可能有更严格要求，以地方口径为准。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `client-report` | 回顾→客户报告 | 组合回顾报告 |
| `financial-plan` | 等级变更→调整方案 | 新SAA方案 |
| `portfolio-rebalance` | 越级/集中度→调整 | 调仓方案 |
