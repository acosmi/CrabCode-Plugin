# 投资方案

依据 `sources/templates/投资方案_脱敏模板.md` 标准格式，生成含客户画像/资产诊断/目标配置/产品推荐/实施路径/风险揭示的完整投资方案。**红线：生成前必须完成适当性匹配校验。**

> **适当性依据**：`sources/laws/投资者适当性管理办法要点.md`（第十九条匹配/第二十二条禁止主动推介）
> **销售规则**：`sources/laws/理财销售管理规则.md`（双录/冷静期24小时）
> **资管新规**：`sources/laws/资管新规_核心条款.md`（第十八条净值化/第五条合格投资者）
> **个税口径**：`sources/mappings/个税口径_v1.md`

## 操作步骤

### Step 1 — 客户画像摘要

**输入**：客户KYC数据
**输出**：

| 维度 | 内容 |
|------|------|
| 年龄/人生阶段 | |
| 金融资产规模 | |
| 负债情况 | |
| 风险承受等级 | **C1-C5** |
| 投资经验 | |
| 投资目标与期限 | |
| 流动性需求 | |
| 专项扣除 | |

### Step 2 — 现有资产诊断

**输入**：客户当前持仓
**输出**：
- 现金类过高→收益损耗
- 固收同质化→底层资产重叠
- 权益过度集中→单一标的 > 20%
- **适当性越级检查**：是否存在产品 R 级 > 客户 C 级
- 个税优化空间

### Step 3 — 适当性匹配校验（强制——推荐前必须执行）

**输入**：客户C级 × 拟推荐产品R级
**输出**：逐产品交叉比对（来自 `sources/laws/投资者适当性管理办法要点.md` 第十九条）

| 客户等级 | 允许购买最高R级 |
|---------|---------------|
| C1 | R1（法定红线，不得突破） |
| C2 | R2 |
| C3 | R3 |
| C4 | R4 |
| C5 | R5 |

**校验规则**：
1. 每只推荐产品的 R 级 ≤ 客户 C 级匹配上限
2. **禁止**向 C1 推荐任何 > R1 产品——法定红线
3. **禁止**向普通投资者主动推介高于其风险等级的产品（第二十二条）
4. 客户坚持越级购买 → 标注"须特别书面警示+客户书面确认"

**停止条件**：任一产品违反规则1/2 → 从方案中移除，不可推荐。

### Step 4 — 目标配置方案（SAA）

**输入**：客户目标/期限
**输出**：五类资产目标权重（现金管理/固收/权益/另类/保险保障），含预期收益区间和历史最大回撤。

### Step 5 — 具体产品建议

**输入**：SAA × 产品库
**输出**：逐产品填：名称/发行机构/风险等级/金额/期限/预期收益/推荐理由。末尾打印适当性确认语。

**数据不可得fallback**：产品库信息不全 → 标注"产品信息待补充"，仅给出产品类型建议。

### Step 6 — 实施路径

| 要素 | 内容 |
|------|------|
| 调仓操作 | 赎回/申购的标的、金额、费率 |
| 税优建议 | 年终奖年金化/专项扣除/个人养老金12,000元/年 |
| 双录安排 | 全部销售须完成双录（保存 ≥ 产品终止后6个月） |
| 冷静期 | 私募产品：**24小时** 冷静期 |
| 费用披露 | 申购/赎回/管理费率+利益冲突声明 |

### Step 7 — 风险揭示与签章

```
⚠️ 非保本声明（资管新规第十八条）：不得承诺保本保收益。过往业绩不代表未来表现。
利益冲突声明：本机构产品收费说明。适当性声明：风险等级匹配确认。
签章区：客户签字/客户经理签字/日期。
```

### Step 8 — 合规声明

```
⚠️ 全量销售过程须录音录像留痕，双录保存至产品终止后 ≥ 6个月。
客户风险测评有效期 ≤ 2年，过期须先重测。
方案须经投顾人工复核签字后方可交付客户。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `client-review` | 生成前适当性复核 | 最新C级/持仓扫描 |
| `financial-plan` | 整体资产配置 | SAA方案 |
| `portfolio-rebalance` | 现有持仓偏离 | 调仓方案 |
| `tax-loss-harvesting` | 税务优化 | 税优建议清单 |
