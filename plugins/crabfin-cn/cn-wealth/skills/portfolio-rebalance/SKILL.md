# 组合再平衡

对客户投资组合执行偏离度检查、再平衡方案生成、合规校验和税务影响评估。

> **资管新规**：`sources/laws/资管新规_核心条款.md`（嵌套/资金池/净值化/期限匹配）
> **适当性依据**：`sources/laws/投资者适当性管理办法要点.md`（第十九条匹配规则）
> **个税口径**：`sources/mappings/个税口径_v1.md`（转让暂免/股息税率/财产转让20%）

## 操作步骤

### Step 1 — 偏离度计算

**输入**：目标SAA vs 实际持仓
**输出**：逐类偏离度

| 触发条件 | 阈值 |
|---------|------|
| 大类偏离 | 绝对 > 5个百分点 或 相对 > 15% |
| 集中度超标 | 单一产品 > 20% / 单一管理人 > 30% |
| 风险等级漂移 | 客户C级变化致持仓越级 |
| 产品R级上调 | 上调后超过客户C级匹配上限 |
| 定期再平衡 | 季度/半年度定期 |

若全部在阈值内 → 输出"无需再平衡" + 下次检查时间，流程终止。

### Step 2 — 产品可交易性检查

**输入**：拟增减产品
**输出**：合规扫描（来自 `sources/laws/资管新规_核心条款.md`）：

| 检查项 | 法规 | 不合规处理 |
|--------|------|-----------|
| 嵌套层数 | 第二十一条 | 标记不可投，说明原因 |
| 资金池 | 第十五条 | 排除（须三单管理） |
| 净值化 | 第十八条 | 排除（禁止刚兑承诺） |
| 期限匹配 | 第十九条 | 标注期限不匹配风险 |

### Step 3 — 适当性复验

**输入**：再平衡后组合
**输出**：
- 每只产品 R级 ≤ 客户 C级匹配上限？
- 再平衡导致新越级配置？→ 标记"不可执行"，生成替代方案
- C1最低风险承受者持仓中 > R1 产品 → **优先降仓**

### Step 4 — 调仓方案生成

**输入**：偏离度 + 可交易性 + 适当性
**输出**：逐项操作（方向/标的/金额/费率/费用）

**优先级**：
1. 红线项（越级→优先减持或清仓）
2. 集中度超标（减至目标权重）
3. 大类偏离（按偏离幅度排序）
4. 优化项（替换同类低费率/好业绩产品）

### Step 5 — 中国税务影响评估

**输入**：每项调仓操作
**输出**（来自 `sources/mappings/个税口径_v1.md`）：

| 资产类型 | 转让所得 | 操作影响 |
|---------|---------|---------|
| A股/公募基金 | **暂免** | 无直接税务影响 |
| 私募基金份额 | **20%** | 转让价−成本−税费 |
| 未上市股权 | **20%** | 同上 |
| 新三板 | **暂免**（参照A股） | 无直接影响 |
| 港股通 | 暂免；股息 20% | 关注股息率 |
| 债券/理财 | 利息 20%（国债/地方债/储蓄存款利息免税） | 税后收益差异 |

**股息持股时间管理**（财税〔2015〕101号）：
- 持有 > 1年 → 股息**免税**；卖出持有 > 1年的权益仓位 → 标注"丧失后续分红免税资格"
- 持有接近1年（如10-11个月）→ 标注"距免税差X个月，建议持有至满1年"

### Step 6 — 综合所得边际税率提示

调仓产生显著收益 → 提示对综合所得边际税率的影响。边际税率 ≥ 20% 时，建议评估是否递延部分调仓至下一纳税年度。

### Step 7 — 输出再平衡报告

1. 偏离度分析表 2. 合规扫描 3. 适当性复验 4. 调仓清单（含费率/费用/税务标注）5. 税务影响汇总 6. 执行时间建议

### Step 8 — 合规声明

```
⚠️ 强制再平衡（越级产品清仓）不受"跨年度递延"策略约束——合规优先于税务优化。
A股转让暂免个税政策仍在延续，但私募基金份额/未上市股权转让须严格按20%计税。
各地对合伙制基金LP分配的执行口径存在差异，涉及私募再平衡时须注明。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `financial-plan` | 目标SAA | 基准配置权重 |
| `client-review` | 风险等级变更 | 最新C级 |
| `tax-loss-harvesting` | 税务影响评估 | 详细税优方案 |
| `investment-proposal` | 再平衡→新方案 | 更新后的投资方案 |
