# 客户报告

按 `sources/templates/客户报告_脱敏模板.md` 七段式结构为客户生成标准化组合回顾报告。

> **适当性依据**：`sources/laws/投资者适当性管理办法要点.md`（第十九条产品匹配/第十条画像变更）
> **资管新规**：`sources/laws/资管新规_核心条款.md`（第十八条刚兑禁止/第五条合格投资者）
> **个税口径**：`sources/mappings/个税口径_v1.md`（股息税率/专项扣除）

## 操作步骤

### Step 1 — 组合概览

**输入**：客户持仓数据
**输出**：

| 指标 | 值 |
|------|-----|
| 组合总市值（期初/期末/变化） | |
| 当期净流入/流出 | |
| 当期收益与收益率（含价差+分红） | |
| 年化收益率（近1年/近3年） | |
| 最大回撤 | |
| 夏普比率（近1年） | |

收益归因按固收类（票息+价差）、权益类（股息+价差）、另类、货币/现金四维度拆分。

**停止条件**：无持仓数据 → 停止生成，请用户提供数据。

### Step 2 — 资产配置 vs 目标

**输入**：目标配置 vs 实际持仓
**输出**：各资产类别偏离度（百分点）。再平衡阈值：目标权重 ±5个百分点 或 ±15% 相对偏离。

**适当性校验（强制）**（来自 `sources/laws/投资者适当性管理办法要点.md`）：

| 检查项 | 标准 | 状态 |
|--------|------|------|
| 产品R级 ≤ 客户C级 | R1-R5 vs C1-C5 | |
| 产品期限 ≤ 客户投资期限 | 封闭期/建议持有期 | |
| 集中度 | 单一产品 ≤ 20% / 单一发行人 ≤ 30% | |

若出现 R 级超过 C 级 → **列为首要高风险行动项**，建议立即赎回或转换。

### Step 3 — 产品表现详情

**输入**：各持仓产品表现数据
**输出**：逐产品：类型/风险等级/持仓金额与占比/当期收益与收益率/同类排名（百分位）

**需关注产品清单**（任一触发）：
- 业绩连续3季度落后同类中位
- 基金经理变更
- 基金规模缩减 > 30%
- 产品风险等级上调
- 超出客户最大回撤容忍线

### Step 4 — 市场回顾与环境展望

**输出**：各基准指数表现（中债综合/沪深300/中证500/创业板指/恒生指数/标普500/黄金AU9999）+ 下期资产评级（超配/标配/低配）。

### Step 5 — 客户画像变更核查

**输入**：客户上次 KYC vs 当前状态
**输出**（来自 `sources/laws/投资者适当性管理办法要点.md` 第十条）：

检查：风险承受等级/投资目标/投资期限/收入负债状况/流动性需求。
若变化 → 触发重新风险测评，根据新等级调整配置。

### Step 6 — 行动建议（四级优先级）

| 优先级 | 类型 | 示例 |
|--------|------|------|
| 高 | 适当性红线 | R级越级须立即赎回 |
| 中 | 再平衡 | 权益偏离 > 5ppt |
| 低 | 税务提示 | 年末专项附加扣除确认 |
| 信息 | 日程提醒 | 产品到期/养老金入账 |

### Step 7 — 风险揭示与声明

```
⚠️ 本报告仅供客户使用，不构成投资、法律或税务建议。过往业绩不代表未来表现。
投资有风险，请根据您的风险承受能力和投资目标审慎决策。
除存款类产品外，所有资管产品均非保本保收益（资管新规第十八条）。
如涉及私募产品，须额外标注合格投资者要求：
家庭金融净资产 ≥ 300万元 或 金融资产 ≥ 500万元 或 近3年人均年收入 ≥ 40万元。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `client-review` | 报告前的客户适当性复核 | KYC变更/风险等级更新 |
| `financial-plan` | 需要调整资产配置 | SAA方案 |
| `portfolio-rebalance` | 偏离度超阈值 | 再平衡方案 |
| `tax-loss-harvesting` | 年末税务优化 | 税务提示清单 |
