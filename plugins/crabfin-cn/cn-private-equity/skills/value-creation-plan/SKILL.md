# 价值创造计划

基于中国私募股权投资实践，为被投企业制定系统化的投后价值创造计划。核心逻辑是将对赌条款中的业绩目标转化为可执行的价值创造路线图，同时满足AMAC投后管理的合规要求。

## 一、价值创造框架：三条路径

```
价值创造 = 运营优化 + 战略升级 + 资本运作
```

| 路径 | 目标 | 时间跨度 | 典型举措 |
|------|------|---------|---------|
| 运营优化 | 提升利润率、改善现金流 | 6-18个月 | 降本增效、流程优化、数字化升级 |
| 战略升级 | 扩大市场规模、强化竞争壁垒 | 12-36个月 | 产品线扩展、区域扩张、第二增长曲线 |
| 资本运作 | 优化资本结构、对接资本市场 | 18-60个月 | 后续融资、并购整合、IPO准备 |

## 二、对赌目标与价值创造计划的对齐

依据 `sources/laws/对赌回购可执行性要点.md`（九民纪要第5条），被投企业的对赌/业绩承诺条款是价值创造计划的核心锚点：

### 2.1 对赌目标拆解

将对赌协议中的业绩承诺分解为可执行的价值创造举措：

| 对赌条款类型 | 承诺内容示例 | 价值创造举措 |
|-------------|-------------|-------------|
| 业绩补偿型 | 第T年度经审计净利润 ≥ [X]万元 | 收入增长计划 + 费用管控计划 |
| IPO时限型 | [YYYY-MM-DD]前完成合格IPO | 上市路径规划 + 合规整改 + 财务规范 |
| 业务里程碑型 | 达到某项运营指标 | 专项业务攻坚计划 |

### 2.2 对赌里程碑时间表

建立对赌目标的逆向时间线：

```
对赌到期日 [YYYY-MM-DD]
← 提前6个月：预评估对赌完成概率，启动补救计划
← 提前12个月：中期评估，必要时调整经营策略
← 提前18个月：完成核心价值创造举措的部署
← 投资完成日：价值创造计划启动
```

### 2.3 对赌条款的可执行性约束

价值创造计划须考虑对赌条款的司法可执行性约束：

- **与创始股东对赌**：可强制执行，价值创造计划应聚焦帮助创始股东完成业绩目标
- **目标公司直接回购**：须先完成减资程序（公司法第177条），价值创造计划中应包含减资预案准备
- **回购利率区间**：约定年化利率在 **10%以下** 的执行风险低于 **15%以上** 的条款；若已达高利率区间，在价值创造计划中应考虑协商调整空间

## 三、运营优化路径

### 3.1 收入增长举措

- 销售效率提升：优化销售漏斗转化率，缩短销售周期
- 定价策略优化：基于单位经济分析调整定价（参见 `unit-economics` 技能）
- 客户留存：建立客户成功体系，将NRR提升至 > **100%**
- 交叉销售/向上销售：在存量客户中挖掘增量收入

### 3.2 成本优化举措

- 采购成本控制：集中采购、供应商谈判、替代材料评估
- 人力效率：人效指标（人均收入/人均利润）对标行业75分位值
- 技术降本：自动化替代人工、系统迁移至低成本架构
- 费用管控：差旅/办公/营销费用的ROI导向管理

### 3.3 现金流管理

- 应收账款管理：DSO（应收账款周转天数）优化目标
- 存货管理：库存周转率提升计划
- 付款条款优化：在不损害供应商关系前提下延长应付账款周期
- 现金跑道延长：确保现金跑道 ≥ **12个月**

## 四、战略升级路径

### 4.1 产品/服务升级
- 从单品到解决方案：提升客单价和客户粘性
- 从项目制到产品化：提升可复制性和毛利率
- 从标准化到定制化：攻击高端市场

### 4.2 区域/市场扩张
- 国内市场扩张：从一线城市下沉到新一线/二线城市
- 海外市场拓展：东南亚/中东/拉美等新兴市场
- 行业扩张：从单一行业切入相邻行业（跨行业解决方案）

### 4.3 第二增长曲线
- 基于核心能力的延伸业务
- 数据/流量变现的新商业模式
- 平台化转型（从自营到平台）

## 五、资本运作路径

### 5.1 后续融资规划
- 下一轮融资的时间窗口和估值目标
- 融资材料准备（BP更新/数据包/财务模型）
- 潜在投资者long-list → short-list → 接触计划

### 5.2 并购整合
- 横向整合（收购竞对，扩大市场份额）
- 纵向整合（收购上下游，提升产业链话语权）
- 技术/人才并购（获得新技术/核心团队）

### 5.3 IPO准备
若对赌目标包含IPO，须前置以下准备工作：
- 上市板块选择（科创板/创业板/北交所/港股/美股）
- 上市中介选聘（券商/律所/审计）
- 三年一期财务规范性整改
- 关联交易规范和披露
- 对赌条款清理（涉及发行人作为对赌义务人的须在申报审计截止日前彻底终止）

## 六、AMAC投后合规

依据 `sources/laws/AMAC私募备案要求.md`（中基协发〔2023〕5号），价值创造计划的实施须与AMAC投后合规要求保持一致：

### 6.1 合规报送义务

- 季度更新（每季度结束后**10个工作日**内）：价值创造计划的推进进展应在季度更新中体现
- 年度报告（每年度结束后**4个月**内）：价值创造计划的年度成果总结
- 重大事项报告（发生后**10个工作日**内）：如价值创造计划涉及重大投资/退出/重组

### 6.2 合规红线

- 价值创造计划不得涉及与私募基金冲突的业务（民间借贷/配资/P2P等）
- 关联交易须经内部审批和信息披露
- 不得利用基金资产为被投企业提供变相刚兑或保本保收益安排

### 6.3 投资者适当性

- 价值创造计划中对LP的信息披露须与其风险等级匹配
- 自然人LP（金融资产 ≥ **300万元** 或近3年年均收入 ≥ **50万元**）与机构LP（净资产 ≥ **1000万元**）可能要求不同颗粒度的投后报告

## 八、成功标准与退出对接

价值创造计划的最终成功标准是退出时实现目标回报：
- MOIC ≥ 基金目标倍数（通常 **2.0-3.0x**）
- IRR > 基金门槛收益率（通常 **8-10%**）
- 对赌条款全部达标或妥善解决

退出前 **6-12个月**，价值创造计划重心从"增长"转向"退出准备"：
- 财务数据包装（审计质量提升、非经常性损益剥离）
- 管理团队展示（团队完整性、继任计划）
- 退出窗口判断（资本市场窗口、行业并购热度）

> 价值创造计划是投资投后管理的核心工具。好的价值创造计划不是"愿望清单"，而是基于被投企业实际情况、对赌条款约束、市场条件和基金退出时间表的有明确时间线和KPI的可执行方案。本技能应与 `portfolio-monitoring` 和 `returns-analysis` 联动，构成"计划-监控-退出"的投后管理闭环。
