# 项目筛选

基于中国私募基金监管体系和司法实践，对潜在投资项目进行系统化的初步筛选。筛选框架整合基金策略约束、监管准入红线、退出路径可行性和投资者适当性要求。

## 一、筛选框架：四层漏斗

```
第一层：基金策略匹配 → 第二层：行业准入合规 → 第三层：退出路径可行 → 第四层：LP约束检查
```

## 二、第一层：基金策略匹配

### 2.1 基金关键参数核对

| 参数 | 检查要点 |
|------|---------|
| 投资阶段 | 标的融资轮次是否在基金授权范围内（天使/A/B/C/Pre-IPO） |
| 投资金额 | 本轮意向投资额是否在基金单笔投资限额内 |
| 行业方向 | 标的所在行业是否属于基金投资策略覆盖范围 |
| 地理范围 | 标的注册地/主要运营地是否在基金授权区域 |
| 投资工具 | 基金是否允许使用意向投资工具（普通股/优先股/可转债/SAFE） |

### 2.2 基金期限匹配
- 基金剩余存续期（投资期+退出期+延长期）是否≥标的预期退出时间+2年 buffer
- 参考 `sources/laws/AMAC私募备案要求.md`：基金合同必须包含存续期限

## 三、第二层：行业准入合规

依据 `sources/laws/AMAC私募备案要求.md`（中基协发〔2023〕5号），核查以下红线：

### 3.1 AMAC禁止/限制投向
私募基金不得投向以下领域（出资人/实际控制人不得从事的冲突业务）：
- 民间借贷
- 配资业务
- P2P网络借贷
- 现金贷
- 其他与私募基金性质相冲突的业务

### 3.2 外商投资负面清单
标的所在行业是否涉及外商投资准入负面清单的限制类/禁止类：
- 如涉及，VIE架构的合规性评估
- 外资比例限制（如增值电信业务外资股比 ≤ **50%**）

### 3.3 国家重点监管行业
如标的属于以下行业，需额外合规审查：
- 教育培训（"双减"政策后学科类培训禁止资本化）
- 医疗健康（两票制/集采对商业模式的冲击评估）
- 互联网金融/金融科技（牌照要求和监管收紧趋势）
- 数据安全敏感行业（《数据安全法》/《个人信息保护法》合规）

## 四、第三层：退出路径可行性

依据 `sources/laws/对赌回购可执行性要点.md`（九民纪要第5条），评估三种退出路径的可行性：

### 4.1 IPO退出
- 标的行业是否为A股IPO鼓励行业（科创属性/硬科技/专精特新）
- 如为科创板：研发投入占营收比例、发明专利数量等硬性指标
- 如为北交所：新三板挂牌满12个月+专精特新属性
- **对赌清理**：如果标的已有含对赌条款的投资协议，涉及发行人作为对赌义务人的必须在申报前彻底终止。附带恢复条款的对赌不被各板块审核接受（参见九民纪要后的IPO审核实践）

### 4.2 并购退出
- 行业内潜在买方（上市公司/产业资本/PE）列表
- 标的在产业链中的战略价值（是否形成"必买"逻辑）

### 4.3 回购退出
- 对赌/回购条款设计的可执行性评估：
  - 与创始股东对赌：**完全有效**，可强制执行（九民纪要确认）
  - 目标公司担保：须经公司内部决议通过
  - 目标公司直接回购：须先完成减资程序（公司法第177条，债权人公告+清偿/担保）
- 创始股东个人偿付能力评估：如其资产以目标公司股权为主，实际执行效果有限

### 4.4 退出时间可行性
- 基金管理人是否有权单方延长基金期限（见LPA条款）
- 基金初始募集规模要求：单一标的 **≥ 2000万元**，组合投资 **≥ 1000万元**（AMAC备案要求）

## 五、第四层：LP约束检查

依据 `sources/mappings/合格投资者认定标准_v1.md`：

### 5.1 LP资格与基金募集合规
- LP是否为合格投资者
  - 自然人：金融资产 ≥ **300万元** 或近3年年均收入 ≥ **50万元**
  - 法人/组织：净资产 ≥ **1000万元**
- 单只基金投资者人数：有限合伙型 ≤ **50人**，契约型 ≤ **200人**
- 穿透核查要求（合伙企业等非法人形式汇集资金的须穿透）

### 5.2 LP特殊权利影响
- LP side letter 中的最惠国条款（MFN）对交易条款的影响
- LP是否有特殊赎回权/信息权的差异化要求
- LP是否对特定行业/地区有限制性要求

## 六、快速筛选打分卡

| 维度 | 权重 | 评分(1-5) | 加权分 |
|------|------|-----------|--------|
| 基金策略匹配度 | 25% | | |
| 行业准入合规 | 20% | | |
| 退出路径可行性 | 25% | | |
| 团队/技术判断 | 15% | | |
| LP约束适配 | 15% | | |
| **总分** | **100%** | | **X.X** |

- 总分 ≥ 4.0：建议进入深度尽调
- 总分 3.0-3.9：值得跟进但需关注特定风险
- 总分 < 3.0：建议pass或降级为观察

## 七、"一票否决"红线

以下任一情况出现直接终止筛选：
1. 标的行业属于AMAC明确禁止的冲突业务领域
2. 创始团队存在重大诚信问题或失信被执行人记录
3. 退出路径完全不可行（含对赌条款经评估无法执行）
4. 基金无匹配的投资授权（行业/阶段/金额/地域四者之一严重不匹配）
5. LP协议中有明确限制条款阻止该类型投资

## 八、输出格式

筛选完成后输出一页纸摘要：
- **项目名称** + **一句话描述**
- **筛选结论**（通过/待定/否决）
- **四层漏斗通过情况**（每层关键发现1-2句）
- **核心风险提示**（不超过3条）
- **下一步建议**（如需推进，列出须优先核查的尽调事项）

> 项目筛选是一个快速过滤机制，目标是在30分钟内完成四层评估。本技能不替代深度尽调（参见 `dd-checklist`），而是确定项目是否值得分配尽调资源。
