# 尽调会议准备

基于 `sources/mappings/尽调必查清单_v1.csv` 的检查项体系和 `sources/laws/对赌回购可执行性要点.md` 的司法审查要点，为不同类型尽调会议生成结构化的议程和访谈提纲。

## 一、会议类型与议程

尽调过程中通常需要安排三类会议，每类会议的目标和参与方不同：

| 会议类型 | 参与方 | 时长建议 | 核心目标 |
|----------|--------|----------|---------|
| 管理层访谈 | 创始人/CEO/CFO/CTO | 2-3小时 | 验证商业逻辑、团队能力、战略规划 |
| 职能部门访谈 | 财务/法务/HR/销售负责人 | 各1-2小时 | 逐项核查尽调清单中的具体事项 |
| 外部专家/客户/供应商访谈 | 行业专家、大客户、核心供应商 | 各30-60分钟 | 交叉验证市场地位和业务质量 |

## 二、管理层访谈提纲

依据 `sources/mappings/尽调必查清单_v1.csv` 中业务尽调和人力尽调子项：

### 2.1 公司与业务
1. 请用3分钟介绍公司目前的核心业务和近3年的关键里程碑
2. 当前客户数量/用户规模？前5大客户占比是否 > 30%？如是，单一客户流失的影响评估
3. 主要竞争对手是谁？与竞对最显著的3个差异化优势
4. 技术壁垒的具体体现：专利数量、研发团队规模、核心技术是否依赖第三方
5. 未来12个月的业务增长规划：驱动因素是客户数增长还是客单价提升

### 2.2 团队与组织
1. 核心管理团队的完整履历和加入时间（对照尽调清单核查5年以上行业经验）
2. 近2年核心团队流失率？关键技术人员的竞业限制/股权激励安排
3. 是否有股权代持安排？如有，代持方和实际权益人的关系
4. 创始人与联合创始人之间的股权分配和决策机制

### 2.3 财务数据
1. 近3年收入/利润的驱动因素拆解（分产品/分客户类型）
2. 收入确认政策：是否存在提前确认收入/渠道压货等情况
3. 应收账款回款周期和坏账情况
4. 公司现金流能否支撑未来12个月运营（burn rate分析）

## 三、法务/合规访谈——对赌专项

依据 `sources/laws/对赌回购可执行性要点.md`（九民纪要第5条），与创始人和法务负责人访谈须覆盖以下必问项：

### 3.1 现行对赌/回购条款盘点
1. 公司目前签署了多少份含对赌/回购条款的投资协议？
2. 每份协议的对赌主体是谁？（创始股东个人/目标公司/目标公司担保）
3. 各对赌条款的触发条件是什么？（业绩指标/IPO时限/其他）
4. 是否已触发或临近触发任何对赌/回购义务？
5. 如有已触发条款，目前处置进展如何？

### 3.2 可执行性评估
1. 创始股东的个人资产状况（是否能覆盖潜在回购义务）
2. 如涉及目标公司担保，担保决议是否经股东会/董事会合法通过
3. 如涉及目标公司回购，是否了解须先完成减资程序（公司法第177条要求）
4. 对赌条款约定的回购利率水平（判断是否处于法院支持区间：年化 **10%以下** 普遍支持，**10%-15%** 多数支持，**15%以上** 存在调整风险）

### 3.3 IPO规划对对赌的影响
1. 公司是否有IPO计划？如有，预计申报时间窗口
2. 现行对赌条款是否涉及发行人作为对赌义务人——如是，须在申报审计截止日前彻底终止
3. 是否了解附带恢复条款的对赌安排不被主板/科创板/创业板审核接受

## 四、客户/供应商访谈提纲

### 4.1 客户访谈（N≥5）
1. 与该公司的合作开始时间和合作规模
2. 选择该供应商的核心原因（价格/质量/服务/技术）
3. 对该公司产品/服务的满意度评分（NPS）
4. 是否有替代供应商？切换成本多高？
5. 未来12个月采购计划（增加/持平/减少）

### 4.2 供应商/渠道访谈
1. 与该公司的合作历史和稳定性
2. 是否对该公司有独家供应或优先供应安排
3. 账期条件和信用政策

## 五、会议议程模板

每次尽调会议按以下结构组织（90-120分钟）：

| 时间段 | 内容 | 负责人 |
|--------|------|--------|
| 0-10分钟 | 开场介绍与目标说明 | 项目VP |
| 10-50分钟 | 业务/市场/竞争核心问题 | 项目团队 |
| 50-80分钟 | 财务/法务/对赌专项 | 财务/法务顾问 |
| 80-100分钟 | 团队/组织/文化 | 项目VP |
| 100-115分钟 | 开放讨论+补充问题 | 全体 |
| 115-120分钟 | 下一步安排与资料需求清单 | 项目分析师 |

## 六、会后交付物

1. **会议纪要**（48小时内发出，经被访谈方确认）
2. **待补充信息清单**（标注优先级和责任人）
3. **关键发现汇总**：新发现的正面信息、新发现的风险点、与原预判的偏差
4. **尽调问题日志（Issues Log）更新**：标注红/黄/绿状态

## 七、常见陷阱提醒

- **避免引导性问题**：不要以"我们觉得你们的优势是XX对吗"方式提问，应让受访者独立表达
- **避免过早关注细节**：管理层访谈优先覆盖"大画面"和战略层面，职能访谈再做逐项核查
- **交叉验证**：同一个关键问题（如客户集中度、对赌触发状态）应在不同访谈中从不同角度交叉确认
- **对赌问题敏感性**：对赌条款访谈可能触发创始人防御心态，建议放在法务专项访谈中由法务顾问主导
- **记录留痕**：所有访谈须有书面纪要并由受访方确认，可作为后续交易文件的谈判依据

> 会议准备输出文件应与 `dd-checklist` 技能生成的尽调清单联动，确保每次会议的议题均有对应的清单检查项映射。
