# 组合监控

基于中国私募基金的投后管理要求和AMAC持续合规义务，建立系统化的组合公司监控框架。覆盖财务经营指标追踪、对赌/回购条款执行进度监控和监管报送合规管理三个核心维度。

## 一、监控框架

### 1.1 监控频次

| 监控维度 | 频次 | 触发方式 |
|----------|------|---------|
| 月度经营数据 | 每月 | 被投企业报送 |
| 季度财务报告 | 每季度 | 被投企业报送+内部复核 |
| 对赌里程碑追踪 | 每季度 | 对照对赌协议条款逐项核查 |
| 重大事项监控 | 实时 | 被投企业主动报告+公开信息扫描 |
| AMAC合规报送 | 季度/年度 | 依据监管时限 |

### 1.2 监控分层

根据投资金额、对赌复杂度和风险评级将被投企业分为三层：

- **A层（高关注）**：投资金额大（> 基金规模的15%）、对赌条款复杂、已出现早期预警信号 → 月度深度复盘
- **B层（常规关注）**：标准投资、运行平稳 → 季度常规监控
- **C层（低关注）**：小额投资、无复杂条款、成熟稳定 → 半年度简报

## 二、财务/经营指标追踪

### 2.1 核心追踪指标

| 类别 | 指标 | 预警阈值 |
|------|------|---------|
| 收入 | 实际vs预算偏差率 | > **15%** 触发预警 |
| 毛利 | 毛利率环比变化 | > **5个百分点** 恶化触发 |
| 现金流 | 现金跑道（月） | < **6个月** 触发预警 |
| 客户 | 大客户流失 | 前五大客户任一流失触发 |
| 团队 | 核心人员离职 | 任一C-level/核心技术人员离职触发 |
| 融资 | 下一轮融资进度 | 超过预期时间 **3个月** 未启动触发 |

### 2.2 预警升级机制

- **黄色预警**：指标偏离但可逆 → 要求管理层书面说明+整改计划
- **橙色预警**：指标持续恶化 → 派驻董事专项沟通+要求董事会讨论
- **红色预警**：出现重大风险 → 启动投资协议中的保护性条款+准备退出方案

## 三、对赌/回购里程碑追踪

依据 `sources/laws/对赌回购可执行性要点.md`（九民纪要第5条），对已签有对赌/回购条款的被投企业进行专项跟踪：

### 3.1 对赌里程碑清单

逐一列出每份对赌协议中的：
- 业绩承诺（第T年度经审计净利润 ≥ X万元）
- IPO时限（YYYY-MM-DD前完成合格IPO）
- 其他触发条件（创始股东诚信/竞业/重大违约等）

### 3.2 进度跟踪

| 对赌条款 | 承诺值 | 当前进度 | 完成概率 | 风险等级 |
|----------|--------|---------|---------|---------|
| 2024年净利润 | ≥ [X]万元 | [Y]万元（[Z]%） | [高/中/低] | [红/橙/黄/绿] |
| IPO时限 | [YYYY-MM-DD] | 当前阶段 | [高/中/低] | [红/橙/黄/绿] |

### 3.3 触发前处置

如预判对赌条款将在下一个报告期触发，须提前启动：
1. 与管理层沟通触发原因和补救方案
2. 评估对赌主体的偿付能力变化
   - 创始股东：个人资产（房产/金融资产/其他投资）是否覆盖回购金额
   - 目标公司：可分配利润是否足够（现金补偿型）或是否具备减资条件（股权回购型）
3. 聘请律师评估条款可执行性（对赌主体是否适格、内部决议是否完备、利率是否在司法支持区间）
4. 备选退出方案准备（寻找并购买方/S基金受让/启动清算程序）

### 3.4 利率合规检查

回顾已签对赌条款中的回购利率（基准：合同成立时 1 年期 LPR 的 4 倍，当前约 12%–15%；旧"24%/36% 两线三区"已于 2020-08-20 废止，仅适用于此前成立的合同）：
- 年化 **未超 4 倍 LPR**：落在民间借贷司法保护区间，通常支持
- 年化 **超过 4 倍 LPR**：超出部分大概率不被法院支持
> 争议提示：回购/对赌溢价是否套用民间借贷 4 倍 LPR 上限，司法实践尚有分歧（"名为股权实为借贷/让与担保"则适用，认定为股权交易对价则不适用），须个案咨询律师，本区间仅供预警。

## 四、AMAC合规报送

依据 `sources/laws/AMAC私募备案要求.md`（中基协发〔2023〕5号），管理人须按时履行以下报送义务：

### 4.1 报送时限与内容

| 报告类型 | 报送时限 | 内容要点 |
|----------|---------|---------|
| **季度更新** | 每季度结束后**10个工作日**内 | 基金基本信息更新、投资者变动、投资进度 |
| **年度报告** | 每年度结束后**4个月**内 | 基金经审计的财务报告、组合公司概况、重大事项说明 |
| **重大事项报告** | 发生后**10个工作日**内 | 管理人/基金重大变更、重大投资/退出、重大诉讼/处罚 |
| **审计报告** | 管理规模 ≥ **10亿元**或集团化运营的强制年度审计 | 经审计的管理人/基金财务报告 |

### 4.2 组合监控对应报送要点

季度/年度报送中涉及组合公司的信息要点：
- 各被投企业的投资金额、持股比例、最新估值
- 被投企业是否存在重大经营风险或合规问题
- 是否存在关联交易及处理情况
- 对赌条款执行进展（如触发）及解决计划

### 4.3 合规红线

- 未按时报送季度/年度报告：可能被AMAC列入异常经营名单
- 重大事项未在**10个工作日**内报告：面临自律管理措施
- 管理规模 ≥ **10亿元**未做年度审计：违反《私募投资基金登记备案办法》要求

## 五、组合仪表盘（Portfolio Dashboard）

### 5.1 组合层面汇总指标

每季度生成组合仪表盘，至少包含：
- 组合总公允价值 vs 总成本（Gross MOIC）
- 各被投企业MOIC分布（>2x / 1-2x / 0.5-1x / <0.5x）
- 对赌完成率（已达标/进行中/已触发未解决）
- 预警企业数量及等级分布（红/橙/黄/绿）
- 组合现金跑道加权平均（月）

### 5.2 异常标注

- 对已触发对赌但超期未解决的企业标红
- 对连续两个季度实际业绩低于承诺 **70%** 的企业标橙
- 对现金跑道 < **6个月** 且无明确融资计划的企业标红

## 六、投后增值记录

除监控外，须记录基金对被投企业的投后增值贡献，用于后续LP沟通和业绩归因：
- 战略对接（引入客户/供应商/合作伙伴）
- 人才引进（帮助招聘核心高管）
- 后续融资支持（领投/跟投后续轮次）
- 上市辅导（引入中介机构/协助合规整改）

## 七、输出节奏

| 输出物 | 频次 | 受众 |
|--------|------|------|
| 月度经营快报 | 每月 | 内部投后团队 |
| 季度组合监控报告 | 每季度 | IC委员/LP（摘要版） |
| 年度组合回顾报告 | 每年 | LP（完整版）+ AMAC报送 |
| 重大事项即时报告 | 实时 | 投后负责人+风控负责人 |

> 组合监控的核心价值在于"早发现、早干预"。对赌条款触发的黄金干预窗口通常在触发前6-12个月——早于触发即介入，可通过管理层调整经营策略、引入资源、调整对赌条款等方式化解；触发后再补救则手段有限且代价高昂。本技能应与 `value-creation-plan` 联动，监控与增值并重。
