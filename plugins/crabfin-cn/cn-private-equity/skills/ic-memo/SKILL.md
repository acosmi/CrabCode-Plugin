# 投资委员会备忘录

依据 `sources/templates/IC备忘录_脱敏模板.md` 的中国私募基金标准IC备忘录格式，整合前置技能输出（尽调清单、项目筛选、回报分析、价值创造计划），生成供投资决策委员会表决用的完整备忘录。备忘录九大章节与脱敏模板一致：执行摘要、公司概览、投资主题、市场与竞争、尽调核心发现、交易条款、退出分析、风险与缓释、团队建议。

## 三、投资主题（核心持股逻辑）

至少列出 **2-3个** 独立投资主题，每个须含：主题名称、论证逻辑（市场机会 → 差异化能力 → 数据支撑 → 趋势预判）、核心假设及验证状态（已证实/待验证/存疑）。投资主题是备忘录的"灵魂"——投决会委员最关注"为什么现在投"，而非堆砌数据。

## 五、尽调核心发现

整合 `dd-checklist` 技能输出，按四大类呈现：

**商业尽调**：客户满意度/NPS、留存率（年度/月度）、LTV/CAC（>3x为佳）、收入可预测性（复购/订阅占比）、供应商/渠道依赖。

**财务尽调**：近3年+1年预测的核心财务数据（营收/YoY/毛利率/净利润/经营现金流/现金余额），关键发现：收入确认质量、毛利率趋势、费用率规模效应、现金流与盈利偏差。

**法律/合规尽调**：股权代持/历史沿革、IP权属、重大诉讼、VIE合规、环保/安全/数据安全。

**人力和文化**：创始人评估、核心团队近2年流失率、激励安排（期权池/归属计划）。

### 6.2 对赌/回购条款设计

**对赌主体选择**（可执行性逐级递减）：
- 首选：与创始股东/实控人对赌（**完全有效**，可强制执行）
- 次选：创始股东对赌+目标公司连带担保（担保须经内部决议）
- 限制：目标公司直接作为义务人——现金补偿须有可分配利润，股权回购须先完成减资程序（公司法第177条）

**回购关键参数**：回购价格 = 投资本金 × (1 + 年化 **8-12%**) × N年 − 已分配股息。年化收益率建议 **10%以下** 以降低法院调整风险。触发情形：（1）未按期完成合格IPO；（2）净利润低于承诺的 **70%**；（3）创始股东重大违约/诚信问题。

**IPO审核影响**：涉及发行人作为对赌义务人的须在申报审计截止日前彻底终止；附带恢复条款的对赌各板块审核不认可；实控人/控股股东的对赌可在四不条件下保留（不作为对赌当事人、不与市值挂钩、不影响控制权稳定、不与业绩挂钩）。

### 6.3 LP合规检查

依据 `sources/mappings/合格投资者认定标准_v1.md`：确认本轮投资者均为合格投资者（自然人金融资产 ≥ **300万元** 或近3年年均收入 ≥ **50万元**；机构净资产 ≥ **1000万元**）；投资者人数在法定上限内（有限合伙 ≤ **50人**、契约 ≤ **200人**）；需穿透核查的投资者结构排查。

## 注意事项

1. **保持客观**：如实呈现正面和负面信息，不可选择性披露
2. **对赌条款须经律师审查**：可执行性判断基于当前司法口径，具体须经并购/基金律师确认
3. **团队建议不可AI替代**：备忘录骨架可参数化生成，判断性内容须人工补充
4. **数据交叉验证**：财务/业务数据须至少两个来源交叉确认

> IC备忘录论证核心为"投资主题+风险+回报"三角验证。本技能基于 `dd-checklist`、`deal-screening`、`returns-analysis`、`value-creation-plan` 四个前置技能的数据自动生成骨架，由人工补充判断性内容。
