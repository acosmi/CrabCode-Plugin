# 项目搜寻

基于中国私募基金监管环境和行业政策导向，系统化构建项目搜寻策略。中国市场的项目搜寻需叠加监管准入审查、产业政策偏好和LP约束三层过滤，确保搜寻方向与可投范围一致。

## 一、搜寻策略制定

### 1.1 基金投资策略映射

项目搜寻的第一步是将基金的核心投资参数转化为具体的搜寻画像：

| 基金参数 | 搜寻约束 |
|----------|---------|
| 投资行业 | 限定目标行业及其上下游延伸领域 |
| 投资阶段 | 限定融资轮次和对应财务/业务特征 |
| 单笔投资金额 | 限定估值区间（投前估值 [X]-[Y] 亿元） |
| 基金存续期 | 限定标的成熟度（以确保在退出期前实现退出） |
| LP偏好 | 排除LP明确反对的行业/地区/商业模式 |

### 1.2 AMAC行业准入红线

依据 `sources/laws/AMAC私募备案要求.md`（中基协发〔2023〕5号），以下行业/业务类型属于私募基金禁止投向或严格限制领域，须在搜寻阶段直接排除：

- **民间借贷**相关商业模式（P2P、现金贷等）
- **配资业务**（场外配资、融资融券配资平台）
- **虚拟货币**（ICO/发币/交易所等，中国境内禁止）
- **与私募基金冲突的业务**（以集资为目的的平台型业务）

### 1.3 国家产业政策导向加分

2023-2025年中国政策重点支持方向（持续更新）：
- **硬科技**：半导体/芯片、高端装备、新材料、工业软件
- **新能源**：光伏/储能/氢能/动力电池、新型电力系统
- **数字经济**：人工智能/大模型、数据要素、智能制造
- **生命科学**：创新药/生物技术、医疗器械、精准医疗
- **专精特新**：国家级/省级专精特新"小巨人"企业

## 二、LP约束对搜寻的影响

依据 `sources/mappings/合格投资者认定标准_v1.md`：

### 2.1 LP来源决定搜寻偏好

- **政府引导基金LP**：通常要求返投当地（返投比例一般为1:1至1:1.5），搜寻范围须以引导基金所在地为核心
- **产业资本LP**：通常偏好与自身产业链相关的标的，搜寻方向须与LP产业战略协同
- **金融机构LP（保险/银行理财）**：偏好低风险、稳定现金流的成熟期标的，排除早期VC
- **家族办公室/高净值个人LP**：搜寻偏好灵活，但须确保LP自身为合格投资者（金融资产 ≥ **300万元** 或近3年年均收入 ≥ **50万元**）

### 2.2 基金规模与搜寻覆盖面

- 基金初始募集规模底线：私募股权基金 ≥ **1000万元**（组合投资）或 ≥ **2000万元**（单一标的）（AMAC备案要求）
- 基金投资者人数上限：有限合伙型 ≤ **50人**，契约型 ≤ **200人**
- 较小的基金规模意味着投资标的数量有限，搜寻须更精准聚焦而非广撒网

## 三、搜寻渠道体系

### 3.1 核心渠道（按优先级排序）

| 渠道 | 特点 | 适用阶段 |
|------|------|---------|
| FA/精品投行 | 项目质量参差不齐，需建立优质FA白名单 | A轮及以上 |
| 行业专家网络 | 通过行业专家获取非公开项目线索 | 各阶段 |
| 产业园区/孵化器 | 区域性早期项目源，适合引导基金返投 | 天使/A轮 |
| 已投项目创始人推荐 | 转化率最高的渠道（信任传递） | 各阶段 |
| 券商投行部 | 侧重Pre-IPO/并购项目，估值通常偏高 | B轮/Pre-IPO |
| 政府招商引资平台 | 引导基金返投项目源，政策扶持明确 | 各阶段 |
| 行业协会/峰会 | 建立行业人脉，批量获取项目线索 | 各阶段 |

### 3.2 渠道管理
- 建立渠道分级体系（A级：高频高质量/ B级：偶尔有优质项目/ C级：观察）
- 每季度评估渠道转化率（推荐项目数→进入筛选数→立项数）
- 新渠道开拓KPI：每季度至少新增2-3个有效渠道

## 四、长名单→短名单筛选流程

```
长名单(50-100个) → 初步评估(20-30个) → 短名单(5-10个) → 立项尽调(2-3个)
```

### 4.1 长名单建设（月度更新）
- 来源：各渠道推荐的潜在标的
- 数据：项目名称、一句话描述、所在行业、融资轮次、融资金额、推荐渠道

### 4.2 初步评估（2周内完成）
调用 `deal-screening` 技能的四层漏斗进行快速筛选：
- 第一层：基金策略匹配（行业/阶段/金额/地域）
- 第二层：行业准入合规（排除AMAC禁止领域、外商投资负面清单）
- 第三层：退出路径可行性（IPO/并购/回购可行性初判）
- 第四层：LP约束检查（LP基金募集合规+特殊权利影响）

### 4.3 短名单建设
- 通过初步评估的项目进入短名单（5-10个）
- 对短名单内项目进行桌面研究（公开信息收集、竞对分析、团队背景调查）
- 安排初次接触（创始人电话会/面谈）

## 五、区域搜寻策略

中国市场的显著区域特征：

| 区域 | 优势行业 | 搜寻策略 |
|------|---------|---------|
| 北京 | 人工智能、企业服务、军工/航空航天 | 重点关注海淀/中关村、亦庄 |
| 上海 | 消费品牌、生物医药、金融科技 | 重点关注张江/临港、外商投资项目 |
| 深圳/大湾区 | 硬件/消费电子、智能制造、跨境出海 | 重点关注南山/宝安/东莞 |
| 杭州 | 电商/SaaS、数字经济 | 重点关注余杭/未来科技城 |
| 成都/重庆 | 游戏/文创、半导体 | 成渝双城经济圈政策导向 |
| 苏州/无锡 | 高端制造、半导体 | 制造业升级+海外人才回流 |

## 六、搜寻产出物

每月输出：
1. **长名单更新**（新增项目+移除项目+状态变化）
2. **短名单进展**（各项目的接触状态和下一步计划）
3. **渠道效能报告**（各渠道推荐数量、转化率、质量评分）
4. **市场动态简报**（所属赛道的新融资事件、政策变动、竞对动向）

> 项目搜寻是持续的长周期工作，核心在于建立系统化的渠道网络和信息处理机制。搜寻阶段的产出质量直接影响基金的投资节奏（deployment pace）和项目质量。本技能应与 `deal-screening` 技能联动使用。
