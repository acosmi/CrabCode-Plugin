# 回报分析

基于中国私募股权投资特征，构建系统化的投资回报分析框架。核心覆盖三大场景：正常退出回报测算、对赌/回购情景模拟和LP层面分级回报计算。

## 一、回报分析核心指标体系

| 指标 | 定义 | 中国私募常用基准 |
|------|------|----------------|
| **MOIC**（投资回报倍数）| 退出总价值 / 投资总成本 | 目标 ≥ 2.5-3.0x（早期VC）/ ≥ 2.0x（成长期） |
| **IRR**（内部收益率）| 年化复合收益率 | 目标 ≥ 20-25%（早期VC）/ ≥ 15-20%（成长期） |
| **DPI**（已分配比例）| 累计分配 / 实缴资本 | LP最关注的指标，目标基金层面 ≥ 1.0x |
| **TVPI**（总价值比例）| （累计分配 + 未退出公允价值）/ 实缴资本 | 反映基金整体表现 |
| **Hurdle Rate**（门槛收益率）| LP优先回报率 | 中国市场常用 **8%**，部分基金采用 **10%** |
| **Carry分配** | GP业绩报酬 | 超过hurdle后的超额收益分成，通常 **20%** |

## 二、正常退出回报测算

### 2.1 退出路径建模

依据中国资本市场实际情况，建立四种退出路径的回报模型：

| 退出路径 | 估值方法 | 流动性折扣 | 退出周期 |
|----------|---------|-----------|---------|
| A股IPO | P/E倍数法（参考同行业上市公司均值） | 0-10%（A股流动性较好） | 3-7年 |
| 港股/美股IPO | P/E或P/S倍数法（参考境外同行） | 10-20%（流动性相对A股偏低） | 2-5年 |
| 并购退出 | 交易对价/可比交易倍数法 | 0-5%（如有竞购）/ 5-15%（单一买方） | 1-3年 |
| 老股转让/S基金 | 最近一轮估值折扣 | 15-30%（非公开市场流动性折价） | 0.5-1年 |

- 基准情景IRR是否 > 基金门槛收益率（通常 **8-10%**）
- 概率加权MOIC是否 > 基金最低要求（通常 **2.0x**）

## 三、对赌/回购情景模拟

依据 `sources/laws/对赌回购可执行性要点.md`（九民纪要第5条），对含有对赌/回购条款的投资进行专项回报模拟：

### 3.1 回购回报计算

回购价格公式（中国私募标准公式）：
```
回购价格 = 投资本金 × (1 + 年化收益率) × N年 − 已分配股息
```

参数设定：
- 年化收益率：依据已签对赌协议约定（通常在 **8-15%** 区间）
- N年：自投资完成日至回购触发日/执行日的实际年限
- 已分配股息：投资期间的累计股息收入

### 3.2 回购可执行性对回报的影响

九民纪要后的司法实践对回购回报的实际回收率产生重要影响，须分层测算：

| 对赌主体 | 回收率假设 | 依据 |
|----------|-----------|------|
| 创始股东对赌（个人资产充足） | 80-100% | 司法完全支持强制执行 |
| 创始股东对赌（资产以股权为主） | 30-60% | 偿付能力不足，须评估创始股东个人资产 |
| 目标公司担保（连带） | 50-80% | 须经内部决议有效，且目标公司有实际偿付能力 |
| 目标公司直接回购 | 20-50% | 须先完成减资程序，流程漫长且不可控 |

### 3.3 回购利率司法风险对回报的影响

> 基准：合同成立时 1 年期 LPR 的 4 倍（当前约 12%–15%）。旧 24%/36% 上限自 2020-08-20 废止；且回购/对赌溢价是否套用该上限司法有争议，下表为保守预警口径。

| 约定年化利率（相对 4×LPR） | 司法支持程度 | 建议回收率调整 |
|-------------|-------------|---------------|
| **未超 4×LPR** | 通常支持 | 不调整 |
| **超过 4×LPR** | 超出部分大概率不被支持 | 按 4×LPR 封顶重算回收现金流 |

## 四、LP层面分级回报计算

依据 `sources/mappings/合格投资者认定标准_v1.md` 和基金LPA中的waterfall分配条款，按LP份额类别计算差异化回报：

### 4.1 Waterfall分配模拟

中国私募基金典型waterfall结构：

```
第1层：返还LP实缴本金（Return of Capital）
第2层：LP优先回报（Hurdle Rate，通常 8-10%）
第3层：GP追补（Catch-up，通常至GP获得总carry的 20%）
第4层：超额收益分配（LP:GP = 80:20）
```

### 4.2 LP差异化分析

考虑LP side letter中的特殊条款对回报的影响：
- 最惠国条款（MFN）：部分LP可能享有更低的费率或更高的分配优先级
- 特殊赎回权：部分LP可能有权提前退出，影响其实际IRR
- Co-investment安排：LP跟投的回报与基金直投的回报须分开计算

### 4.3 合格投资者回报报告

LP报告须满足合格投资者适当性管理要求：
- 向自然人LP（金融资产 ≥ **300万元** 或近3年年均收入 ≥ **50万元**）提供与其风险等级匹配的回报报告
- 向机构LP（净资产 ≥ **1000万元**）提供更详细的财务分析和风险披露

## 六、输出格式

回报分析最终输出：
1. **回报概览表**：基准情景MOIC/IRR + 概率加权MOIC/IRR
2. **三情景详细测算表**
3. **对赌回购回报路径分析**（如适用）
4. **LP层面waterfall模拟**（如为基金层面分析）
5. **敏感性矩阵**
6. **与基金基准对比结论**（是否达到门槛收益率和最低MOIC要求）

> 回报分析中的核心假设（退出估值倍数、退出年限、对赌回收率）直接影响结论。建模时须明确标注假设来源和不确定性范围，不可将模型产出等同于确定性预测。本技能基于fin-core模型框架，中国市场特定参数（hurdle rate、对赌利率司法区间、A股流动性折扣等）依据上述source材料设定。
