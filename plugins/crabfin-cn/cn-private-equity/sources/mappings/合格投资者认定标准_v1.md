# 合格投资者认定标准

## 法规依据
- 《关于规范金融机构资产管理业务的指导意见》(银发〔2018〕106号,简称"资管新规")
- 《私募投资基金监督管理条例》(国务院令第762号,自2023年9月1日起施行)
- 《私募投资基金监督管理暂行办法》(证监会令第105号)
- 《证券期货投资者适当性管理办法》(证监会令第130号,2020年修订)

## 版本日期
2025-06-20

## 一、合格投资者认定标准(核心标准)

### 1.1 资管新规口径(适用于银行理财/信托/证券期货/保险资管等)

| 投资者类型 | 认定标准 | 认定条件(满足其一即可) |
|-----------|----------|----------------------|
| **自然人合格投资者** | 具备相应风险识别能力与风险承受能力 | ① 家庭金融净资产 ≥ **300万元**<br>② 家庭金融资产 ≥ **500万元**<br>③ 近3年本人年均收入 ≥ **40万元** |
| **法人/组织合格投资者** | 具备相应风险识别与承受能力 | ① 最近1年末净资产 ≥ **1000万元** |

### 1.2 私募基金口径(私募证券/私募股权)

| 项目 | 标准 |
|------|------|
| **自然人** | 金融资产 ≥ **300万元** 或 近3年年均收入 ≥ **50万元** |
| **法人/组织** | 净资产 ≥ **1000万元** |
| **单只私募基金投资门槛** | 最低认购金额 **100万元** |

### 1.3 专业投资者 vs 普通投资者

| 类型 | 认定标准(自然人) |
|------|------------------|
| **专业投资者(法人)** | ① 金融资产 ≥ **2000万元**(最近1年末经审计净资产 ≥ **2000万元**)<br>② 金融资产 ≥ **1000万元**(最近1年末经审计净资产 ≥ **1000万元**) + 近2年以上证券/基金/期货投资经历 |
| **专业投资者(自然人)** | ① 金融资产 ≥ **500万元** 或 近3年年均收入 ≥ **50万元**<br>② 具有 **2年以上** 证券/基金/期货/黄金/外汇投资经历或相关工作经历 |
| **普通投资者** | 不符合上述专业投资者标准的投资者 |

## 二、投资者适当性匹配(五级风险等级)

| 投资者风险承受等级 | 可匹配产品风险等级 | 风险等级描述 |
|:---:|:---:|------|
| C1(保守型/最低风险承受) | R1 | 保本型 |
| C2(谨慎型/较低) | R1, R2 | 低风险 |
| C3(稳健型/中等) | R1, R2, R3 | 中等风险 |
| C4(进取型/较高) | R1, R2, R3, R4 | 中高风险 |
| C5(激进型/高) | R1, R2, R3, R4, R5 | 高风险 |

**不匹配处理：** C1~C4 投资者购买高于自身风险等级的产品时,经营机构须做特别书面风险警示,投资者坚持购买需签署书面确认。

## 三、合格投资者人数限制

| 私募基金组织形式 | 人数上限 | 依据 |
|:---|:---:|------|
| 有限责任公司型 | 50人 | 《公司法》 |
| 有限合伙型 | 50人(合伙人) | 《合伙企业法》 |
| 契约型/信托型 | **200人** | 《证券投资基金法》第八十八条 |

> **穿透核查：** 以合伙企业、契约等非法人形式汇集多数投资者资金直接/间接投资私募基金的,须穿透核查最终投资者是否为合格投资者,并合并计算人数。但社保基金、企业年金、慈善基金等可豁免穿透。

## 四、合格投资者认定流程

| 步骤 | 要求 |
|------|------|
| 1. 风险测评 | 填写风险测评问卷,确定风险等级(C1-C5) |
| 2. 资产/收入证明 | 提供近3年收入证明或金融资产证明 |
| 3. 适当性匹配 | 确认投资者风险等级与产品风险等级匹配 |
| 4. 风险揭示 | 签署风险揭示书/风险告知书 |
| 5. 冷静期 | 私募证券基金须设 ≥ **24小时** 投资冷静期 |
| 6. 回访确认 | 冷静期届满后,募集机构进行回访确认 |
| 7. 签署合同 | 完成上述步骤后正式签署基金合同 |

---

> **红线提醒：** 
> - 以上标准适用于私募基金领域。公募基金、银行理财、信托等另有适用标准(见资管新规口径)。
> - 2023年9月起施行的《私募投资基金监督管理条例》(国务院令第762号)将合格投资者制度上升为行政法规层面,违反者将面临行政处罚。
> - 具体合规以律师和合规官意见为准。
