## 一、私募基金管理人登记必备事项

### 1.1 基本准入条件

| 事项 | 要求 |
|------|------|
| **实缴资本** | 实缴货币资本 ≥1000万元人民币(或等值可自由兑换货币;**限货币出资**,不得以实物/知识产权/土地使用权等非货币财产出资,《登记指引第1号》第五条;创投基金管理人另有规定) |
| **经营场所** | 独立、稳定,不可与股东混用 |
| **高级管理人员** | 至少2名(法定代表人+合规风控负责人),须持基金从业资格 |
| **投资负责人** | 5年以上从业经验+可验证的历史业绩 |
| **合规风控负责人** | 3年以上合规/风控/法务从业经验 |
| **全体员工** | ≥5名专职员工 |
| **内部控制** | 完善的内部控制制度和流程文件 |

### 1.2 出资人/实际控制人要求

| 条件 | 要求 |
|------|------|
| **无重大违法违规** | 近3年无重大违法违规记录 |
| **出资稳定** | 出资架构清晰,不存在代持 |
| **无关联冲突业务** | 不从事与私募冲突的业务(如民间借贷、配资、P2P等) |

### 1.3 禁止性情形(管理人及高管)

- 因违法违规被基金业协会注销登记未满3年
- 被列入失信被执行人名单
- 近3年存在重大违法违规行为
- 存在到期未清偿债务

## 二、私募基金备案必备事项

### 2.1 基金备案基本要求

| 事项 | 要求 |
|------|------|
| **初始募集规模**(《登记备案办法》第三十三条) | 私募证券基金≥1000万元;私募股权基金≥1000万元(组合投资),投向**单一标的**的≥2000万元;**创投基金备案时首期实缴≥500万元,且须在基金合同约定备案后6个月内补足至1000万元** |
| **合格投资者** | 全部投资者须为合格投资者(见合格投资者认定标准) |
| **投资者人数** | 累计不得超过200人(有限责任公司/有限合伙各 ≤ 50人) |
| **托管安排** | 原则上需托管(契约型强制托管) |
| **命名规范** | 须含"私募"字样 |

### 2.2 备案材料清单

| 类别 | 必备材料 |
|------|---------|
| **基金文本** | 基金合同/合伙协议/公司章程(含必备条款) |
| **投资者信息** | 合格投资者承诺函、风险揭示书、资产证明文件 |
| **募集文件** | 募集说明书、风险评级与投资者适当性匹配文件 |
| **托管文件** | 托管协议(如有托管)/无托管安排说明 |
| **管理人文件** | 管理人登记证明、合规承诺函 |
| **其他** | 投资标的说明、关联交易制度(如有) |

### 2.3 备案时限

| 阶段 | 时限要求 |
|------|---------|
| **首期募集完成** | 募集完毕后**20个工作日**内提交备案 |
| **备案审核** | 协会收到完整材料后**20个工作日**内办理 |
| **备案完成前** | 不得进行投资运作(临时投资除外) |

## 三、持续合规要求

| 事项 | 时限 |
|------|------|
| 季度更新 | 每季度结束后10个工作日 |
| 年度报告 | 每年度结束后4个月内 |
| 重大事项报告 | 发生后10个工作日内 |
| 审计报告 | 管理规模≥10亿元或集团化运营的,年度审计 |

---

## 四、备案"必备条款"要点(基金合同)

基金合同必须包括：
1. 基金名称、类型、运作方式
2. 管理人与托管人(如有)权利义务
3. 私募基金的存续期限
4. 投资目标、范围、策略和限制
5. 投资者出资方式、数额和认缴期限
6. 收益分配和亏损分担方式
7. 管理费、托管费及业绩报酬计提方式
8. 信息披露的内容、方式和频率
9. 基金份额的认购、赎回或转让程序
10. 基金合同终止、解除及清算安排
11. 争议解决方式
12. 关联交易认定及审批机制
13. 投资者适当性匹配、冷静期(不少于24小时)及回访确认(依据《私募投资基金募集行为管理办法》第二十九、三十、三十一条,非证监会令130号)
14. 风险揭示(私募基金投资风险+基金运营风险+基金合同法律风险)
