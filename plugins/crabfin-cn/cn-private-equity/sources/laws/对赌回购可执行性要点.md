## 一、对赌(估值调整机制 VAM)的司法演变路径

### 1.1 里程碑一：海富案 (2012)

| 裁判规则 | 内容 |
|---|---|
| 判决核心 | **与股东对赌有效**,**与目标公司对赌无效**(损害公司利益 + 债权人利益) |
| 影响 | 此后私募/创投领域 VAM 条款几乎全部改为"与创始股东对赌 + 目标公司连带担保"模式 |

### 1.2 里程碑二：九民纪要 (2019) —— 核心变革

九民纪要第 5 条明确：

> **投资人与目标公司对赌,协议原则上有效。但实际履行须满足公司法"股东不得抽逃出资"、利润分配和减资程序要求。**

即:对赌协议**合同效力**与**实际履行**是两个层次的问题。合同可以有效,但如果目标公司不完成利润分配/减资程序,法院不支持强制履行。

### 1.3 当前实务操作(九民纪要后)

| 对赌模式 | 效力 | 可执行性条件 |
|---|---|---|
| **与创始股东/实控人对赌** | ✅ 完全有效 | 一般可强制执行(现金补偿/股权回购) |
| **目标公司提供连带担保** | ✅ 原则有效 | 须经目标公司内部决议(股东会/董事会)通过担保决议 |
| **目标公司直接作为对赌义务人(现金补偿)** | ✅ 合同有效 | ⚠️ 实际执行须目标公司有**可分配利润** → 必须从可分配利润中支付,不得损害资本维持原则 |
| **目标公司直接作为对赌义务人(股权回购)** | ✅ 合同有效 | ⚠️ 实际执行须先完成**减资程序**(新《公司法》2024-07-01施行**第二百二十四条**,决议后10日内通知债权人、30日内公告,债权人可要求清偿/担保;对应旧公司法第177条) |
| **与目标公司对赌 + IPO 前解除/恢复** | ❌ 审核实践不支持 | 绝大多数 IPO 项目要求在申报前**彻底终止**对赌条款,不接受"恢复条款"(附带条件恢复视为未终止) |

## 二、常见对赌条款类型

### 2.1 业绩补偿型
```
若目标公司 [第T年度] 经审计的净利润低于 [X万元],
创始股东应在审计报告出具后 [30日] 内,
以现金/股权方式向投资人补偿:
补偿金额 = (承诺净利润 - 实现净利润) × 补偿倍数
```

### 2.2 股权回购型(最常用)
```
若出现以下任一情形,投资人有权要求
创始股东和/或目标公司按以下价格回购全部或部分股权:
回购价格 = 投资本金 × (1 + 年化收益率 [8-15%]) × N(年数) − 已分配股息

触发情形:
(1) 目标公司在 [YYYY-MM-DD] 前未完成合格 IPO
(2) 目标公司 [第T年度] 实际净利润低于承诺净利润的 [70%]
(3) 创始股东出现重大违约/诚信问题
```

### 2.3 反稀释型
```
若目标公司未来以低于本轮投后估值的价格进行新增融资,
本轮投资人有权获得:
(1) 完全棘轮(Full Ratchet):股权调整至与新低价等效
(2) 加权平均(Weighted Average):按加权平均调整
[中国市场多采用加权平均,完全棘轮较少见]
```

## 三、实务要点与风险地图

### 3.1 对赌主体选择
| 主体 | 优势 | 风险 |
|---|---|---|
| **创始股东** | 可执行性强;不受减资/分配程序约束 | 创始股东个人偿付能力有限;若其资产以目标公司股权为主则无实际执行效果 |
| **目标公司担保** | 增强偿付保障 | 须经公司内部决议;存在被认定无效风险 |
| **目标公司** | 资金保障最强 | 实际执行须完成减资/利润分配;流程长且不可控 |

### 3.2 IPO 审核中的对赌处理
IPO 审核实践(上交所/深交所/北交所)：
- 涉及发行人作为对赌义务人的,**必须在申报审计截止日前彻底终止**
- 涉及实控人/控股股东的对赌条款,原则上应在申报前清理；保留的须同时满足：① 发行人不作为对赌当事人,② 对赌不与市值挂钩,③ 对赌不影响发行人控制权稳定,④ 对赌不与发行人业绩挂钩
- 附带恢复条款(如"若 IPO 被否则恢复")**主板/创业板/科创板审核不认可**⼀需彻底终止

### 3.4 回购利率(可执行性参考)

> ⚠️ 现行口径:民间借贷利率司法保护上限自 **2020-08-20** 起为**合同成立时 1 年期 LPR 的 4 倍**(当前约 12%–15%);旧"24%/36% 两线三区"已废止,仅适用于此前成立的合同。**且回购/对赌溢价是否套用民间借贷上限本身司法有争议**(认定为让与担保/名为股权实为借贷则适用,认定为股权交易对价则不适用),须个案核查。

| 利率水平(相对合同成立时 4×LPR) | 法院态度(保守预警口径,须个案核实) |
|---|---|
| **未超 4×LPR** | 落在民间借贷司法保护区间,通常支持 |
| **超过 4×LPR** | 超出部分大概率不被支持 |

## 四、与境外私募 VAM 条款的关键差异

| 维度 | 中国实践 | 境外(以美国为参照) |
|---|---|---|
| 与公司对赌 | 有条件有效但执行受限 | 普遍可执行(受州法律管辖,如特拉华) |
| IPO 清理 | 几乎必须终止 | 通常不做清理 |
| 回购利率 | 司法保护上限参照 4×LPR(约12%–15%,旧24%已废止),且对对赌的适用性有争议 | 市场化利率 |
| 连带责任| 须审查公司担保决议 | 商业合理性审查为主 |

**对 CrabFin-CN 技能的影响**：`deal-screening` / `returns-analysis` / `value-creation-plan` 中的条款模板不可将境外实践直接照搬,应基于中国司法口径构建条款审查逻辑和风险评级体系。
