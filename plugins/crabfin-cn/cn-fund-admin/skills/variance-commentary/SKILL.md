# 基金差异说明

对基金期间内重大差异撰写管理层分析说明。

> **估值依据**：`sources/mappings/估值层级判定_v1.md` / `sources/laws/AMAC基金估值指引.md`

## 操作步骤

### Step 1 — 触发阈值检查

| 差异类型 | 触发阈值 |
|---------|---------|
| NAV变动 | 期间 > ±10% |
| 投资收益偏离 | 实际 vs 基准 > ±5% |
| 单一资产估值变动 | > ±20% |
| 费用超预算 | 实际 > 预算 × 110% |
| 申赎异动 | 单日净申赎 > NAV的10% |

若全在阈值内 → 输出"本期间无重大差异触发说明"。

### Step 2 — 总体表现

NAV变动（期初→期末/±%）/ 期间收益率 / 比较基准收益率 / 超额收益(Alpha)

### Step 3 — 收益/亏损归因

| 归因维度 | 贡献(bps) | 占比 |
|---------|-----------|------|
| 行业配置 | | |
| 个股选择 | | |
| 时机选择 | | |
| 费用 | | |
| 汇率(如有) | | |

### Step 4 — Top 3/Bottom 3 贡献者

分别列示最大正向和负向贡献的资产名称/贡献bps/原因。

### Step 5 — L3资产专项说明

| 资产 | 期初 | 期末 | 变动% | 原因 |
|------|------|------|-------|------|

须额外说明：估值方法是否变更 / 关键假设变化（折现率/增长率/可比乘数）/ 敏感性分析。

### Step 6 — 异常事项 + 下期展望

**数据不可得fallback**：基准收益率数据不可得 → 使用沪深300作为代理基准，标注"代理基准"。

### 合规声明

```
⚠️ 差异说明为管理层分析，不替代正式财务报告。
L3资产估值涉及重大判断，假设变更须经估值委员会审议。
```

## 协作接口

| fin-core子技能 | 何时调用 | 期望取回的数据 |
|---------------|---------|--------------|
| `roll-forward` | NAV变动数据 | 期间滚存 |

| `accrual-schedule` | 费用明细 | 费用计提 |
| `break-trace` | 重大差异需追溯 | 差异根因 |
