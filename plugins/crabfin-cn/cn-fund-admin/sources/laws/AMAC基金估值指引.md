# 中基协(AMAC)基金估值指引

## 文件信息
- **核心文件**：《中国基金估值标准体系》/《私募投资基金估值参考指引》
- **发布机构**：中国证券投资基金业协会(AMAC)
- **核心文件**：《私募投资基金非上市股权投资估值参考指引》(2018年发布)
- **配套文件**：《固定收益品种的估值处理标准》(2014年发布,后更新)
- **参考依据**：企业会计准则第39号—公允价值计量(CAS 39)
- **出处**：https://www.amac.org.cn/

---

## 一、估值基本原则

| 原则 | 说明 |
|------|------|
| **公允价值优先** | 坚持公允价值计量,鼓励使用**市场法**估值 |
| **实质重于形式** | 根据标的实质选取适当估值方法,而非仅看法律形式 |
| **一致性** | 估值方法和参数应在前后期间保持一致,如有变更须披露 |
| **审慎性** | 在不确定性较高的情形下取更保守的估值 |

## 二、三层级公允价值估值体系(基于CAS 39)

### 第一层级(Level 1)：未调整的活跃市场报价

| 适用条件 | 估值方法 |
|---------|---------|
| 存在**活跃市场**(如交易所每日交易) | 直接取活跃市场**未调整报价** |
| 相同资产的报价可直接观察 | 以该报价确认公允价值 |

**典型资产**：
- 沪深/北交所股票的收盘价
- 活跃交易的债券净价
- 公募基金净值(NAV)
- 期货/期权交易所结算价

### 第二层级(Level 2)：直接或间接可观察输入值

| 适用条件 | 估值方法 |
|---------|---------|
| 无活跃市场但有**可观察输入值** | 可比上市公司法(乘数法) |
| 类似资产/负债有活跃市场 | 近期交易价格法(倒推法) |
| 可观察市场利率/收益率曲线 | 现金流折现法(折现率可用观察利率) |

**典型资产**：
- 非公开交易债券(使用市场收益率+估值模型)
- 限售股(基于流通股价—缺乏流动性折扣)
- 非活跃交易股票(做市商报价参考)

### 第三层级(Level 3)：不可观察输入值

| 适用条件 | 估值方法 |
|---------|---------|
| **无可观察输入值** | 现金流折现法(DCF) |
| 非上市股权投资为主 | 净资产法/可比交易法/期权定价模型 |
| 管理者须使用最佳估计 | 须详细披露估值假设和敏感性 |

**典型资产**：
- 非上市股权(创投/PE投资)
- 结构化衍生品(无活跃市场)
- 各类复杂收益权

## 三、非上市股权投资估值方法(AMAC指引重点)

### 3.1 市场法(首选,当有可参考交易时)

| 方法 | 适用条件 | 操作要点 |
|------|---------|---------|
| **近期交易价格法** | 被投企业近期有融资/股权转让 | 以最近一轮融资价格为基础,考虑企业价值变化调整 |
| **可比上市公司法** | 有类似行业/规模的上市公司 | 选取可比公司,用PE/PB/PS等乘数乘以被投企业财务指标 |
| **可比交易法** | 近期同行业并购/交易案例 | 参考可比交易 EV/EBITDA 等乘数 |

### 3.2 收益法(DCF)

适用于有明确财务预测的非上市企业：
- 预测未来自由现金流(通常3-5年+终值)
- 选取合理折现率(WACC/权益成本)
- 须披露关键假设(增长率、折现率)

### 3.3 净资产法

适用于重资产型/控股型企业：
- 以被投企业经审计的净资产为基础
- 调整表外资产/负债

## 四、估值调整(流动性折扣/控制权溢价)

| 调整项 | 情形 | 参考区间 |
|--------|------|---------|
| **缺乏流动性折扣(DLOM)** | 非上市/限售股 | 通常在 **15%-35%**(据锁定期/企业阶段而定) |
| **控制权溢价** | 控股权/战略性持股 | 通常在 **10%-30%** |
| **少数权益折扣(DLOM+DOM)** | 少数股东权益 | 累计折扣可达25%-50% |

---

## 五、估值频率要求

| 产品类型 | 估值频率 | 备注 |
|---------|:---:|------|
| 公募基金 | 每个交易日 | 开放式每日NAV |
| 私募证券基金 | ≥ 每周 | 推荐每日 |
| 私募股权/创投基金 | ≥ 每季度 | 建议月度复核 |
| 专户/信托 | ≥ 每月 | 合同约定 |

---

## 六、估值主体责任

| 主体 | 职责 |
|------|------|
| **基金管理人** | 估值第一责任人,制定估值政策和程序 |
| **基金托管人** | 复核估值结果,发现偏差须要求管理人改正 |
| **估值委员会** | 审议估值政策,解决估值争议(大型管理人必备) |
| **外部估值机构** | 独立第三方估值(如中债估值、中证估值) |

> **红线提醒：**
> - 以上要点依据 AMAC 估值指引 + CAS 39(公允价值计量准则)整理。

> - 估值具有高度专业性和判断空间。具体方法选择以基金合同约定和会计师/基金行政管理人专业判断为准。
