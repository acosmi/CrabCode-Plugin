# 选题研究

先读取 `../../PRACTICE.md`，再按需读取 `../../../editorial/references/research-standard.md`。

1. 明确主题、读者、时效窗口和目标平台。
2. 用确定性热点工具检索与中文聚类；没有官方接口的来源由研究流程访问，不伪造抓取结果。
3. 优先原始文件、监管或机构官网、公司公告和原始研究；记录 URL、发布日期、访问日期。
4. 为每个候选选题输出：新信息、公共价值、作者可形成的独立判断、强反方、证据缺口和旧闻风险。
5. 建立主张台账，区分事实、来源解释和作者推论。

输出研究 brief，不直接输出成稿。下一步交给对应创作技能并保存 `researched` manifest。
