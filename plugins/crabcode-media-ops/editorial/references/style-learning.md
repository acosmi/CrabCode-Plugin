# 风格学习与版本规则

- 按 brandId 隔离；读取和写入都要求同一品牌作用域。
- 样本先登记 referenceId、角色、权利状态和允许用途；profile 只保存 referenceId、必要来源元数据、日期、置信度和抽象特征，不保存原文或可还原的大段表达。
- 单篇样本只形成低置信观察；稳定特征需要多个自有或获授权样本支持。
- 提案列出 form value、corpus value 和冲突 ID；所有冲突解决后才能确认。
- profile 采用不可变版本，current 只是当前指针；回滚会生成新版本而不是删除历史。
- 插件升级不得写入或覆盖安装目录中的用户风格数据。
