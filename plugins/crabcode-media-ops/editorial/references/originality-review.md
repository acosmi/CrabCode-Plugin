# 原创与发布风险标准

## 审核维度

1. 独立论点：是否能用一句可争辩判断概括；
2. 因果链：证据是否真正支持中间推理；
3. 反方与边界：是否诚实处理最强反例；
4. 来源覆盖：关键事实是否可追溯；
5. 表达近似：是否连续复现对照文本的独特短语或结构；
6. 第一人称真实性：亲测、访谈、内幕、反馈是否有材料；
7. 权利：图片、引语、样本和改编权限；
8. 声誉、隐私和时效：负面断言、个人信息与旧闻复炒。

门禁扫描在人工编辑后运行，并绑定 revision、ArticleDoc 语义主体以及每份参考原文哈希；改动一字都要重扫。证据至少包括 4/8/12 字符 n-gram、最长连续重合、双向覆盖、段落相似对齐和标题/段落结构风险。

高字面重合为 blocked，不可人工覆盖；中风险为 changes_required，必须改稿；存在第三方/授权样本或结构风险时，即使字面很低也必须由与作者、扫描者不同的人逐项判断观点来源、案例组合、独特比喻、论证路径和结论推进。人工 pass 必须明确确认结构与论证均独立。

扫描只比较已登记文本，不能代表全网查重、微信原创声明或法律结论。不得输出虚构的“原创率”，也不得把 low_risk 描述为原创证明。
