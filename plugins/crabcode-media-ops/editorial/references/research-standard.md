# 研究与来源标准

## 来源层级

1. 法规、监管、司法、机构官网、公司公告、原始数据或论文；
2. 具名专业机构和有编辑责任的媒体；
3. 当事人公开表述；
4. 二次转载、社交内容和匿名爆料，只能作为线索。

记录标题、URL、发布日期、访问日期、支持的具体主张和局限。关键事实尽量回到一级来源；网页找不到证据时写“未核实”，不能用常识补齐。

## 主张台账

每项主张标为：`verified / doubtful / unsourced`。`verified` 必须有 URL；后两者在发布前修订或由具名人工对具体 claimId 给出理由放行。

明确区分：

- 事实：来源直接支持；
- 来源解释：来源作者的分析；
- 作者推论：本文基于事实的推导，必须写出因果和边界。
