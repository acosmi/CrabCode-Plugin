# Editorial Practice

先读取 `../media-core/PRACTICE.md`。编辑技能不得改变其中的事实、真实性、权利、法律路由或人工审批门禁。

- 写作前建立主张台账；来源不能支持的内容标为推论或删除。编辑复核时还要覆盖工具从标题、摘要、正文、引文元数据、图片 alt/图注和披露提取的全部可见句，并按 `verified_fact / author_inference / opinion / non_claim` 分类，不能只复核写作者主动列出的 claims。
- 编辑结论不等于法律意见、平台审核结果、原创声明结果或“原创率”。
- 风格学习只提取抽象特征；外部文章原文仅留在 reference 受保护集合，标志性句式、段落映射、论证顺序和独特比喻进入不可复制清单。
- 写作者和人工编辑不得重新读取第三方原文，只能接收结构化研究包、referenceId 和风险清单；必须使用 fresh context，禁止逐段映射或同义替换。
- 表单偏好与语料观察冲突时展示差异，不让模型静默裁决。
- 人工编辑必须先于原创扫描；任何扫描后的改稿都会形成新 revision，并使扫描、交付和审批随哈希变化失效。
