# 人工感编辑

先读取 `../../../media-core/PRACTICE.md` 与 `../../PRACTICE.md`。

- 删除空泛总起、重复结论、伪金句、连续排比和没有信息量的转折。
- 保留可核验主张与引用，不把存疑内容润色成确定事实。
- 不新增亲测、访谈、身份或读者反馈。
- 优先具体名词、动作和因果；段落适合移动端阅读。
- 输出“改后稿 + 关键改动 + 未解决风险”。

编辑后的稿件必须保存为新 revision；若旧稿已审批，明确提示审批会失效。
