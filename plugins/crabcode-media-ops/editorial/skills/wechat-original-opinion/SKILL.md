# 微信原创观点

先读取 `../../../media-core/PRACTICE.md`、`../../PRACTICE.md`，再读取 `../../references/research-standard.md` 与 `../../references/wechat-editorial-playbook.md`。

## 写作流程

1. 外部文章先登记 referenceId；没有完成 `mediaops.research.complete` 不进入写作。第三方参考只能提供选题线索、抽象风格特征和不可复制清单。
2. 在不包含参考原文、附件或原路径的 fresh context 中，先独立给出一句可争辩的中心判断，不用“重要、值得关注”代替观点。
3. 使用结构化主张台账；关键事实至少有可追溯、已打开且相互独立的来源。
4. 写出独立的因果链、案例组合、最强反方和结论边界；禁止沿用参考文的论证顺序、段落映射、独特比喻或只做同义替换。
5. 开头提供冲突或具体问题；正文按“事实 → 解释 → 推论”推进；结尾回到普通人的认知或行动。
6. 禁止虚构亲测、访谈、业内消息、读者反馈或第一人称经历。
7. 不保证原创率或微信原创声明；成稿先保存为 drafted，人工编辑后交 `media-originality-review`，不得自行宣称可发布。

输出标题备选、摘要、正文、主张台账、反方/边界和风险提示。原创风险由 `media-originality-review` 独立复核。
