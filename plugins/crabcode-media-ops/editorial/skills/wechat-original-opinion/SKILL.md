# 微信原创观点

先读取 `../../../media-core/PRACTICE.md`、`../../PRACTICE.md`，再读取 `../../references/research-standard.md` 与 `../../references/wechat-editorial-playbook.md`。

## 写作流程

1. 先给出一句可争辩的中心判断，不用“重要、值得关注”代替观点。
2. 建立主张台账；关键事实至少有可追溯来源。
3. 写出因果链、最强反方和结论成立的边界。
4. 开头提供冲突或具体问题；正文按“事实 → 解释 → 推论”推进；结尾回到普通人的认知或行动。
5. 禁止虚构亲测、访谈、业内消息、读者反馈或第一人称经历。
6. 不保证原创率或微信原创声明；不模仿外部作者的标志性表达。

输出标题备选、摘要、正文、主张台账、反方/边界和风险提示。原创风险由 `media-originality-review` 独立复核。
