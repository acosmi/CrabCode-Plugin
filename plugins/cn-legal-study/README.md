# 中国法律学习(法考/法硕备考)

面向中国法律学习与法考/法硕备考的 CrabCode 插件:复习计划、知识体系大纲、记忆卡片、案例研习、客观题与主观题训练、法条背诵、模拟问答与答疑讲解。仅供学习备考使用,不构成个案法律意见。

## 安装

在 CrabCode 插件市场中添加 `cn-legal-study`,或通过 `/plugin` 安装。

## 技能(按需自动触发)

| 技能 | 说明 |
|---|---|
| `study-plan` | 法考(客观/主观)或法硕备考的阶段化复习计划 |
| `outline-builder` | 部门法/专题知识体系大纲,搭建可记忆的逻辑框架 |
| `study-session` | 具体知识点答疑讲解,随问随答讲透考点 |
| `flashcards` | 法条、考点、易混概念的问答式记忆卡片(间隔重复) |
| `statute-recite` | 重点法条背诵:填空默写、关键词提示、理解性记忆 |
| `case-study` | 指导性/典型案例研习:请求权基础与法律关系分析 |
| `exam-question-practice` | 客观题真题与模拟练习,逐题精讲复盘错因 |
| `essay-practice` | 主观题(案例分析/论述/法律文书)作答与评讲打分 |
| `legal-writing` | 法律写作训练:论述结构、逻辑层次、法言法语 |
| `mock-interview` | 追问式问答训练答题思路与口头表达 |
| `exam-forecast` | 基于命题规律与法律修订预测考点趋势、划定重点 |

## 配套插件(可选)

复习材料需要成品文档时(如把大纲导出为 Word 讲义、把卡片整理成 Excel 表),建议同时安装 `crabcode-office-suite`,相应技能会按 `crabcode-office-suite:crabcode-documents` / `crabcode-office-suite:crabcode-spreadsheets` 全限定名路由(详见仓库根 `docs/capability-routing.md`)。

## 使用入口

直接说"帮我制定三个月法考复习计划""讲一下善意取得"即可触发对应技能。
