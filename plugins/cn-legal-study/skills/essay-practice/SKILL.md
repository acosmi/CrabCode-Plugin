# /cn-legal-study:essay-practice

【教育备考用途，非针对个案的法律意见】

围绕法考主观题(案例分析、法律论述、法律文书题)开展作答训练与评讲打分。

## Workflow

1. 确认题型:区分案例分析题、论述题、法律文书题(如起诉状、答辩状、法律意见书框架);明确学科与考查重点。
2. 接题或出题:接收考生提供的题目与作答,或按主观题命题风格生成题目;来源标 `[用户提供]` 或 `[已核验-来源]`。
3. 拆解题眼:提炼争点、采分点与作答结构要求(以请求权基础/法律关系分析法组织实体作答,替代英美 IRAC)。
4. 示范作答框架:给出"结论先行 → 法律依据 → 要件涵摄 → 结论回扣"的答题骨架与法言法语表达。
5. 评讲考生答案:对照采分点逐项打分,指出漏点、说理不足、法条引用错误;法条口径不确定处标 `[模型知识-待核]`,提示以现行法条与官方教材为准。
6. 给提升清单:列出本题暴露的能力短板与下一步训练动作。

## Output

- 题目与采分点清单。
- 参考答案/答题框架(含法言法语示范)。
- 考生答案逐项评分与批注。
- 漏点与说理改进建议。
- 提升清单。
- 来源标签标注。
