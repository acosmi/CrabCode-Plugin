# /cn-legal-study:mock-interview

【教育备考用途，非针对个案的法律意见】

通过逐层追问训练法考主观题的答题思路与口头表达,以中国法考主观题思路推进式追问替代英美 Socratic/cold-call 方法。

## Workflow

1. 设定主题:确认要训练的学科、争点或案情,明确以"理清思路"还是"演练表达"为主。
2. 开场设问:从基础概念或争点起手,要求考生口头/书面作答。
3. 逐层追问:按"识别法律关系 → 锁定请求权基础/罪名 → 拆要件 → 涵摄事实 → 得结论"的中国主观题思路链条递进追问,每一步基于考生上一答案深挖。
4. 即时纠偏:对错误前提、漏掉的要件、混淆的概念当场点出并引导修正;法条口径不确定处标 `[模型知识-待核]`,提示以现行法条与官方教材为准。
5. 表达打磨:点评法言法语、条理性与结论明确度,示范更规范的口头表述。
6. 收束复盘:总结本轮思路链的薄弱环节与改进方向。

## Output

- 追问对话记录(问—答—纠偏)。
- 思路链条还原(从争点到结论)。
- 表达与法言法语点评。
- 暴露的思路/知识短板。
- 改进方向清单。
- 来源标签标注(涉及法条处)。
