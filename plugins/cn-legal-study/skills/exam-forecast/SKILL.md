# /cn-legal-study:exam-forecast

【教育备考用途，非针对个案的法律意见】

依据历年命题规律、考试大纲与近期立法修订动态,预测法考/法硕考点趋势并划定复习重点(仅为概率性研判,非命题信息)。

## Workflow

1. 明确口径:确认考试类型、目标学科与预测年度;声明本预测基于公开规律研判,不掌握真实命题信息。
2. 复盘历年规律:梳理近年高频考点、轮考冷热点与题型分布;数据口径标 `[模型知识-待核]`,提示以官方大纲为准。
3. 跟踪法律修订:识别新颁布/修订法律、司法解释与指导性案例可能带来的新增或调整考点,标 `[已核验-来源]` 或 `[模型知识-待核]`。
4. 研判热点交叉:结合社会热点与立法导向,标注可能的命题切入角度。
5. 划定重点层级:把考点分为"必拿分核心区 / 高概率提升区 / 了解性边缘区"。
6. 给资源配比:据重点层级建议时间与精力分配,并提示风险(预测有不确定性,基础全覆盖优先)。

## Output

- 考点趋势研判摘要(附不确定性声明)。
- 历年高频/轮考规律表。
- 新法新解释带来的潜在新增考点。
- 重点层级划分(核心/提升/边缘)。
- 精力配比与风险提示。
- 来源标签一览。
