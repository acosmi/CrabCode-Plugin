# /cn-legal-study:study-plan

【教育备考用途，非针对个案的法律意见】

为国家统一法律职业资格考试(法考)或法律硕士入学考试制定可执行的阶段化备考计划。

## Workflow

1. 厘清目标:确认考试类型(法考客观题/主观题、法硕法学/非法学)、目标分数线、距考试可用周数、每周可投入小时数。
2. 评估基线:盘点已学学科、薄弱学科、是否有过往真题正确率数据;记录考生自述信息来源 `[用户提供]`。
3. 划分阶段:按"基础精讲 → 强化提升 → 真题演练 → 主观题/背诵冲刺"四阶段排布,每阶段给出周数与里程碑。
4. 分配学科权重:按各部门法在该考试中的相对分值与难度配比时间(民法、刑法、行政法、三大诉讼法、商经法、理论法、国际法等);引用分值口径时标 `[模型知识-待核]`,提示考生以官方考试大纲与现行规定为准。
5. 嵌入复习节律:安排周复盘、错题回顾、记忆曲线滚动复习与每周自测。
6. 设定可量化检查点:每阶段末给出"应完成章节/应达正确率"的自检指标。

## Output

- 备考概览(考试类型、目标、总周数、每周时长)。
- 阶段甘特式时间表(阶段/周次/学科/任务/里程碑)。
- 学科时间权重表(标注分值口径来源标签)。
- 每周固定节律模板(精讲/练题/复盘/自测)。
- 风险与弹性预案(进度滞后时的取舍优先级)。
- 阶段检查点清单。
