# CrabCode-Plugin v0.3.0 发布说明

发布日期：2026-07-12

## 重点：crabcode-media-ops 0.3.0

- 新增九技能伞形架构与共享 Media Gate；
- 新增微信公众号原创观点、原创风险复核、选题研究能力；
- 新增快速、完整、增量创作者风格表单，支持草稿、恢复、提交、冲突确认、profile 版本历史与回滚；
- 内容改为不可变 revision、统一 manifest 和 SHA-256 contentHash；
- 审批增加 pending/approved/rejected/revoked 状态并绑定平台、revisionId 与 contentHash；
- 未审、未完成原创/法律路由、披露不完整、规则过期、审批后改稿均由确定性工具阻断；
- 发布包复制实际资源，并写入来源、profile、审稿、披露和审批追溯字段；
- 平台规则增加来源、核验日期、适用范围、规则类型和失效策略；
- 修复中文热点按空格分词导致的聚类失效；
- 统一插件 manifest、MCP package、server、capabilities 与 marketplace 版本为 0.3.0。

## 合规口径修正

AI 辅助内容记录平台原生标识、正文提示和文件元数据中的实际披露方式，不再把固定正文句子描述成所有平台唯一法定形式。

## 范围边界

真实平台 API、浏览器最终点击、自动评论、非官方爬虫、全网查重和平台原创声明保证仍未开放。
