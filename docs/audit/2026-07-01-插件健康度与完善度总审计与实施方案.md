## 第二部分 已定决策(第一性原理,无待决项)

推导公理:①产品价值=产出物被接受概率(平台不违规/版权中心不驳回)×省下的人工时间;②确定性可判定交代码、需判断交模型、责任归属交人(mediaops MCP server 自身"只做确定性 I/O、从不调模型"即此公理);③已承诺未兑现(文档-代码不符)是信任债,等同 bug,优先于一切新能力。
