## 第二部分 已定决策(第一性原理,无待决项)

推导公理:①产品价值=产出物被接受概率(平台不违规/版权中心不驳回)×省下的人工时间;②确定性可判定交代码、需判断交模型、责任归属交人(mediaops MCP server 自身"只做确定性 I/O、从不调模型"即此公理);③已承诺未兑现(文档-代码不符)是信任债,等同 bug,优先于一切新能力。

| # | 决策点 | 裁决 | 推导 |
|---|---|---|---|
| D1 | bio-research 镜像改动 | **回退 `.mcp.json` 至 claude.com 端点**;镜像部署+DNS 建好后另行切换(基础设施事项,不阻塞本仓) | NXDOMAIN 必挂,提交即破坏 4 连接器;公理③ |

| D4 | 阶段 0 止血 | **立即执行**:small-business 摘除 paypal(不通)/quickbooks/docusign(403) | 已实测三端点当前网络不可用,留着只产生 failed 噪音 |
| D5 | agent `model` 字段 | **放宽 schema:Required→No,缺省=inherit**;不去补 25 个文件 | schema 应描述现实与加载器行为;改 1 行文档优于改 25 个文件 |
