# CrabFin-CN 监管口径核对指南(给律师 / 合规官 / 会计师)

> 用途:本套件 6 个中国子板块的 `sources/` 监管/会计素材**由 AI 依据公开规则整理,未逐条对照原文核对**,现统一标注为"非合规依据"。本指南告诉你:**每个板块要核对哪些具体条目、对照哪个官方原文、由谁核对、优先级**。核对通过后,请在对应源文件顶部横幅处改标:`✅ 已核对 — 核对人/资质/日期/所据版本`。
>
> 原则:① **只认官方原文**(下表"对照源"列),不以二手解读为准;② 重点核对**数字/日期/阈值/令号/条文号**——这些是错得最隐蔽、危害最大的;③ 发现与现行法规不符的,直接在源文件中改正并记录修改人。

## 四、核对节奏(建议)

1. **先做 🔴 高优先级**(每板块 1–2 项),它们决定技能"算得对不对"。
2. **单点突破**:建议从 `cn-equity-research` 起(CAS 科目映射 + 披露规则两项),核对通过后该板块即可由主控升级为"可用",再复制到其余板块。
3. 每核对完一项,在源文件顶部横幅改标 `✅ 已核对 — 人/资质/日期/版本`;有改动的同步记录。
4. 一个板块全部 🔴 项过关后,通知主控:主控据此把该板块从"草稿/不作合规依据"升级为"可用",并更新 PRACTICE.md 与 marketplace 描述。
